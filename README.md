# ML Stats

Interactive study guide for Murphy's *Machine Learning* textbook.

## Overview

A web-based reader with interactive simulators that complement the textbook content. Includes chapter navigation, page-by-page reading, and hands-on ML concept demonstrations.

## Getting Started

```bash
./run.sh
```

Open `index.html` in your browser, or use `reader.html` for the reading view. Explore `simulators/` for interactive demos.

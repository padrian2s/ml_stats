Proprietatile de independenta conditionata ale UGM-urilor

Aceasta pagina introduce proprietatile Markov ale modelelor grafice nedirectionate (UGM), incluzand patura Markov, proprietatea Markov locala si proprietatea Markov pe perechi.

Patura Markov (Markov blanket): Multimea de noduri care face un nod t conditionat independent de toate celelalte noduri din graf. In UGM-uri, patura Markov a unui nod este formata din vecinii sai directi, ceea ce se numeste proprietatea Markov locala nedirectionata.

Comentariu

Rezumat

Pagina introduce proprietatile fundamentale de independenta conditionata ale UGM-urilor. In comparatie cu DGM-urile, determinarea independentei conditionate in UGM-uri este mult mai simpla: se verifica doar daca eliminarea unui set de noduri deconecteaza alte noduri in graf. Sunt definite trei proprietati Markov: pe perechi, locala si globala.

Tipare Cheie

Patura Markov in UGM-uri este mai simpla decat in DGM-uri: include doar vecinii directi, fara a necesita copii si co-parinti. Proprietatea Markov pe perechi permite verificarea independentei conditionate doar prin inspectarea structurii grafului. Moralizarea este procesul de conversie DGM -> UGM, dar poate pierde informatii de independenta conditionata.

Aplicatii Practice

UGM-urile sunt folosite in procesarea imaginilor, fizica statistica si modelarea retelelor sociale, unde relatiile simetrice sunt mai naturale decat cele directionate. Proprietatile Markov permit inferenta eficienta in aceste modele.

Exemplu din Lumea Reala

Intr-o retea de senzori, fiecare senzor masoara temperatura locala. Un senzor este conditionat independent de senzorii indepartati daca se cunosc valorile senzorilor vecini - aceasta este exact proprietatea Markov locala a UGM-urilor.

Matricea de tranzitie si ecuatiile Chapman-Kolmogorov

Aceasta pagina detaliaza matricea de tranzitie, diagramele de stare, tranzitia stanga-dreapta si ecuatiile Chapman-Kolmogorov.

Matrice de tranzitie stanga-dreapta: O matrice de tranzitie cu A_{ij} = 0 pentru j < i, permitand doar tranzitii inainte. Folosita in recunoasterea vorbirii unde secventa de foneme progreseaza intr-o singura directie.
Ecuatiile Chapman-Kolmogorov: A_{ij}(m+n) = sum_k A_{ik}(m) * A_{kj}(n), echivalent cu A(m+n) = A(m) * A(n). Implica A(n) = A^n: tranzitia in n pasi se obtine ridicand matricea A la puterea n.

Comentariu

Rezumat

Figura 17.1 ilustreaza doua diagrame de tranzitie: (a) un lant cu 2 stari cu parametri alpha si beta, si (b) un lant stanga-dreapta cu 3 stari unde tranzitiile sunt doar inainte (starea 3 este absorbanta). Ecuatiile Chapman-Kolmogorov permit calculul probabilitatilor de tranzitie pe n pasi prin inmultirea matriceala A(n) = A^n. Aceasta este eficient calculata prin 'matrix powering' (ridicare repetata la patrat).

Tipare Cheie

Matricea stanga-dreapta este naturala pentru procese cu progresie temporala unidirectionala. Starea absorbanta (A_{33}=1) indica sfarsitul procesului. Lantul cu 2 stari este complet caracterizat de doar 2 parametri (alpha, beta), demonstrand parsimonia modelului Markov.

Aplicatii Practice

In recunoasterea vorbirii, fiecare cuvant este modelat ca un HMM stanga-dreapta cu stari corespunzand fonemelor. In bioinformatica, genele sunt modelate ca secvente de stari (exon, intron, inter-genic) cu tranzitii stanga-dreapta.

Exemplu din Lumea Reala

In modelarea degradarii echipamentelor industriale, un lant Markov stanga-dreapta cu stari 'Nou' -> 'Uzat' -> 'Defect' permite estimarea duratei de viata ramase si planificarea mentenantei preventive.

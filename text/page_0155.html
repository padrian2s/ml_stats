Comentariu

Rezumat

Pagina face o legatura importanta: media posterioara a modelului Gaussian liniar este echivalenta cu solutia problemei de optimizare Tikhonov. Aceasta leaga inferenta Bayesiana de regularizare: priorul Gaussian corespunde penalizarii L2 (Tikhonov), iar varianta zgomotului determina balanta intre fidelitate si netezire. Demonstratia Teoremei 4.4.1 procedeaza prin formarea distributiei comune si aplicarea formulelor de conditionare.

Tipare Cheie

Echivalenta intre inferenta Bayesiana si regularizare este profunda: MAP = regularizare Tikhonov, priorul = penalizare, varianta priorului = 1/λ. Aceasta interpretare probabilistica ofera justificari pentru alegerea tipului si cantitatii de regularizare.

Aplicatii Practice

Regularizarea Tikhonov apare in probleme inverse (deconvolutie, tomografie), procesarea imaginilor (denoising, super-rezolutie), geostatistica si orice problema slab conditionata care necesita regularizare.

Exemplu din Lumea Reala

In seismologie, estimarea structurii subterane din unde seismice este o problema inversa slab conditionata. Regularizarea Tikhonov (echivalenta cu un prior Gaussian de netezire) produce imagini ale subsolului care sunt atat compatibile cu datele seismice cat si suficient de netede pentru a fi geologicamente plauzibile.

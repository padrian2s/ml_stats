Exerciții - Capitolul 3 (Partea 4): Analiza Bayesiană a Distribuției Exponențiale

Pagina prezintă exerciții detaliate despre analiza Bayesiană a distribuției exponențiale, MLE și estimarea MAP pentru Bernoulli cu priori neconjugate.

Exercițiul 3.11 - Distribuția exponențială: p(x|θ) = θe^{−θx}. MLE: θ̂ = 1/x̄ = N/Σxᵢ. Prior conjugat exponențial: p(θ|λ) = Expon(θ|λ). Prior conjugat general: Ga(θ|a,b).
Posteriorul exponențial: p(θ|D,λ̂) ∝ θ^N·e^{−θ(Σxᵢ+λ̂)} = Ga(θ|N+1, Σxᵢ+λ̂). Media posterioară: (N+1)/(Σxᵢ+λ̂).
Exercițiul 3.12 - MAP cu prior neconjugat: Prior discret pe θ: p(θ=0.5) = 0.5, p(θ=0.4) = 0.5. MAP: se compară p(D|θ=0.5)·0.5 vs p(D|θ=0.4)·0.5 pentru diferite N.
Diferența MLE vs media posterioară: MLE: 1/x̄. Media posterioară: (N+1)/(Σxᵢ+λ̂). Cu date puține, acestea pot diferi semnificativ; media posterioară este mai stabilă.

Comentariu

Rezumat

Distribuția exponențială modelează timpi de așteptare cu proprietatea de lipsă de memorie. Priorul conjugat Gamma produce un posterior tractabil, similar modelului beta-binomial.

Tipare Cheie

Priorul exponențial este un caz special de prior Gamma (a=1). Cu priori neconjugate, posteriorul nu are formă analitică simplă, dar MAP poate fi calculat prin enumerare sau optimizare.

Aplicatii Practice

Distribuția exponențială modelează durata de viață a componentelor (fiabilitate), timpii între evenimente (procese Poisson), și timpii de așteptare în cozi (telecomunicații).

Exemplu din Lumea Reala

Estimarea ratei de defectare a serverelor dintr-un centru de date: cu doar 3 observații (5, 6, 4 ani), media posterioară cu un prior Gamma informativ oferă estimări mai fiabile decât MLE.

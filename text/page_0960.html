Rephil - Inferenta si modele cu ecuatii structurale (SEM)

Aceasta pagina continua descrierea inferentei in Rephil si introduce modelele cu ecuatii structurale (Structural Equation Models).

Modelul cu ecuatii structurale (SEM): Un graf mixt dirijat (posibil ciclic) unde toate CPD-urile sunt gaussiene liniare si muchiile bi-directionate reprezinta zgomot gaussian corelat. Cunoscut si sub numele de "diagrama de cai" (path diagram).
Forma SEM: x_i = mu_i + suma w_{ij}*x_j + epsilon_i, unde epsilon ~ N(0, Psi). In forma matriciala: x = Wx + mu + epsilon, deci x = (I-W)^{-1}(epsilon + mu).
SEM recursiv: Daca W este triunghiulara inferioara si Psi este diagonala, SEM-ul este echivalent cu un DGM gaussian aciclic. Daca Psi nu este diagonala, muchiile bi-directionate indica cauze comune ascunse.
Distributia comuna: p(x) = N(mu, Sigma) unde Sigma = (I-W)^{-1} Psi (I-W)^{-T}. Se traseaza un arc X_i <-> X_j pentru fiecare element nenul off-diagonal din Psi.

Comentariu

Rezumat

Pagina face tranzitia de la Rephil la SEM-uri, modele larg folosite in stiintele sociale si economie. SEM-urile permit cicluri (interpretate ca bucle de feedback), ceea ce le distinge de DGM-urile standard. Forma matriciala x = (I-W)^{-1}(epsilon+mu) arata ca modelul specifica o distributie gaussiana comuna, chiar daca relatiile individuale par "cauzale".

Tipare Cheie

SEM-urile nu sunt inerent "cauzale" desi sunt adesea interpretate astfel. Ele sunt pur si simplu o parametrizare a unei distributii gaussiene comune. Distinctia intre SEM-uri recursive (aciclice) si non-recursive (ciclice) este importanta: cele recursive sunt echivalente cu DGM-uri gaussiene.

Aplicatii Practice

SEM-urile sunt standardul in psihologie, sociologie, marketing si economie pentru modelarea relatiilor intre variabile latente si observate. Software-ul LISREL, Mplus si lavaan sunt larg folosite in aceste domenii.

Exemplu din Lumea Reala

In psihologia organizationala, SEM-urile modeleaza relatiile intre satisfactia la locul de munca (latenta), performanta (observabila prin evaluari), si intentia de plecare (masurabil prin sondaje), permitand testarea teoriilor cauzale complexe.

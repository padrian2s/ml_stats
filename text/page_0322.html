Exemple de GLM-uri: regresia liniara, binomiala si Poisson

Aceasta pagina prezinta trei exemple concrete de GLM-uri: regresia liniara (Gaussiana), regresia binomiala (logistica) si regresia Poisson.

Regresia liniara ca GLM: y_i ~ N(μ_i, σ²), θ_i = μ_i = w^T x_i (legatura identitate), A(θ) = θ²/2, deci E[y_i] = μ_i si var[y_i] = σ².
Regresia binomiala ca GLM: y_i ~ Bin(N_i, π_i), θ_i = log(π_i/(1-π_i)) = w^T x_i (legatura logit), A(θ) = N_i log(1+e^θ), E[y_i] = N_i π_i, var[y_i] = N_i π_i(1-π_i).
Regresia Poisson ca GLM: y_i ~ Poi(μ_i), θ_i = log(μ_i) = w^T x_i (legatura log), A(θ) = e^θ, E[y_i] = μ_i = e^(w^T x_i), var[y_i] = μ_i. Folosita in bio-statistica, epidemiologie si genomica.
Proprietatea egalitatii medie-varianta in Poisson: E[y_i] = var[y_i] = μ_i. Aceasta presupunere este adesea violata in practica (supra-dispersie), necesitand modele mai flexibile (negative binomiala, quasi-Poisson).

Comentariu

Rezumat

Pagina demonstreaza concret cum trei modele de regresie familiare sunt cazuri speciale ale cadrului GLM. Regresia liniara foloseste legatura identitate, logistica foloseste legatura logit, iar Poisson foloseste legatura logaritmica. Toate au aceeasi structura generala si pot fi estimate cu acelasi algoritm (IRLS), doar cu functii de legatura diferite.

Tipare Cheie

1) Regresia liniara este singurul GLM cu varianta constanta (homoscedasticitate). 2) In regresia logistica, varianta depinde de media (π(1-π)), deci este heteroscedastica. 3) In Poisson, varianta = media, o restrictie puternica care trebuie verificata. 4) Legatura log in Poisson asigura ca μ_i > 0 (numarul asteptat de evenimente este pozitiv).

Aplicatii Practice

Regresia Poisson este folosita in epidemiologie (incidenta bolilor), in transporturi (numarul de accidente pe un segment de drum), in genomica (numarul de citiri per gen), si in economie (numarul de brevete per firma). Regresia binomiala modeleaza proportii: rata de mortalitate, rata de defecte, rata de conversie.

Exemplu din Lumea Reala

In genomica, secventierea ARN produce numere de citiri per gen care urmeaza aproximativ o distributie Poisson. Regresia Poisson (sau negativa binomiala pentru supra-dispersie) cu predictori (grup experimental, batch) este metoda standard pentru identificarea genelor diferential exprimate intre conditii (pachetul DESeq2 in R).

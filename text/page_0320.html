Derivarea familiei exponentiale din principiul entropiei maxime si introducerea GLM-urilor

Aceasta pagina justifica familia exponentiala prin principiul entropiei maxime (MaxEnt) si incepe prezentarea modelelor liniare generalizate (GLM).

Principiul entropiei maxime (MaxEnt): Distributia care face cele mai putine presupuneri, sub constrangerile ca Σ f_k(x) p(x) = F_k (momentele empirice sunt respectate), este de forma: p(x) = (1/Z) exp(-Σ_k λ_k f_k(x)), care este exact forma familiei exponentiale (distributia Gibbs).
Modele liniare generalizate (GLM): Modele in care distributia outputului apartine familiei exponentiale si media este o functie neliniara a unei combinatii liniare a inputurilor. Include regresia liniara (Gaussiana), logistica (Bernoulli) si Poisson ca cazuri speciale.
Distributia Gibbs = familia exponentiala: Distributia maxent p(x) = (1/Z) exp(-Σ λ_k f_k(x)) are exact forma familiei exponentiale. Aceasta inseamna ca familia exponentiala nu este doar convenabila matematic, ci este fundamentata de un principiu de inferenta profund: face cele mai putine presupuneri compatibile cu datele.

Comentariu

Rezumat

Pagina ofera o justificare filozofica profunda pentru familia exponentiala: este distributia care face cele mai putine presupuneri sub constrangerile date de momentele empirice. Aceasta justifica utilizarea Gaussienei (cand stim doar media si varianta), a Poisson (cand stim doar media), etc. Apoi se introduce cadrul GLM, care unifica regresia liniara, logistica si Poisson intr-un singur cadru.

Tipare Cheie

1) MaxEnt = metoda multiplicatorilor Lagrange pe entropia Shannon sub constrangeri de momente. 2) Familia exponentiala este unica in a fi maxent sub constrangeri de momente. 3) GLM = distributie din familia exponentiala + combinatie liniara de inputuri + functie de legatura. 4) Diferite functii de legatura produc diferite modele: identitate (liniar), logit (logistic), log (Poisson).

Aplicatii Practice

MaxEnt este folosit in procesarea limbajului natural (clasificatoare MaxEnt = regresie logistica), in fizica statistica (distributia Boltzmann), in economie (alegeri discrete), si in ecologie (modelarea distributiei speciilor). GLM-urile sunt fundamentul analizei statistice in epidemiologie, asigurari, si marketing.

Exemplu din Lumea Reala

Cand un actuar modeleaza numarul de accidente auto per an, alege distributia Poisson nu arbitrar, ci pentru ca este distributia maxent pentru numere intregi non-negative cu medie data. GLM-ul Poisson cu predictori (varsta soferului, tipul masinii, zona geografica) permite estimarea primelor de asigurare personalizate.

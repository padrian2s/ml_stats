Distributia NIχ² si posteriorul predictiv

Aceasta pagina prezinta distributia Normal-Inverse-Chi-patrat (NIχ²) pentru cazul univariat, posteriorul predictiv Student multivariat, si vizualizarea distributiei NIχ².

Distributia NIχ²: NIχ²(μ, σ² | m_0, κ_0, ν_0, σ_0²) = N(μ | m_0, σ²/κ_0) × χ^(-2)(σ² | ν_0, σ_0²). Aceasta este versiunea scalara a distributiei NIW, folosita frecvent in literatura statistica.
Posteriorul predictiv: p(x | D) = T(x | m_N, (κ_N + 1)/(κ_N(ν_N - D + 1)) S_N, ν_N - D + 1). Distributia predictiva este Student multivariata, cu cozi mai grele decat Gaussiana, reflectand incertitudinea asupra parametrilor.

Cresterea κ_0 ingusteaza distributia pe axa μ, iar cresterea ν_0 o ingusteaza pe axa σ². Posteriorul predictiv este derivat prin integrarea peste parametrii necunoscuti.

Tipare Cheie

1) Contururile distributiei NIχ² sunt mai ascutite pentru valori mici ale σ², ceea ce inseamna ca media este estimata mai precis cand varianta este mica. 2) Posteriorul predictiv Student are cozi mai grele decat Gaussiana, incorporand natural incertitudinea parametrica. 3) Pe masura ce N creste, posteriorul predictiv devine Gaussian.

Aplicatii Practice

Posteriorul predictiv este esential in planificarea experimentala (posterior predictive checks), in detectia de anomalii (un punct este anomal daca are probabilitate foarte mica sub predictivul posterior), si in selectia de modele prin verosimilitatea marginala.

Exemplu din Lumea Reala

In controlul proceselor industriale, distributia predictiva posterioara este folosita pentru a genera benzi de toleranta adaptive: pe masura ce se acumuleaza masuratori, benzile se ingusteaza (incertitudinea scade). Forma Student a predictivei ofera protectie impotriva alarmelor false cauzate de variabilitatea naturala nemodelata.

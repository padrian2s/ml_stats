Clasificatorul cu Centroizi Contractati (Nearest Shrunken Centroids)

Pagina 109 introduce clasificatorul cu centroizi contractati, o metoda care selecteaza automat trasaturile relevante prin regularizare cu sparsitate, aplicata pe date de expresie genica.

Figura 4.7: Eroarea vs. cantitatea de contractie (shrinkage) pentru clasificatorul cu centroizi contractati aplicat pe datele SRBCT de expresie genica. Numarul de gene selectate scade de la 2308 la 0 pe masura ce λ creste. Eroarea pe test ramane zero pentru un interval larg de λ.
Clasificatorul cu centroizi contractati (Nearest Shrunken Centroids): Se realizeaza estimarea MAP pentru LDA diagonala cu un prior Laplace (promotor de sparsitate). Media specifica clasei se descompune: μ_cⱼ = mⱼ + Δ_cⱼ, unde mⱼ este media globala si Δ_cⱼ este abaterea specifica clasei.
Selectia automata a trasaturilor (Ecuatia 4.66): μ_cⱼ = mⱼ + Δ_cⱼ. Daca Δ_cⱼ = 0 pentru toate clasele c, trasatura j este independenta de clasa si este eliminata automat din clasificare.
Rezultate pe date genomice: Pe setul SRBCT (2308 gene, 4 clase, 63 antrenare, 20 test): LDA diagonala produce 5 erori, iar centroizi contractati produce 0 erori folosind doar 39 din 2308 gene. Pe setul cu 14 tipuri de cancer (16603 gene): 17 erori pe test cu doar 6520 gene selectate.

Comentariu

Rezumat

Clasificatorul cu centroizi contractati combina puterea LDA cu regularizarea prin sparsitate. Prin impunerea unui prior Laplace pe abaterile de la medie Δ_cⱼ, trasaturile neinformative sunt eliminate automat (Δ_cⱼ → 0), rezultand un model mai simplu, mai interpretabil si adesea mai precis.

Tipare Cheie

Regularizarea prin sparsitate (prior Laplace/L1) produce selectie automata de trasaturi, spre deosebire de regularizarea L2 care doar micsoreaza coeficientii fara sa-i elimine. Curba eroare vs. shrinkage in Figura 4.7 arata o "platforma" de performanta optima, indicand robustetea la alegerea lui λ.

Aplicatii Practice

Aceasta metoda este standard in genomica si proteominca pentru identificarea biomarkerilor - genele selectate (Δ_cⱼ ≠ 0) sunt candidati pentru biomarkeri de diagnostic. Este implementata in pachetul R "pamr".

Exemplu din Lumea Reala

In diagnosticul cancerului pediatric (SRBCT - Small Round Blue Cell Tumors), clasificatorul cu centroizi contractati identifica 39 de gene din 2308 care sunt suficiente pentru a distinge perfect cele 4 subtipuri de tumori. Aceste gene devin tinte pentru investigatii biologice ulterioare si potentiale teste de diagnostic.

Clustere Kikuchi, hiper-grafuri si energia libera Kikuchi

Aceasta pagina detaliaza metoda clusterelor Kikuchi, cu exemple pe grile 3x3, numerele de supranumarare (overcounting) si energia libera Kikuchi.

Clustere Kikuchi: O partitionare ierarhica a grafului in clustere suprapuse, organizate intr-un poset (set partial ordonat). De exemplu, pe o grila 3x3, clusterele de nivel superior sunt patratele 2x2, iar clusterele de nivel inferior sunt perechile de noduri.
Numere de supranumarare (overcounting numbers): c(g) este numarul de supranumarare al clusterului g, legat de numerele Mobius din teoria multimilor. Asigura ca fiecare regiune a grafului este numarata exact o data in aproximarea entropiei.
Energia libera Kikuchi: F_Kikuchi(tau) = -[theta^T*tau + H_Kikuchi(tau)], unde H_Kikuchi(tau) = sum_{g in E} c(g)*H_g(tau_g). Generalizeaza energia libera Bethe (care este un caz special cu clustere de dimensiune 2).
Exemplu grila 3x3: H_Kikuchi = [H_1245 + H_2356 + H_4578 + H_5689] - [H_25 + H_45 + H_56 + H_58] + H_5. Clusterele mari contribuie pozitiv, cele mici negativ, si nodul central apare o data pozitiv.

Comentariu

Rezumat

Pagina detaliaza constructia aproximarii Kikuchi: se aleg clustere (de exemplu, patrate 2x2 pe o grila), se calculeaza numerele de supranumarare folosind formula de incluziune-excludere (numere Mobius), si se defineste energia libera Kikuchi ca generalizare a lui Bethe.

Tipare Cheie

Principiul incluziunii-excluderii din combinatorica apare natural in definirea numerelor de supranumarare. Aceasta asigura ca entropia fiecarei regiuni este contabilizata exact o data, corectand supranumararea din ierarhia de clustere.

Aplicatii Practice

Alegerea clusterelor Kikuchi este critica: clustere prea mari fac calculul intractabil, clustere prea mici nu imbunatatesc semnificativ acuratetea. Pentru grile 2D, clustere patrate sunt o alegere naturala.

Exemplu din Lumea Reala

In simularea materialelor magnetice (modelul Ising pe retele cristaline), aproximarea Kikuchi cu clustere patrate ofera predictii termodinamice semnificativ mai precise decat aproximarea Bethe/mean field simpla.

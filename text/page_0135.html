Analiza Discriminanta Liniara (LDA) - Continuare si LDA cu Doua Clase

Pagina 104 continua prezentarea LDA, aratand ca frontierele de decizie sunt liniare, introduce regresia logistica multi-clasa si detaliaza cazul cu doua clase.

Posteriorul LDA (Ecuatia 4.38): p(y=c|x, θ) = e^(β_cᵀx + γ_c) / Σ_c' e^(β_c'ᵀx + γ_c') = S(η)_c, unde η = [β₁ᵀx + γ₁, ..., β_Cᵀx + γ_C]
Frontiera de decizie liniara (Ecuatia 4.43): Frontiera intre clasele c si c' este definita de xᵀ(β_c' - β) = γ_c' - γ_c, care este un hiperplan (linie dreapta in 2D).
LDA cu doua clase (Ecuatia 4.45): p(y=1|x, θ) = 1/(1 + e^((β₀-β₁)ᵀx + (γ₀-γ₁))) = sigm((β₁ - β₀)ᵀx + (γ₁ - γ₀))
Regresia logistica multi-clasa: O alternativa discriminativa la LDA: p(y|x, W) = Cat(y|Wx), unde W este o matrice C x D de ponderi. Numita si regresie logistica multinomiala sau model de entropie maxima.

Comentariu

Rezumat

LDA produce frontiere de decizie liniare deoarece termenul patratic xᵀΣ⁻¹x se anuleaza in raportul de probabilitati (fiind independent de clasa). In cazul binar, LDA se reduce la regresia logistica cu parametri derivati din μ₀, μ₁ si Σ. Regresia logistica multi-clasa este alternativa discriminativa care estimeaza direct frontierele de decizie.

Tipare Cheie

Relatia intre LDA (generativ) si regresia logistica (discriminativ) este fundamentala. LDA modeleaza p(x|y) si aplica Bayes, in timp ce regresia logistica modeleaza direct p(y|x). Ambele produc frontiere liniare, dar cu compromisuri diferite in ceea ce priveste eficienta datelor si robustetea la incalcarea presupunerilor.

Aplicatii Practice

LDA este utilizata in analiza fata (Eigenfaces), clasificarea documentelor, genomica (clasificarea subtipurilor de cancer pe baza expresiei genice) si analiza sentimentului. Regresia logistica multi-clasa este standardul in retelele neuronale profunde.

Exemplu din Lumea Reala

In recunoasterea faciala, LDA (Fisherfaces) proiecteaza imaginile pe un spatiu de dimensiune redusa care maximizeaza separarea intre persoane. Frontiera liniara de decizie este suficienta deoarece variabilitatea intra-clasa (iluminare, expresie) este similara pentru toate persoanele.

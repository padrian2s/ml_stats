Exercitiul 11.2 - Pasul M pentru GMM (Eq. 11.114-11.115): mu_k = suma(r_ik * x_i) / r_k, Sigma_k = suma(r_ik * (x_i - mu_k)(x_i - mu_k)^T) / r_k. Media si covarianta ponderate cu responsabilitatile.
Exercitiul 11.3 - Pasul M pentru amestec de Bernoulli (Eq. 11.116-11.117): ML: mu_kj = suma(r_ik * x_ij) / suma(r_ik). MAP cu prior Beta(alpha, beta): mu_kj = (suma(r_ik * x_ij) + alpha - 1) / (suma(r_ik) + alpha + beta - 2).

Comentariu

Rezumat

Pagina combina rezultate vizuale ale imputarii cu exercitii fundamentale. Formulele pasului M pentru GMM si amestecuri de Bernoulli sunt derivate explicit, aratand ca sunt medii ponderate cu responsabilitatile r_ik. Exercitiile 11.4-11.5 extind la distributii Student t si gradient descent pentru GMM.

Tipare Cheie

Pasul M are intotdeauna aceeasi structura: media ponderata cu responsabilitatile. Pentru ML, formula este direct ponderata; pentru MAP, se adauga pseudo-numarari din prior (Laplace smoothing pentru Bernoulli).

Aplicatii Practice

Formulele pasului M pentru amestecuri de Bernoulli sunt fundamentale in clustering de documente (bag-of-words binar) si in modelarea genomica (date SNP binare). MAP cu prior Beta previne estimari extreme de 0 sau 1.

Exemplu din Lumea Reala

In analiza textelor cu naive Bayes nesupervizat, pasul M calculeaza probabilitatile cuvintelor per topic ca medii ponderate ale frecventelor, exact dupa formula din Exercitiul 11.3.

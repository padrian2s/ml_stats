Selectia Modelului Polinomial si Calculul Verosimilitatii Marginale

Pagina prezinta exemple concrete de selectie bayesiana a modelului cu polinoame de grade diferite si introduce calculul verosimilitatii marginale prin constante de normalizare.

Selectie model cu N=5: Cu doar 5 puncte, modelul MAP este d=1 (liniar). Cu N=30 puncte, modelul corect d=2 (parabolic) este selectat cu probabilitate posterioara mare.
Formula verosimilitatii marginale: p(D) = Z_N / (Z₀ · Z_l) (Ecuatia 5.18), unde Z₀ este constanta de normalizare a prior-ului, Z_l a verosimilitatii si Z_N a posterioarei.
Figura 5.7: (a-c) Polinoame de grade 1, 2 si 3 ajustate la N=5 puncte cu Bayes empiric. (d) Posterioara peste modele p(d|D) favorizeaza d=1 din cauza datelor insuficiente.
Figura 5.8: Acelasi experiment cu N=30 puncte. Acum posterioara favorizeaza puternic d=2 (modelul corect), demonstrand ca mai multe date permit selectia corecta a complexitatii.

Comentariu

Rezumat

Sectiunea 5.3.2 introduce calculul practic al verosimilitatii marginale. Cand avem un prior conjugat, verosimilitatea marginala se calculeaza simplu ca raport de constante de normalizare: p(D) = Z_N/(Z₀·Z_l). Prior-ul este p(θ) = q(θ)/Z₀, verosimilitatea este p(D|θ) = q(D|θ)/Z_l, iar posterioara este p(θ|D) = q(θ|D)/Z_N.

Tipare Cheie

Figurile 5.7 si 5.8 demonstreaza un tipar fundamental: cu date putine, modelul bayesian prefera simplitatea (d=1), iar cu date suficiente, identifica corect complexitatea adevarata (d=2). Aceasta este o manifestare directa a lamei lui Occam bayesiene.

Aplicatii Practice

Calculul verosimilitatii marginale prin constante de normalizare este eficient computational si exact. Este direct aplicabil in toate modelele cu prior conjugat: Beta-Binomial, Dirichlet-Multinomial, Gaussiana-Gaussiana etc.

Exemplu din Lumea Reala

In genomica, selectia modelului bayesian este folosita pentru a determina numarul de clustere de gene co-exprimate. Cu putine esantioane, modelul simplu (2-3 clustere) este preferat; pe masura ce se acumuleaza date, structura fina devine vizibila.

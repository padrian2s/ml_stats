GP si modele liniare generalizate (GLM)

Aceasta pagina introduce extinderea GP la clasificare si alte GLM-uri, prezentand clasificarea binara si tabelul cu verosimilitate, gradient si Hessian pentru regresie logistica si probit.

Clasificare GP binara: p(y_i|x_i) = sigma(y_i * f(x_i)), unde sigma(z) = sigm(z) pentru regresie logistica sau sigma(z) = Phi(z) pentru regresie probit, iar f ~ GP(0, kappa). Priorul Gaussian nu este conjugat cu verosimilitatea Bernoulli/multinoulli.
Aproximari pentru posteriorul ne-conjugat: Se pot folosi: aproximare Gaussiana (Laplace), propagarea expectatiei, metode variatioanale, sau MCMC. Aproximarea Gaussiana este cea mai simpla si rapida.

Comentariu

Rezumat

Extensia GP la GLM-uri necesita tratarea ne-conjugatiei dintre priorul Gaussian si verosimilitatea non-Gaussiana. Tabelul 15.1 sumarizeaza log-verosimilitatea, gradientul si Hessianul pentru cazurile logistic si probit. Distributia predictiva a etichetelor se obtine integrandu-se functia latenta f prin sigma(f).

Tipare Cheie

Problema centrala este ca posteriorul p(f|X,y) nu mai este Gaussian. Aproximarea Laplace este cea mai simpla: gasim modul posteriorului prin IRLS si aproximam cu o Gaussiana centrata pe mod cu covarianta egala cu inversul Hessianului negativ.

Aplicatii Practice

GP pentru clasificare sunt folosite in aplicatii unde incertitudinea predictiei este critica: diagnostic medical, detectie anomalii, si sisteme de decizie. Ofera probabilitati calibrate spre deosebire de SVM-uri.

Exemplu din Lumea Reala

In diagnosticul medical, un GP clasificator poate prezice probabilitatea unei boli pe baza simptomelor, oferind nu doar diagnosticul ci si nivelul de incertitudine, ajutand medicul sa decida daca sunt necesare investigatii suplimentare.

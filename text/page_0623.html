Statistici n-gram si MLE pentru modele de limbaj Markov

Aceasta pagina prezinta vizualizarea statisticilor unigram si bigram si derivarea estimatorului MLE pentru matricea de tranzitie.

MLE pentru lantul Markov: pi_hat_j = N_j^1 / sum N_j^1 (frecventa starii initiale) si A_hat_jk = N_jk / sum_k N_jk (frecventa tranzitiilor normalizata pe rand), unde N_jk este numarul de tranzitii j -> k observate.
Statistici din 'On The Origin Of Species': Figura 17.2 arata diagrama Hinton a distributiei bigram la nivel de caracter. Caracterele cele mai frecvente sunt spatiu, 'e', 't'. Bigramele cele mai frecvente reflecta structura limbii engleze.

Comentariu

Rezumat

MLE-ul pentru lantul Markov este intuitiv: numaram tranzitiile si normalizam. Log-verosimilitatea se descompune in termeni pentru distributia initiala si matricea de tranzitie, permitand estimarea separata. Figura 17.2 vizualizeaza statisticile la nivel de caracter: diagrama Hinton arata distributia comuna a perechilor de caractere consecutive, cu patrate albe mai mari indicand perechi mai frecvente.

Tipare Cheie

MLE este simplu si eficient dar sufera de problema conturilor zero: daca o tranzitie j->k nu este observata, A_hat_jk = 0, ceea ce face ca orice secventa continand aceasta tranzitie sa aiba probabilitate zero. Aceasta este o problema grava pentru modele de limbaj cu vocabulare mari.

Capitolul 1: Filtrare Colaborativa si Analiza Cosului de Cumparaturi

Competitia Netflix Prize, analiza cosului de cumparaturi si diferentele intre data mining si machine learning.

Netflix Prize: Competitie lansata in 2006, cu premiu de 1 milion USD, pentru imbunatatirea predictiei evaluarilor filmelor. Matricea contine ~18.000 filme x ~500.000 utilizatori (~9 x 10‚Åπ intrari), dar doar ~1% sunt observate.
Analiza Cosului de Cumparaturi (Market Basket Analysis): Analiza unei matrice binare rare unde x_ij = 1 daca produsul j a fost cumparat in tranzactia i. Scopul: prezicerea altor produse pe care consumatorul le-ar putea cumpara.
Frequent Itemset Mining: Tehnica comuna pentru analiza cosului de cumparaturi, care creeaza reguli de asociere (de exemplu, "cine cumpara paine si unt va cumpara probabil si lapte").

Data Mining vs. Machine Learning: Data mining pune accent pe modele interpretabile (reguli de asociere), pe cand machine learning pune accent pe modele cu acuratete predictiva ridicata.

Comentariu

Rezumat

Pagina continua discutia despre filtrarea colaborativa cu exemplul Netflix Prize si introduce analiza cosului de cumparaturi. Netflix Prize a demonstrat puterea metodelor de ML pentru sisteme de recomandare. Analiza cosului de cumparaturi identifica corelatii intre produse cumparate impreuna. Se face distinctia intre data mining (modele interpretabile) si ML (modele predictive).

Tipare Cheie

Matricea Netflix este extrem de rara (~1% observata), ceea ce face problema foarte dificila. Analiza cosului de cumparaturi poate fi abordata prin reguli de asociere (interpretabile) sau modele probabilistice de densitate conjuncta (mai precise). Aceeasi tehnica se aplica si la dependentele intre fisiere in sisteme software complexe.

Aplicatii Practice

Sistemele de recomandare bazate pe filtrare colaborativa sunt omniprezente: Netflix, Amazon ("Clientii care au cumparat X au cumparat si Y"), Spotify. Analiza cosului de cumparaturi optimizeaza plasarea produselor in magazine si promotiile.

Exemplu din Lumea Reala

Amazon foloseste analiza cosului de cumparaturi pentru a afisa recomandari "Cumparate frecvent impreuna". Daca un client adauga un telefon in cos, sistemul sugereaza automat o husa si un incarcator, bazandu-se pe tiparele de cumparare ale milioanelor de clienti anteriori.

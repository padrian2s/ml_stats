Teoria deciziei secventiale si exercitii: prag optimal, optiune de respingere

Aceasta pagina introduce teoria deciziei secventiale si prezinta exercitii fundamentale despre pragul optimal de clasificare, optiunea de respingere si problema newsvendor.

Teoria deciziei secventiale: Generalizare a deciziilor one-shot la probleme in mai multi pasi, unde actiunile curente afecteaza starile viitoare. Este strans legata de invatarea prin recompensa (reinforcement learning).
Optiunea de respingere (reject option): In clasificare, cand incertitudinea este prea mare, poate fi optim sa se refuze clasificarea si sa se trimita cazul la un expert uman. Costul respingerii λ_r trebuie sa fie mai mic decat costul clasificarii gresite λ_s.
Pragul optimal de clasificare: Decizia y_hat care minimizeaza pierderea asteptata este echivalenta cu setarea unui prag θ = λ_01 / (λ_01 + λ_10) pe probabilitatea posterioara p_1 = P(Y=1|x). Diferite costuri asimetrice produc praguri diferite de 0.5.

Comentariu

Rezumat

Pagina trece de la decizii punctuale la cele secventiale si prezinta exercitii cheie. Pragul optimal de clasificare depinde de raportul costurilor erorilor, nu este intotdeauna 0.5. Optiunea de respingere este cruciala in practrica: daca costul clasificarii gresite este mult mai mare decat costul consultarii unui expert, este rational sa refuzam clasificarea pentru cazurile incerte.

Tipare Cheie

1) Pragul de decizie θ = λ_01/(λ_01+λ_10) adapteaza clasificatorul la costuri asimetrice. 2) Cu optiune de respingere, exista doua praguri: sub θ_0 se prezice clasa 0, peste θ_1 se prezice clasa 1, intre ele se respinge. 3) Pe masura ce costul respingerii creste relativ la costul erorii, zona de respingere se micsoreaza.

Aplicatii Practice

Pragurile asimetrice sunt esentiale in detectia cancerului (preferabil fals pozitiv decat fals negativ), in detectia fraudelor (costul unei fraude nedetectate >> costul unei tranzactii blocate gresit), si in sistemele de securitate. Optiunea de respingere este folosita in OCR (caractere neclare trimise la verificare umana) si in diagnosticul medical automatizat.

Exemplu din Lumea Reala

Exercitiul 5.4 prezinta un scenariu practic: clasificarea gresita costa $10, dar trimiterea la un operator uman costa doar $3. Daca P(y=1|x) = 0.2, cel mai bine este sa prezicem clasa 0 (pierderea asteptata = 2). Daca P(y=1|x) = 0.4, pierderea minima este respingerea ($3), deoarece atat prezicerea clasei 0 (pierdere 4) cat si clasei 1 (pierdere 6) sunt mai costisitoare.

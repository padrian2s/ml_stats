Teorema Marginalelor si Conditionalelor Gaussiene

Pagina 111 prezinta Teorema 4.3.1, unul dintre cele mai importante rezultate din carte: formulele pentru distributiile marginale si conditionale ale unei distributii Gaussiene multivariate.

Teorema 4.3.1 (Ecuatia 4.67-4.69): Fie x = (x₁, x₂) distribuit Gaussian cu μ = (μ₁, μ₂) si Σ = [[Σ₁₁, Σ₁₂], [Σ₂₁, Σ₂₂]]. Atunci: Marginale: p(x₁) = N(x₁|μ₁, Σ₁₁), p(x₂) = N(x₂|μ₂, Σ₂₂). Conditionala: p(x₁|x₂) = N(x₁|μ₁|₂, Σ₁|₂)
Media conditionala (Ecuatia 4.69): Trei forme echivalente: μ₁|₂ = μ₁ + Σ₁₂ Σ₂₂⁻¹ (x₂ - μ₂) = μ₁ - Λ₁₁⁻¹ Λ₁₂ (x₂ - μ₂) = Σ₁|₂ (Λ₁₁ μ₁ - Λ₁₂ (x₂ - μ₂))
Covarianta conditionala (Ecuatia 4.69): Doua forme echivalente: Σ₁|₂ = Σ₁₁ - Σ₁₂ Σ₂₂⁻¹ Σ₂₁ = Λ₁₁⁻¹
Proprietati cheie: Media conditionala este o functie liniara de x₂. Covarianta conditionala este o constanta (independenta de x₂). Complementul Schur Σ₁₁ - Σ₁₂ Σ₂₂⁻¹ Σ₂₁ apare frecvent in algebra liniara.

Comentariu

Rezumat

Aceasta teorema este fundamentala in statistica si invatarea automata. Marginalizarea unei Gaussiene este triviala (selecteaza submatricea corespunzatoare). Conditionarea necesita mai mult calcul, dar media conditionala ramane liniara in variabila observata, iar covarianta conditionala este constanta - proprietati unice ale distributiei Gaussiene.

Tipare Cheie

Cele trei forme ale mediei conditionate si cele doua forme ale covariantei conditionate sunt echivalente matematic dar utile in circumstante diferite. Forma cu matricea de precizie Λ este adesea mai eficienta computational. Complementul Schur Σ₁₁ - Σ₁₂ Σ₂₂⁻¹ Σ₂₁ este un concept central in algebra liniara.

Aplicatii Practice

Aceste formule sunt baza filtrarii Kalman, a proceselor Gaussiene, a modelelor grafice Gaussiene si a oricarui sistem care necesita actualizarea credintelor dupa observarea unor date noi.

Exemplu din Lumea Reala

In navigatia GPS, pozitia si viteza vehiculului formeaza un vector Gaussian. Cand primim o masurare GPS (observam o parte din stare), aplicam exact aceste formule de conditionare pentru a actualiza estimarea pozitiei. Covarianta conditionala ne spune cat de precisa este estimarea actualizata.

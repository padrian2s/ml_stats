Probabilitatea de acceptare MH si Gibbs ca caz special

Aceasta pagina analizeaza probabilitatea de acceptare Metropolis-Hastings, demonstreaza corectia Hastings si arata ca esantionarea Gibbs este un caz special al MH.

Corectia Hastings: Raportul q(x_t|x')/q(x'|x_t) din probabilitatea de acceptare corecteaza asimetria propunerii. Pentru propuneri simetrice (q(x'|x)=q(x|x')), aceasta se simplifica la alpha = min(1, p*(x')/p*(x_t)), algoritmul Metropolis original.
Gibbs ca MH: Esantionarea Gibbs este un caz special al MH unde propunerea este conditionala completa: q(x'|x) = p(x_i'|x_{-i}). Cu aceasta propunere, probabilitatea de acceptare este intotdeauna 1, deci toate propunerile sunt acceptate.
Balanta detaliata: Corectitudinea MH se bazeaza pe satisfacerea conditiei de balanta detaliata: p(x)*T(x'|x) = p(x')*T(x|x'), unde T este nucleul de tranzitie. Aceasta garanteaza ca p este distributia stationara a lantului Markov.

Comentariu

Rezumat

Pagina stabileste fundamentul teoretic al MH: corectia Hastings asigura balanta detaliata indiferent de propunere. Demonstratia ca Gibbs este un caz special al MH unifica cele doua metode MCMC sub acelasi cadru teoretic.

Tipare Cheie

Unificarea Gibbs-MH este eleganta: in Gibbs, propunerea perfecta (conditionala completa) duce la acceptare 100%. In MH general, propuneri mai simple (dar mai usor de calculat) duc la rate de acceptare mai mici, dar fiecare pas este mai ieftin. Aceasta flexibilitate face MH extrem de versatil.

Aplicatii Practice

Intelegerea Gibbs ca MH permite strategii hibride: in modelele cu multe variabile, unele sunt esantionate cu Gibbs (cand conditionala este tractabila) si altele cu MH (cand nu este). Aceasta combinatie este standard in software-ul MCMC modern.

Exemplu din Lumea Reala

In modelele de ecologie populationala, parametrii de crestere si de interactiune intre specii pot fi esantionati cu Gibbs (daca conditionalele sunt conjugate), iar parametrii de observatie non-standard sunt esantionati cu MH, intr-un singur algoritm hibrid.

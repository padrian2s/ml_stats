Posteriorul NIW, modul posterior si marginalele posterioare

Aceasta pagina prezinta formulele de actualizare pentru posteriorul NIW, modul posterior conjunct si marginalele posterioare pentru media si matricea de covarianta.

Posteriorul NIW: p(μ, Σ | D) = NIW(μ, Σ | m_N, κ_N, ν_N, S_N), unde parametrii actualizati sunt: m_N = (κ_0 m_0 + N x_bar) / κ_N, κ_N = κ_0 + N, ν_N = ν_0 + N.
Marginala posterioara pentru Σ: p(Σ | D) = IW(Σ | S_N, ν_N), cu estimarile MAP si media E[Σ] = S_N / (ν_N - D - 1).
Marginala posterioara pentru μ: p(μ | D) = T(μ | m_N, S_N / (κ_N(ν_N - D + 1)), ν_N - D + 1), o distributie Student multivariata. Aceasta are cozi mai grele decat o Gaussiana, reflectand incertitudinea suplimentara din necunoasterea Σ.

Comentariu

Rezumat

Pagina demonstreaza eleganta conjugarii Bayesiene: posteriorul ramane in familia NIW cu parametri actualizati simplu. Media posterioara m_N este o combinatie convexa intre media prior m_0 si media esantionului x_bar, cu ponderi determinate de κ_0 si N. Matricea S_N include atat priorul S_0, cat si matricea de dispersie empirica, plus un termen suplimentar care reflecta incertitudinea in medie.

Aplicatii Practice

Aceste formule sunt utilizate direct in clasificarea Bayesiana (analiza discriminanta), in algoritmii de clustering Bayesian (Dirichlet Process Mixture Models), si in filtrele Kalman adaptive unde parametrii zgomotului sunt necunoscuti.

Exemplu din Lumea Reala

In sistemele de detectie a anomaliilor in retele de calculatoare, traficul normal este modelat cu o Gaussiana multivariata. Posteriorul NIW permite actualizarea continua a modelului pe masura ce se acumuleaza date, iar marginala Student pentru medie ofera intervale de incredere mai realiste (mai largi) decat cele Gaussiene, evitand alarmele false excesive.

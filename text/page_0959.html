Studiu de caz: Google Rephil

Aceasta pagina prezinta Rephil, un model grafic dirijat masiv invatat automat de Google din date text la scala milioanelor de noduri (Figura 26.12).

Rephil: Un DGM binar cu noisy-OR CPDs. Frunzele (12 milioane) reprezinta prezenta/absenta cuvintelor sau frazelor. Nodurile latente (~1 milion) reprezinta clustere de cuvinte co-ocurente ("concepte"). Modelul are ~350 milioane de legaturi.
Invatarea structurii Rephil: Se bazeaza pe principiul "neurons that fire together wire together". Se ruleaza inferenta si se adauga muchii intre clustere-cuvinte si clustere-clustere care co-activeaza frecvent. Se initializeaza cu un cluster per "document" semantic.
Scala: Modelul a fost antrenat pe ~100 miliarde de fragmente de text, folosind o arhitectura distribuita paralela. Cel mai lung drum in graf are lungimea 555, demonstrand un model foarte profund.

Comentariu

Rezumat

Rephil este un exemplu remarcabil de invatare a structurii la scala industriala. Modelul a fost folosit intern la Google pentru AdSense (potrivirea reclamelor la continut) si alte sisteme. Inferenta exacta este infezabila, dar se exploateaza faptul ca majoritatea frunzelor sunt inactive (cuvintele absente pot fi eliminate analitic). Se foloseste ICM (Iterated Conditional Modes) pentru a gasi moduri locale ale posteriorului.

Tipare Cheie

Scalabilitatea este obtinuta prin exploatarea raritatii (sparsity): majoritatea cuvintelor sunt absente intr-un document dat, iar frunzele inactive pot fi marginalizate eficient. Principiul hebbian ("neurons that fire together wire together") ghideaza invatarea structurii la scala.

Aplicatii Practice

Rephil demonstreaza ca modelele grafice probabilistice pot fi aplicate la scala web-ului. Principii similare sunt folosite in sisteme de recomandare, clasificare de documente, si extragere de informatii la scala mare.

Exemplu din Lumea Reala

Google AdSense foloseste modele similare cu Rephil pentru a intelege "conceptele" din paginile web si a potrivi reclame relevante. De exemplu, o pagina despre "retete de gatit" activeaza concepte latente legate de alimentatie, permitand afisarea de reclame pentru ingrediente sau ustensile de bucatarie.

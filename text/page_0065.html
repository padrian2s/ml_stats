Capitolul 2: Distributiile Binomiala si Bernoulli

Distributii discrete fundamentale: binomiala pentru numararea succeselor si Bernoulli ca caz special.

Distributia Binomiala: Bin(k|n, θ) = C(n,k) θ^k (1-θ)^(n-k) (ecuatia 2.28), unde X este numarul de succese in n incercari independente, fiecare cu probabilitate θ de succes.
Coeficientul Binomial: C(n,k) = n! / ((n-k)!k!) (ecuatia 2.29) - numarul de moduri de a alege k elemente din n. Se pronunta "n alege k".
Distributia Bernoulli: Ber(x|θ) = θ^I(x=1) (1-θ)^I(x=0) (ecuatia 2.31). Caz special al binomialei cu n = 1. Modeleaza un singur experiment binar (succes/esec).
Media si Varianta Binomialei: media = nθ, varianta = nθ(1-θ) (ecuatia 2.30).
Forma Compacta Bernoulli: Ber(x|θ) = θ daca x = 1; 1 - θ daca x = 0 (ecuatia 2.32).

Comentariu

Rezumat

Sectiunea 2.3.1 introduce distributiile binomiala si Bernoulli. Distributia Bernoulli modeleaza un singur experiment binar (aruncare de moneda, spam/non-spam, boala/sanatos). Distributia binomiala modeleaza numarul de succese in n experimente independente. Ambele sunt parametrizate de θ ∈ [0, 1], probabilitatea de succes.

Tipare Cheie

Distributia Bernoulli este blocul fundamental al clasificarii binare (regresia logistica foloseste p(y|x) = Ber(y|sigm(w^T x))). Distributia binomiala apare in testarea statistica (testul binomial) si in modelarea numarului de click-uri din n afisari de reclame. Coeficientul binomial apare frecvent in combinatorica si statistica.

Aplicatii Practice

Bernoulli: modelarea click/non-click pe reclame, succes/esec in teste clinice, spam/ham in filtrarea email-ului. Binomiala: numarul de defecte intr-un lot de produse, numarul de pacienti vindecati dintr-un grup de tratament, numarul de conversii dintr-un set de vizitatori web.

Exemplu din Lumea Reala

Un test A/B pe un site web masoara rata de conversie: din n = 1000 de vizitatori care vad varianta A, k = 50 fac o achizitie. Aceasta se modeleaza ca X ~ Bin(1000, θ), iar scopul este de a estima θ si de a decide daca varianta A este mai buna decat varianta B.

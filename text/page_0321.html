Structura GLM-urilor: functia de legatura, functia medie si functia canonica

Aceasta pagina detaliaza componentele unui GLM: distributia raspunsului, predictorul liniar, functia de legatura si functia de legatura canonica.

Componentele unui GLM: 1) Raspunsul y_i ~ EF(θ_i, σ²), din familia exponentiala. 2) Predictorul liniar η_i = w^T x_i. 3) Functia medie μ_i = g^(-1)(η_i), unde g^(-1) este inversa functiei de legatura. 4) Functia de legatura canonica: g = ψ, astfel incat θ_i = η_i = w^T x_i.
Tabelul 9.1 - Functii de legatura canonice: Gaussiana: g(μ) = μ (identitate), μ = θ. Binomiala: g(μ) = log(μ/(1-μ)) (logit), μ = sigm(θ). Poisson: g(μ) = log(μ) (log), μ = e^θ.
Alegerea functiei de legatura: Functia de legatura canonica simplifica formulele (gradientul devine Σ(y_i - μ_i)x_i pentru toate GLM-urile), dar nu este obligatorie. Orice functie monotona invertibila care mapeaza [-∞, +∞] la domeniul mediei este valida.

Comentariu

Rezumat

Pagina disecteaza structura unui GLM in componentele sale. Diagrama din Figura 9.1 arata fluxul: w si x_i produc predictorul liniar η_i, care trece prin inversa functiei de legatura g^(-1) pentru a produce media μ_i, care la randul sau determina parametrul natural θ_i prin functia ψ. Cu functia de legatura canonica (g = ψ), θ_i = η_i = w^T x_i, simplificand dramatic formulele.

Tipare Cheie

1) GLM = distributie + predictor liniar + legatura. 2) Functia de legatura canonica face ca θ_i = w^T x_i, simplificand gradientul. 3) Media raspunsului este A'(θ_i), varianta este A''(θ_i)σ² - ambele derivate din functia log-partitie. 4) Parametrul de dispersie σ² este adesea 1 (Bernoulli, Poisson) sau necunoscut (Gaussiana).

Aplicatii Practice

Intelegerea structurii GLM permite alegerea informata a modelului: daca raspunsul este binar, folosim logit; daca sunt contorizari, folosim log; daca sunt proportii, folosim logit sau probit. Tabelul 9.1 este un ghid practic de referinta folosit zilnic de statisticieni si data scientists.

Exemplu din Lumea Reala

In asigurari, numarul de reclamatii se modeleaza cu GLM Poisson (legatura log), iar valoarea fiecarei reclamatii cu GLM Gamma (legatura log sau inversa). Combinand cele doua, se obtine modelul complet: costul total asteptat = frecventa x severitate medie. Acesta este cadrul standard in actuariat, formalizat prin regulamentele Solvency II ale UE.

9.6 Modele Liniare Mixte Generalizate - Figuri

Pagina contine Figura 9.2 care ilustreaza modelul grafic directionat pentru GLMM si rezultatele experimentale pe date medicale privind densitatea minerala osoasa.

Figura 9.2(a): Model grafic directionat pentru modelul liniar mixt generalizat cu J grupuri. Nodurile includ: μ_β si σ²_β (hiperparametrii priorului pentru efectele aleatoare β_j), μ_α si σ²_α (hiperparametrii pentru efectele fixe α), σ²_y (varianta zgomotului), x_j (trasaturi la nivel de grup), x_ij (trasaturi la nivel individual) si y_ij (raspunsul observat). Placa exterioara indexeaza J grupuri, iar placa interioara indexeaza N_j observatii per grup.
Figura 9.2(b): Densitatea minerala osoasa spinala (SBMD) in functie de varsta pentru patru grupuri etnice: Hispanic, White (Alb), Asian (Asiatic) si Black (Negru). Datele brute sunt afisate cu puncte gri deschis. Modelul ajustat este afisat cu linii negre: linia continua reprezinta media predictiva posterioara, iar linia punctata reprezinta varianta predictiva posterioara. Se observa relatii non-liniare diferite intre varsta si SBMD pentru fiecare grup etnic.

Comentariu

Rezumat

Figura 9.2 ofera o vizualizare clara a modelului GLMM. Partea (a) prezinta structura grafica ierarhica cu doua niveluri de variabile latente (efecte aleatoare β_j si efecte fixe α), iar partea (b) demonstreaza aplicarea practica pe date reale de densitate minerala osoasa, aratand cum modelul captureaza atat tendintele non-liniare cat si diferentele intre grupuri etnice.

Tipare Cheie

Structura de placi (plate notation) din modelul grafic arata clar ierarhia: un nivel exterior pentru grupuri si un nivel interior pentru observatii individuale. Hiperparametrii (μ_β, σ²_β, μ_α, σ²_α) controleaza distributia priorului la nivel global. Graficele din partea (b) arata ca relatia SBMD-varsta este non-liniara si difera intre grupurile etnice.

Aplicatii Practice

Modelele grafice directionate sunt instrumente esentiale pentru vizualizarea si comunicarea structurii modelelor statistice complexe. In medicina, analiza SBMD este importanta pentru diagnosticarea osteoporozei si evaluarea riscului de fracturi.

Exemplu din Lumea Reala

Un spital poate utiliza acest model pentru a prezice riscul de osteoporoza al pacientilor, luand in considerare atat varsta (efect non-liniar) cat si etnia (efect categoric), cu ajustari personalizate pentru fiecare pacient (efect aleator β_j), obtinand predictii mai precise decat un model unic pentru toti pacientii.

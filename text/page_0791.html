Capitolul 21. Inferenta Variationala - Algoritmul Complet si Limita Sigmoid (p.760)

Aceasta pagina prezinta formulele finale de actualizare, Algoritmul 21.1 (pseudocod complet), si introduce limitele pentru functia sigmoid (cazul binar).

Formulele de actualizare simplificate: Vₙ = (V₀ + A ⊗ Σᵢ xᵢxᵢᵀ)⁻¹ si mₙ = Vₙ(V₀⁻¹m₀ + Σᵢ(yᵢ+bᵢ) ⊗ xᵢ), exploatand structura bloc-diagonala a Xᵢ si constanta matricei A.
Algoritmul 21.1: Inferenta variationala pentru regresie logistica multi-clasa cu limita Bohning. Bucla: (1) calculeaza ψ = Xmₙ, (2) G = exp(Ψ-lse(Ψ)), (3) B = AΨ-G, (4) actualizeaza mₙ = Vₙ(V₀⁻¹m₀ + Xᵀ(y+b)), (5) calculeaza L_QJ, repeta pana la convergenta.
Cazul binar (sigmoid): Cand yᵢ ∈ {0,1}, M=1, verosimilitatea devine log(1+eᶯ), iar limita Bohning se simplifica cu a = 1/4 (curbura constanta).

Comentariu

Rezumat

Pagina prezinta algoritmul complet in forma de pseudocod, exploatand proprietatile structurale (produsul Kronecker, constanta A) pentru eficienta. Se introduce apoi cazul binar, mai simplu, unde limita Bohning are curbura constanta a = 1/4.

Tipare Cheie

Exploatarea produsului Kronecker (⊗) reduce semnificativ costul computational. Matricea Vₙ se calculeaza o singura data si se reutilizeaza. In cazul binar, a = 1/4 este curbura maxima a functiei logistice, atinsa la η = 0.

Aplicatii Practice

Algoritmul 21.1 este implementabil direct in orice limbaj de programare. Produsul Kronecker permite scalare eficienta la probleme cu multe clase. Cazul binar este cel mai frecvent in practica.

Exemplu din Lumea Reala

In clasificarea imaginilor medicale (ex: benign vs malign), algoritmul binar cu a = 1/4 ofera o solutie eficienta si exacta pentru inferenta bayesiana, cu incertitudine calibrata in predictii.

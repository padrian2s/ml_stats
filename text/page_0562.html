Conexiunea GP cu modele liniare si alte metode

Aceasta pagina compara GP cu regresia liniara Bayesiana, aratand ca aceasta din urma este echivalenta cu un GP cu functie de covarianta degenerata.

Functie de covarianta degenerata: kappa(x, x') = x^T Sigma x', obtinuta din regresia liniara Bayesiana cu prior p(w) = N(0, Sigma). Are cel mult D valori proprii nenule, limitand flexibilitatea modelului.
Echivalenta regresie liniara - GP: Regresia liniara Bayesiana cu K = X Sigma X^T este echivalenta cu un GP avand covarianta kappa(x,x') = x^T Sigma x'. Aceasta permite 'kernelizarea' prin inlocuirea produsului scalar cu o functie kernel arbitrara.

Comentariu

Rezumat

Sectiunea 15.4 stabileste conexiuni intre GP si alte metode. Regresia liniara Bayesiana este un caz particular de GP cu kernel degenerat. Trasaturile apar doar in combinatii de tip X Sigma X^T, x_*^T Sigma x_* sau x_*^T Sigma X^T, ceea ce permite kernelizarea directa. Totusi, kernelul degenerat limiteaza expresivitatea si poate produce supraincredere.

Tipare Cheie

Diferenta fundamentala intre GP si modelele liniare: kernelul GP poate fi non-degenerat (rang infinit), permitand reprezentarea oricarei functii din spatiul RKHS. Modelul liniar, chiar si kernelizat, este limitat de dimensionalitatea D a spatiului trasaturilor.

Aplicatii Practice

Aceasta echivalenta justifica teoretic utilizarea kernel trick-ului in SVM si alte metode kernel. In practica, GP cu kerneluri non-degenerate (SE, Matern) sunt preferabile modelelor liniare kernelizate cand flexibilitatea si calibrarea incertitudinii sunt importante.

Exemplu din Lumea Reala

In analiza sentimentului textelor, un model liniar cu kernel bag-of-words (degenerat) poate fi comparat cu un GP cu kernel de similaritate semantica, unde GP-ul captureaza relatii non-liniare intre cuvinte pe care modelul liniar le pierde.

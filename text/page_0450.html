Exercitiul 12.11 - PPCA vs FA (detalii): Cu x_1 = z, x_2 = z + 0.001*z_2, x_3 = 10*z_3 si z ~ N(0,I): PCA gaseste directia de varianta maxima (x_3, care are varianta 100), dar FA gaseste factorul latent comun (z, partajat intre x_1 si x_2).
Distinctia fundamentala PCA vs FA: PCA = maximizarea variantei totale (incluzand zgomot). FA = identificarea variantei partajate (factori comuni), separand-o de varianta specifica (zgomot unic per variabila).

Comentariu

Rezumat

Ultimele exercitii ale Capitolului 12 consolideaza distinctia practica dintre PCA si FA. FA este superioara cand obiectivul este identificarea factorilor latenti comuni (varianta partajata), pe cand PCA este adecvata cand obiectivul este reducerea dimensionalitatii pastrand varianta totala maxima.

Tipare Cheie

Alegerea intre PCA si FA depinde de obiectiv: pentru compresie/reconstructie, PCA este optima. Pentru descoperirea structurii latente si interpretare, FA este preferabila. PPCA este un compromis care ofera avantajele probabilistice ale FA cu simplitatea PCA.

Aplicatii Practice

In psihometrie si stiinte sociale, FA este standardul deoarece obiectivul este identificarea factorilor latenti (inteligenta, personalitate). In inginerie si viziune computationala, PCA este preferata pentru compresie si denoising.

Exemplu din Lumea Reala

In evaluarea riscului de credit, FA identifica factorii de risc comuni (macroeconomici) separati de riscul specific fiecarui debitor, permitand diversificarea corecta a portofoliului.

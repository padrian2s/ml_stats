RBPF - Algoritmul complet si propunerea optima look-ahead

Aceasta pagina prezinta algoritmii 23.2 si 23.3 pentru RBPF cu SLDS, inclusiv varianta cu propunere optimala look-ahead.

Algoritmul 23.2 - RBPF cu propunere prior: (1) Esantioneaza q_t din prior p(q_t|q_{t-1}^s); (2) Actualizeaza filtrul Kalman: (mu_t^s, Sigma_t^s, L_{tk}^s) = KFupdate(mu_{t-1}^s, Sigma_{t-1}^s, y_t, theta_k); (3) Ponderea: w_t^s = w_{t-1}^s * L_{ts}^s.
Algoritmul 23.3 - RBPF look-ahead: Se trece fiecare particula prin toate cele K modele, calculand verosimilitatile predictive L_{tk}^s. Ponderile se calculeaza inainte de esantionare: w_t^s = w_{t-1}^s * sum_k L_{tk}^s * p(q_t=k|q_{t-1}^s). Apoi se re-esantioneaza si se esantioneaza q_t din propunerea optima.
Propunerea optima: p(q_t=k|q_{1:t-1}^s, y_{1:t}) proportional cu L_{tk}^s * p(q_t=k|q_{t-1}^s). Aceasta incorporeaza observatia curenta y_t prin verosimilitatile predictive, imbunatatind semnificativ eficienta particulelor.

Comentariu

Rezumat

Pagina prezinta doua variante ale RBPF: cea simpla (propunere prior) si cea cu look-ahead (propunere optimala). Look-ahead RBPF calculeaza ponderile inainte de esantionare, permitand re-esantionarea bazata pe informatia cea mai recenta si esantionarea starii discrete din propunerea optima.

Tipare Cheie

Look-ahead RBPF exploateaza faptul ca spatiul de stari discrete este mic (K), deci evaluarea tuturor alternativelor este fezabila. Costul O(K*S) este modest dar ofera o imbunatatire semnificativa.

Aplicatii Practice

RBPF cu look-ahead este folosit in urmarirea tintelor cu manevre (avioane, rachete), unde starea discreta indica tipul de manevra (zbor drept, viraj, accelerare).

Exemplu din Lumea Reala

In sistemele de aparare aeriana, RBPF urmareste avioane care pot efectua manevre evasive: starea discreta indica tipul de manevra (viraj stanga, viraj dreapta, accelerare), iar filtrul Kalman estimeaza pozitia si viteza in cadrul fiecarui mod de zbor.

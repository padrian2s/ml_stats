Vizualizarea Distributiei Wishart si Inferenta Parametrilor MVN

Pagina 127 discuta cum sa vizualizam distributia Wishart (care este o distributie peste matrice) si introduce Sectiunea 4.6 despre inferenta parametrilor unei Gaussiene multivariate.

Figura 4.16: Vizualizarea distributiei Wishart Σ ~ Wi(S, ν) cu S = [[3.17, -0.03], [-0.03, 0.65]] si ν = 3. Stanga: esantioane sub forma de elipse (vectori proprii definesc orientarea, valori proprii definesc marimea). Cu ν mic, matricile esantionate sunt foarte variabile si unele aproape singulare. Dreapta: marginalele diagonalei (distributii Gamma) si marginala coeficientului de corelatie ρ (aproape uniforma cand ν = 3).
Estimarea coeficientului de corelatie asteptat (Ecuatia 4.169): E[R_ij] ≈ (1/S) Σₛ R(Σ^(s))_ij, unde Σ^(s) ~ Wi(Σ, ν) si R_ij = Σ_ij / √(Σ_ii Σ_jj).
Inferenta parametrilor MVN (Sectiunea 4.6): Data xᵢ ~ N(μ, Σ), se doreste inferenta p(μ, Σ|D). Se procedeaza in trei pasi: (1) p(μ|D, Σ), (2) p(Σ|D, μ), (3) p(μ, Σ|D) comun.

Comentariu

Rezumat

Vizualizarea distributiei Wishart este provocatoare deoarece este o distributie peste matrice. Solutia eleganta in 2D este sa reprezentam fiecare matrice esantionata ca o elipsa. Cu ν mic (putina informatie a priori), elipsele variaza enorm, reflectand incertitudinea mare. Sectiunea 4.6 trece la problema practica de a estima Bayesian parametrii unei Gaussiene.

Tipare Cheie

Marginala coeficientului de corelatie ρ sub Wishart cu ν mic este aproape uniforma pe [-1, 1], indicand ca avem foarte putina informatie a priori despre corelatie. Pe masura ce ν creste, distributia se concentreaza in jurul valorii implicate de S.

Aplicatii Practice

Vizualizarea incertitudinii asupra covariantei este importanta in analiza riscului financiar (incertitudinea matricei de corelatie intre active), in meta-analiza medicala (variabilitatea intre studii) si in orice problema unde covarianta trebuie estimata cu date limitate.

Exemplu din Lumea Reala

In managementul riscului bancar, incertitudinea asupra corelatiei intre defaulturi ale companiilor este critica. Cu doar 10 ani de date (ν ≈ 10), distributia posterioara a corelatiei este larga, indicand ca estimarile de risc bazate pe o singura valoare a corelatiei pot fi inselatoare.

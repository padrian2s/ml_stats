Definitia familiei exponentiale si exemplul Bernoulli

Aceasta pagina defineste formal familia exponentiala si demonstreaza cum distributia Bernoulli se exprima in aceasta forma.

Forma familiei exponentiale: p(x|θ) = h(x) exp[θ^T φ(x) - A(θ)], unde θ sunt parametrii naturali (canonici), φ(x) sunt statisticile suficiente, A(θ) = log Z(θ) este functia log-partitie (functia cumulantelor), si h(x) este constanta de scalare.
Familia exponentiala curbata: Daca dim(θ) < dim(η(θ)), unde η este functia care mapeaza parametrii la parametrii canonici, familia este „curbata" si are mai multe statistici suficiente decat parametri.
Bernoulli in forma exponentiala: Ber(x|μ) = (1-μ) exp[x log(μ/(1-μ))], unde θ = log(μ/(1-μ)) este log-odds ratio, φ(x) = x, si Z = 1/(1-μ). Parametrul natural θ este legat de media μ prin functia sigmoida: μ = sigm(θ) = 1/(1+e^(-θ)).

Comentariu

Rezumat

Pagina stabileste cadrul matematic al familiei exponentiale. Definitia formala identifica patru componente: parametrii naturali θ, statisticile suficiente φ(x), functia log-partitie A(θ), si masura de baza h(x). Exemplul Bernoulli ilustreaza cum o distributie familiara se rescrie in aceasta forma, revelland ca log-odds ratio este parametrul natural si ca functia sigmoida face legatura cu probabilitatea.

Tipare Cheie

1) Reprezentarea minimala vs. supra-completa: encodarea one-hot a Bernoulli este supra-completa (doua statistici pentru un parametru). 2) Parametrii naturali θ traiesc intr-un spatiu nerestrictional (toata axa reala), spre deosebire de μ care trebuie sa fie in [0,1]. 3) Functia log-partitie A(θ) joaca un rol central - din derivatele sale se obtin media si varianta.

Aplicatii Practice

Log-odds ratio este fundamental in regresia logistica (coeficientii sunt interpretati ca log-odds), in meta-analiza (combinarea studiilor), si in epidemiologie (odds ratio ca masura a efectului). Forma exponentiala permite implementarea unificata a algoritmilor de estimare.

Exemplu din Lumea Reala

Cand un medic spune ca „riscul de boala este de 3 ori mai mare pentru fumatori", aceasta este un odds ratio. In cadrul familiei exponentiale, acest factor 3 se traduce direct in parametrul natural θ: log(3) = 1.1 este diferenta in log-odds intre fumatori si nefumatori. Regresia logistica estimeaza exact acesti parametri naturali.

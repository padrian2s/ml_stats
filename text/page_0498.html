Algoritmi ARD: Actualizari EM, Punct Fix si Reweighted l1

Pagina continua cu actualizarile EM pentru alpha si beta, introduce algoritmul cu punct fix (mai rapid) si algoritmul iterativ reweighted l1 pentru ARD.

Actualizari EM simplificate (Eq. 13.167-13.168): Cand alpha_j = alpha (partajat) si a=b=0: alpha = D / (mu^T mu + tr(Sigma)). Actualizare beta: beta_new^{-1} = (||y - X mu||^2 + beta^{-1} suma (1 - alpha_j Sigma_jj) + 2d) / (N + 2c).
Algoritmul cu punct fix (Sectiunea 13.7.4.2, Eq. 13.169-13.171): Optimizarea directa a obiectivului (Eq. 13.155): alpha_j <- (gamma_j + 2a) / (m_j^2 + 2b), beta^{-1} <- (||y - X mu||^2 + 2d) / (N - suma gamma_j + 2c), unde gamma_j = 1 - alpha_j Sigma_jj masoara cat de bine este determinat w_j de date.
Algoritmul iterativ reweighted l1 (Sectiunea 13.7.4.3, Eq. 13.172): ARD reformulat ca problema MAP: w^{t+1} = argmin NLL(w) + suma lambda_j^{(t)} |w_j|. La fiecare iteratie, penalitatile lambda_j sunt actualizate in functie de ponderile curente, producand regularizare adaptiva.

Comentariu

Rezumat

Pagina prezinta trei algoritmi pentru ARD cu compromisuri diferite: EM (simplu, convergenta lenta), punct fix (mai rapid, echivalent la convergenta), si reweighted l1 (reformulare ca secventa de probleme LASSO). Cantitatea gamma_j (grade efective de libertate) este centrala: masoara influenta datelor vs. priorului pentru fiecare pondere.

Tipare Cheie

Gradele efective de libertate gamma = suma gamma_j sunt analogul Bayesian al numarului de parametri. Cand alpha_j -> infinit, gamma_j -> 0 (parametru eliminat). Cand alpha_j -> 0, gamma_j -> 1 (parametru complet determinat de date). Aceasta masura apare si in teoria informatiei (complexitate stochastic).

Aplicatii Practice

Algoritmul cu punct fix converge mai rapid decat EM si este recomandat in practica. Algoritmul reweighted l1 permite reutilizarea solverilor LASSO existenti (glmnet, FISTA) ca sub-rutine, facilitand implementarea ARD in framework-uri existente.

Exemplu din Lumea Reala

In compressed sensing pentru imagistica medicala (MRI accelerat), algoritmul reweighted l1 imbunatateste reconstructia imaginilor fata de LASSO standard, deoarece reduce bias-ul pe coeficientii mari (semnalul real) prin penalitati adaptive.

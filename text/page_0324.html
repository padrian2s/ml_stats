Inferenta Bayesiana pentru GLM-uri si regresia Probit

Aceasta pagina discuta inferenta Bayesiana pentru GLM-uri (necesitand metode MCMC sau aproximari) si introduce regresia probit ca alternativa la regresia logistica.

Inferenta Bayesiana pentru GLM: Nu exista posterior in forma inchisa (priorul Gaussian pe w nu este conjugat cu verosimilitatea GLM). Se folosesc: Metropolis-Hastings, Gibbs sampling cu ARS, aproximare Gaussiana (Laplace), sau inferenta variationala.
Regresia probit: p(y=1|x, w) = Φ(w^T x), unde Φ este CDF-ul distributiei normale standard. Functia probit este foarte similara cu sigmoida, dar are cozi putin mai subtiri.
Tabelul 9.2 - Functii medie pentru clasificare binara: Logistica: sigm(η), Probit: Φ(η), Log-log: exp(-exp(-η)), Complementar log-log: 1-exp(-exp(η)). Functiile asimetrice (log-log) sunt utile cand clasele au prevalente foarte diferite.
Fisher scoring vs. Newton: Pentru legaturi non-canonice, Hessianul observat difera de cel asteptat. Metoda Fisher scoring (care foloseste Hessianul asteptat) poate fi mai stabila numeric, si se numeste IRLS generalizat.

Comentariu

Rezumat

Pagina trateaza doua subiecte: limitarile inferentei Bayesiene exacte pentru GLM-uri (necesitand metode aproximative) si regresia probit. Spre deosebire de regresia liniara Bayesiana (posterior in forma inchisa), GLM-urile necesita MCMC sau aproximari deoarece posteriorull nu este Gaussian. Regresia probit este prezentata ca alternativa la logistica, cu avantaje in interpretarea ca model cu variabile latente.

Tipare Cheie

1) Inferenta Bayesiana exacta pentru GLM necesita metode computationale (MCMC). 2) Aproximarea Laplace (posterior ~ Gaussiana centrata la MAP) este rapida dar poate fi imprecisa. 3) Probit si logit sunt aproape indistinguibile in practica pentru majoritatea datelor. 4) Functiile de legatura asimetrice sunt utile cand una din clase este foarte rara.

Aplicatii Practice

Regresia probit este standard in econometria aplicata (economia muncii, economia sanatatii) si in psihometrie. Inferenta Bayesiana pentru GLM-uri via MCMC este implementata in Stan, JAGS, si PyMC3, si este folosita in studiile clinice Bayesiene si in analiza sondajelor (MRP - Multilevel Regression and Poststratification).

Exemplu din Lumea Reala

In econometria muncii, modelul probit este folosit pentru a modela decizia de participare pe piata muncii (angajat vs. neangajat) in functie de educatie, varsta, si caracteristici familiale. Interpretarea probit ca model cu utilitate latenta (utilitatea muncii vs. utilitatea inactivitatii) este naturala in teoria economica, oferind o justificare structurala pentru forma functionala.

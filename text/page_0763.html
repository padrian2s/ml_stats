Inferenta variationala: derivarea obiectivului

Aceasta pagina deriveaza obiectivul inferentei variationale: minimizarea divergentei KL inverse intre q(x) si posterioara nenormalizata, echivalenta cu maximizarea ELBO.

KL directa vs KL inversa: KL(p*||q) necesita asteptari sub p* (intractabila). KL(q||p*) necesita asteptari sub q (tractabila daca q este simpla). De aceea se minimizeaza KL inversa.
Energia libera variationala: J(q) = KL(q||p̃) = KL(q||p*) - log Z, unde p̃ este posterioara nenormalizata. Minimizarea lui J(q) minimizeaza KL(q||p*) deoarece log Z este o constanta.
ELBO (Evidence Lower Bound): L(q) = -J(q) = -KL(q||p*) + log Z ≤ log p(D). ELBO este o margine inferioara a log-verosimilitutii evidentei. Maximizarea ELBO este echivalenta cu minimizarea KL(q||p*) si este strans legata de algoritmul EM.

Comentariu

Rezumat

Derivarea obiectivului variational procede in pasi: (1) dorim sa minimizam KL(q||p*) dar p* contine Z intractabil, (2) inlocuim p* cu p̃ = p*Z (nenormalizata, tractabila), obtinand J(q) = KL(q||p̃), (3) demonstram ca J(q) = KL(q||p*) - log Z, deci minimizarea lui J echivaleaza cu minimizarea KL, (4) L(q) = -J(q) este ELBO, o margine inferioara a log-evidentei.

Tipare Cheie

ELBO are doua interpretari: (1) -KL(q||p*) + log Z si (2) E_q[log p̃(x)] + H(q). Prima interpretare arata ca ELBO este tight cand q = p*. A doua arata ca ELBO echilibreaza potrivirea datelor (energia) cu complexitatea modelului (entropia).

Aplicatii Practice

ELBO este obiectivul standard in auto-encodere variationale (VAE), LDA, si inferenta variationala stocastica. Monitorizarea ELBO permite verificarea convergentei algoritmului variational.

Exemplu din Lumea Reala

In antrenarea unui VAE pe imagini, ELBO combina eroarea de reconstructie (cat de bine decodorul reconstruieste imaginea) cu un termen de regularizare KL (cat de aproape este posterioara latenta de prior). Maximizarea ELBO produce o reprezentare latenta utila.

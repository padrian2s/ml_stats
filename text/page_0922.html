Clustering spectral - Laplacianul grafului

Aceasta pagina prezinta formularea matematica a clustering-ului spectral, incluzand taietura normalizata, Laplacianul grafului si teorema care conecteaza componentele conexe cu vectorii proprii.

Laplacianul grafului: L = D - W, unde D = diag(d_i) este matricea diagonala a gradelor ponderate si W este matricea de pondere. L este simetrica si pozitiv semi-definita.
Forma patratica a Laplacianului: f^T L f = (1/2) * suma w_{ij}(f_i - f_j)^2, masoara cat de "neteda" este functia f pe graf - valori mici inseamna ca f variaza putin intre noduri conectate.
Teorema 25.4.1: Multimea vectorilor proprii ai lui L cu valoare proprie 0 este generata de vectorii indicator 1_{A_1},...,1_{A_K}, unde A_k sunt componentele conexe ale grafului. Numarul de valori proprii zero egaleaza numarul de componente conexe.

Comentariu

Rezumat

Pagina construieste fundatia matematica a clustering-ului spectral. Laplacianul grafului L = D - W este cheia: forma sa patratica masoara netezimea unei functii pe graf, iar vectorii proprii cu valori proprii mici indica partitii naturale. Teorema 25.4.1 face legatura formala: daca graful are K componente conexe, exista exact K vectori proprii cu valoare proprie 0, fiecare indicand o componenta.

Tipare Cheie

Laplacianul grafului conecteaza algebra liniara cu teoria grafurilor. Pozitivitatea semi-definita garanteaza valori proprii non-negative. Vectorii proprii corespunzatori celor mai mici valori proprii (nu cele mai mari, ca in PCA) sunt cei relevanti pentru clustering.

Aplicatii Practice

Laplacianul grafului este folosit nu doar in clustering, ci si in procesarea semnalelor pe grafuri, invatarea semi-supervizata (regularizarea pe graf), si generarea de retele (mesh) in grafica computationala si elemente finite.

Exemplu din Lumea Reala

In analiza retelelor de transport (rutier, feroviar), Laplacianul grafului identifica componentele slab conectate (de exemplu, regiuni separate de un rau fara poduri), informand decizii de infrastructura si planificare urbana.

Regularizarea l1: Priorul Laplace si LASSO

Pagina introduce formal regularizarea l1 ca estimare MAP cu prior Laplace, LASSO (Least Absolute Shrinkage and Selection Operator) si Basis Pursuit Denoising (BPDN).

Figura 13.3: Comparatie geometrica l1 vs l2. Stanga: "diamantul" l1 intersecteaza elipsele RSS in colturile axelor (solutii sparse). Dreapta: cercul l2 intersecteaza elipsele in puncte generice (solutii ne-sparse).
Priorul Laplace (Eq. 13.32): p(w|lambda) = prod Lap(w_j|0, 1/lambda) proportional cu prod exp(-lambda|w_j|). Estimarea MAP cu acest prior produce penalizarea l1.
Obiectivul LASSO/BPDN (Eq. 13.33-13.36): f(w) = NLL(w) + lambda||w||_1. Pentru regresie liniara: f(w) = RSS(w) + lambda'||w||_1. Forma constranata echivalenta: min RSS(w) s.t. ||w||_1 <= B.

Comentariu

Rezumat

Regularizarea l1 inlocuieste penalizarea l0 (ne-convexa, NP-hard) cu norma l1 (convexa, solvabila eficient). Intuitia geometrica din Figura 13.3 este fundamentala: "colturile" diamantului l1 sunt pe axele de coordonate, corespunzand solutiilor cu componente exact zero. Cercul l2 nu are colturi, deci nu produce sparsitate.

Tipare Cheie

LASSO este un compromis elegant: combinatia unui obiectiv convex (RSS) cu o penalitate convexa (l1) produce o problema de optimizare convexa cu solutii sparse. Este singurul Lp cu p >= 1 care produce sparsitate (p < 1 produce si mai multa sparsitate dar nu este convex).

Aplicatii Practice

LASSO este probabil cea mai utilizata metoda de selectie automata de variabile in stiinta datelor. Implementari eficiente (glmnet, scikit-learn) permit aplicarea pe milioane de variabile.

Exemplu din Lumea Reala

In genomica, LASSO pe date de expresie genica (D=20000 gene, N=100 pacienti) identifica un set mic de gene biomarker predictive pentru raspunsul la tratament, producand un test diagnostic simplu si ieftin.

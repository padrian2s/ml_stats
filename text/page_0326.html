9.4 Regresia Probit (continuare)

Pagina continua discutia despre regresia probit, prezentand regresia probit ordinala si modelele probit multinomiale.

Regresie probit ordinala: Extensie a regresiei probit pentru variabile de raspuns ordinale, unde variabila poate lua C valori discrete ordonate (de exemplu: scazut, mediu, ridicat). Se introduc C + 1 praguri γ_j.
Model probit multinomial: Model pentru variabile de raspuns categorice neordonate, unde y_i ∈ {1, ..., C}. Se defineste z_ic = w^T x_ic + ε_ic, cu ε ~ N(0, R), si y_i = argmax_c z_ic.
Formula regresiei ordinale (9.106): y_i = j daca γ_{j-1} < z_i ≤ γ_j, unde γ_0 ≤ ... ≤ γ_C, cu γ_0 = -∞, γ_1 = 0 si γ_C = ∞.
Modelul probit multinomial (9.107-9.109): z_ic = w^T x_ic + ε_ic, ε ~ N(0, R), y_i = argmax_c z_ic.
Probit multivariat: Varianta in care y_ic = I(z_ic > 0), modeland C rezultate binare corelate, cu R ca matrice de corelatie.

Comentariu

Rezumat

Pagina prezinta doua extensii importante ale regresiei probit: regresia probit ordinala (pentru variabile de raspuns ordonate) si modelul probit multinomial (pentru variabile categorice neordonate). Regresia ordinala foloseste praguri pe axa reala pentru a partitiona spatiul in intervale corespunzatoare claselor, iar modelul multinomial foloseste variabile latente gaussiene cu selectia prin argmax.

Tipare Cheie

Ambele modele se bazeaza pe variabile latente gaussiene, extinzand cadrul probit binar. Regresia ordinala pastreaza ordinea naturala a claselor prin constrangeri pe praguri, iar modelul multinomial trateaza clasele ca neordonate. Legatura cu regresia logistica multinomiala apare cand se foloseste distributia Gumbel in loc de gaussiana.

Aplicatii Practice

Regresia probit ordinala este utila in sondaje (de exemplu, scale Likert: total dezacord, dezacord, neutru, acord, total acord). Modelul probit multinomial se aplica in analiza alegerii discrete, marketing si economie comportamentala.

Exemplu din Lumea Reala

Un magazin online poate folosi regresia probit ordinala pentru a prezice nivelul de satisfactie al clientilor (1-5 stele) pe baza caracteristicilor comenzii, sau modelul probit multinomial pentru a prezice ce marca de produs va alege un client dintr-un set de optiuni.

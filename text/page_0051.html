Capitolul 1: Regresia Liniara si Expansiunea in Functii de Baza

Modelul de regresie cu zgomot Gaussian, expansiunea in functii de baza si regresia polinomiala.

Distributia Gaussiana (Normala): Distributie de probabilitate cu forma de clopot, notata N(μ, σ²), unde μ este media si σ² este varianta. Eroarea reziduala ε este adesea modelata ca Gaussiana.
Model de Densitate Conditionata: p(y|x, θ) = N(y|μ(x), σ²(x)) - modelul probabilistic care exprima explicit ca raspunsul y urmeaza o distributie Gaussiana conditionata de intrarea x (ecuatia 1.5).
Expansiune in Functii de Baza (Basis Function Expansion): Inlocuirea intrarii x cu o transformare neliniara φ(x), permitand modelarea relatiilor neliniare: p(y|x, θ) = N(y|w^T φ(x), σ²) (ecuatia 1.7).
Regresie Polinomiala: Caz special al expansiunii in functii de baza unde φ(x) = [1, x, x², ..., x^d], cu d gradul polinomului.
Figura 1.18: Polinoame de grad 14 si 20 ajustate prin metoda celor mai mici patrate la 21 puncte de date. Polinomul de grad 20 prezinta oscilatii extreme (overfitting).
Formule Cheie: μ(x) = w₀ + w₁x = w^T x (ecuatia 1.6); p(y|x, θ) = N(y|w^T φ(x), σ²) (ecuatia 1.7), unde φ(x) = [1, x, x², ..., x^d].

Comentariu

Rezumat

Pagina dezvolta modelul de regresie liniara, aratand ca eroarea reziduala Gaussiana conduce la modelul p(y|x,θ) = N(y|μ(x), σ²). Se introduce termenul de interceptare w₀ (bias) si panta w₁. Pentru relatii neliniare, se inlocuieste x cu φ(x) - o transformare neliniara. Regresia polinomiala (φ(x) = [1,x,...,x^d]) este un exemplu important, dar polinoamele de grad inalt pot produce oscilatii extreme.

Tipare Cheie

Expansiunea in functii de baza este un concept unificator: SVM-uri, retele neuronale, arborii de clasificare si regresie pot fi vazute ca metode diferite de estimare a functiilor de baza din date. Aceasta perspectiva unificata este una dintre temele centrale ale cartii.

Aplicatii Practice

Regresia polinomiala este folosita in calibrarea senzorilor, modelarea curbelor de crestere in biologie, si ajustarea datelor experimentale. Expansiunea in functii de baza permite modelarea oricarei relatii neliniare pastrand cadrul liniar in parametri.

Exemplu din Lumea Reala

Un inginer care calibreaza un senzor de temperatura poate folosi regresie polinomiala: relatia intre tensiunea masurata si temperatura reala poate fi neliniara, dar un polinom de grad 2 sau 3 o aproximeaza bine. Un polinom de grad 20 ar ajusta perfect datele de calibrare dar ar da predictii absurde intre punctele de calibrare.

SIR si Filtrarea particulelor - Introducere

Aceasta pagina completeaza discutia SIR si introduce filtrarea particulelor (particle filtering) ca metoda Monte Carlo pentru inferenta Bayesiana secventiala.

Filtrarea particulelor (PF): Un algoritm Monte Carlo bazat pe simulare pentru inferenta Bayesiana recursiva (ciclul predict-update). Aproximeaza distributia de filtrare p(z_t|y_{1:t}) cu un set ponderat de particule. Este utilizat pe scara larga in urmarire, prognoza si invatare online.
SIR pentru inferenta Bayesiana: Cu priorul ca propunere, ponderile sunt w_s = p(D|theta_s) / sum p(D|theta_{s'}). Daca priorul si posterioarul difera mult, varianta ponderilor este mare si sunt necesare foarte multe esantioane.

Comentariu

Rezumat

Pagina face tranzitia de la SIR static la filtrarea particulelor dinamica. PF este una dintre cele mai importante metode Monte Carlo, permitand inferenta secventiala in modele cu stari latente care evolueaza in timp.

Tipare Cheie

Filtrarea particulelor combina trei ingrediente: (1) esantionarea prin importanta secventiala (SIS), (2) re-esantionarea pentru a combate degenerarea, (3) o distributie propunere buna. Fiecare ingredient rezolva o problema specifica.

Aplicatii Practice

PF este standard in robotica (localizare, SLAM), finante (filtrarea volatilitatii stochastice), ecologie (estimarea populatiilor), si procesarea semnalelor (urmarirea tintelor radar).

Exemplu din Lumea Reala

Un robot autonom foloseste filtrarea particulelor pentru a-si estima pozitia pe o harta: fiecare particula reprezinta o pozitie posibila, ponderata de cat de bine se potriveste cu citirile senzorilor.

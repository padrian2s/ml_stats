Inferenta Bayesiana pentru familia exponentiala: prior conjugat si posterior

Aceasta pagina deriveaza formele generale ale priorului conjugat, verosimilitatii si posteriorului pentru orice membru al familiei exponentiale.

Priorul conjugat general: p(θ | ν_0, τ_0) ∝ g(θ)^(ν_0) exp(η(θ)^T τ_0), unde ν_0 este „marimea" pseudo-datelor prior si τ_0 = ν_0 τ_bar_0 contine media statisticilor suficiente pe pseudo-date.
Posteriorul conjugat: p(θ | D) = p(θ | ν_0 + N, τ_0 + s_N), unde s_N = Σ s(x_i) este suma statisticilor suficiente. Actualizarea se face prin simpla adunare a pseudo-datelor prior cu datele reale.
Media posterioara ca medie ponderata: Hiperparametrii posteriori sunt o combinatie convexa intre hiperparametrii prior si media statisticilor suficiente: τ_bar_N = (ν_0 τ_bar_0 + N s_bar)/(ν_0 + N). Ponderea datelor creste liniar cu N, iar cea a priorului ramane constanta ν_0.

Comentariu

Rezumat

Pagina generalizeze inferenta Bayesiana la toata familia exponentiala. Priorul conjugat are o forma simpla si intuitiva: se comporta ca si cum am fi observat ν_0 pseudo-date cu statisticile suficiente medii τ_bar_0. Posteriorul se obtine prin „adunarea" datelor reale la pseudo-date. Aceasta unificare inseamna ca orice rezultat Bayesian pentru Gaussiana, Bernoulli, Poisson etc. este un caz special al formulelor generale de pe aceasta pagina.

Tipare Cheie

1) Actualizarea Bayesiana = adunare in spatiul parametrilor extinsi (ν, τ). 2) ν_0 controleaza „taria" priorului: ν_0 = 0 este prior neinformativ, ν_0 = N inseamna prior la fel de puternic ca datele. 3) Media posterioara interpoleaza intre prior si date cu ponderi ν_0 si N. 4) Toate formulele specifice (Beta-Binomial, Gamma-Poisson, NIW-Gaussiana) sunt cazuri speciale.

Aplicatii Practice

Aceasta generalizare permite implementarea unui singur motor de inferenta Bayesiana care functioneaza pentru orice distributie din familia exponentiala. In software-ul Bayesian modern (Stan, PyMC), aceasta structura este exploatata pentru optimizarea automata a algoritmilor de esantionare.

Exemplu din Lumea Reala

Un startup de e-commerce vrea sa faca inferenta Bayesiana pe mai multi indicatori simultan: rata de click (Bernoulli/Beta), numarul de achizitii pe sesiune (Poisson/Gamma), si valoarea comenzii (Gaussiana/NIG). Cadrul unificat al familiei exponentiale permite utilizarea aceluiasi cod si a acelorasi principii pentru toti acesti indicatori, doar schimband distributia si priorul conjugat.

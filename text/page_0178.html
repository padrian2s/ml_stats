Exercitii: QDA cu 3 Clase si QDA Scalar

Pagina prezinta exercitii de clasificare cu QDA pentru 3 clase si un exercitiu practic de QDA scalar aplicat la clasificarea genului pe baza inaltimii.

QDA cu 3 clase: Problema de clasificare cu probabilitati prior egale P(Y=1) = P(Y=2) = P(Y=3) = 1/3 si densitati conditionale de clasa normale multivariate cu parametri diferiti (Ecuatia 4.290).
Matrici de covarianta: Σ₁ = [[0.7, 0], [0, 0.7]], Σ₂ = [[0.8, 0.2], [0.2, 0.8]], Σ₃ = [[0.8, 0.2], [0.2, 0.8]] (Ecuatia 4.292)
QDA Scalar - Estimare MLE: p(x|y=c) = N(x; μ_c, σ_c) cu prior p(y=c) = π_c (Ecuatiile 4.293-4.294)

Comentariu

Rezumat

Exercitiul 4.22 cere clasificarea punctelor x=[-0.5, 0.5] si x=[0.5, 0.5] folosind QDA cu 3 clase. Exercitiul 4.23 aplica QDA scalar la un set de date real: clasificarea genului pe baza inaltimii unor studenti americani, cu date x=(67,79,71,68,67,60) si y=(m,m,m,f,f,f).

Tipare Cheie

Se observa ca Σ₂ = Σ₃, ceea ce inseamna ca clasele 2 si 3 au aceeasi forma de elipsoida dar centre diferite. Clasa 1 are covarianta diagonala (caracteristici independente), in timp ce clasele 2 si 3 au corelatie pozitiva intre caracteristici.

Aplicatii Practice

QDA scalar este cel mai simplu clasificator bayesian parametric si serveste ca baza pentru intelegerea clasificatorilor mai complexi. Estimarea MLE a parametrilor μ_c si σ_c din date este un pas fundamental in constructia oricarui clasificator generativ.

Exemplu din Lumea Reala

Clasificarea genului pe baza inaltimii este un exemplu clasic: barbatii au in medie inaltime mai mare (μ_m > μ_f) dar si variabilitate diferita. Prin plug-in prediction cu parametrii MLE, putem calcula p(y=m|x=72, θ_hat) pentru a prezice genul unei persoane cu inaltimea de 72 inch.

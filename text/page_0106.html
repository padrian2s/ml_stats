Estimarea MAP Dirichlet-Multinomială și Multiplicatorii Lagrange

Pagina derivă estimarea MAP pentru modelul Dirichlet-multinomial utilizând multiplicatori Lagrange pentru a impune constrângerea Σθ_k = 1.

Funcția Lagrangiană: ℓ(θ,λ) = Σ N_k·log θ_k + Σ(α_k−1)·log θ_k + λ(1 − Σθ_k), combinând log-verosimilitatea, log-priorul și constrângerea de normalizare.
Estimarea MAP: θ̂_k = (N_k + α_k − 1)/(N + α₀ − K), unde α₀ = Σα_k. Cu prior uniform (α_k = 1), se reduce la MLE: θ̂_k = N_k/N.
Multiplicatorul Lagrange: λ = N + α₀ − K, obținut din constrângerea Σθ_k = 1. Rezolvarea: ∂ℓ/∂θ_k = N'_k/θ_k − λ = 0, deci N'_k = λ·θ_k.
MLE multinomială: θ̂_k = N_k/N, simpla fracțiune empirică - frecvența relativă a fiecărei categorii.

Comentariu

Rezumat

Derivarea estimării MAP utilizează multiplicatori Lagrange pentru a gestiona constrângerea simplex. Rezultatul este o generalizare directă a cazului binar: se adaugă pseudo-numărătorile la numărătorile empirice.

Tipare Cheie

Multiplicatorii Lagrange sunt un instrument standard pentru optimizare cu constrângeri. Constrângerea Σθ_k = 1 este naturală pentru distribuții de probabilitate pe simplex.

Aplicatii Practice

Estimarea MAP/MLE multinomială este utilizată în modelele de limbaj unigram, în estimarea probabilităților de tranziție în lanțuri Markov, și în clasificarea documentelor.

Exemplu din Lumea Reala

Estimarea frecvenței literelor într-o limbă (ex: „e" apare ~12% în engleză) prin MLE multinomială este baza criptanalizei clasice și a algoritmilor de compresie adaptivă.

ARD pentru Regresia Liniara: Verosimilitatea Marginala si Posteriorul

Pagina deriveaza verosimilitatea marginala si posteriorul pentru ARD in regresia liniara, inclusiv formulele de actualizare pentru alpha si beta.

Verosimilitatea marginala (Eq. 13.149-13.153): p(y|X, alpha, beta) = N(y|0, C_alpha), unde C_alpha = beta^{-1} I_N + X A^{-1} X^T. Log-verosimilitate: l(alpha, beta) = log|C_alpha| + y^T C_alpha^{-1} y.
Obiectivul regularizat (Eq. 13.154-13.155): Cu priori Gamma pe alpha_j si beta: l(alpha, beta) += suma (a*log(alpha_j) - b*alpha_j) + c*log(beta) - d*beta. Cu prior neinformativ (a=b=c=d=0), se obtine sparsitate maxima.
Posteriorul pe w (Eq. 13.156-13.158): p(w|D, alpha_hat, beta_hat) = N(mu, Sigma), cu Sigma^{-1} = beta_hat X^T X + A si mu = beta_hat Sigma X^T y. Media posterioara mu este (aproximativ) sparsa datorita valorilor mari ale alpha_hat.

Comentariu

Rezumat

ARD optimizeaza log-verosimilitatea marginala (Type II ML) fata de hiperparametrii alpha si beta. Aceasta este analoga cu selectia Bayesiana de variabile, dar cu parametri continui in loc de binari. Obiectivul are mai putine optime locale decat l0, facand optimizarea mai stabila. Posteriorul pe w este Gaussian cu media sparse.

Tipare Cheie

ARD produce sparsitate prin mecanismul de "automatic relevance determination": variantele alpha_j^{-1} ale ponderilor irelevante converg la zero (alpha_j -> infinit), eliminand efectiv aceste ponderi din model. Aceasta forma de regularizare adaptiva este mai sofisticata decat regularizarea fixa (l1 sau l2).

Aplicatii Practice

ARD este implementat in scikit-learn (ARDRegression), PyMC3, Stan. Este util cand se doreste o estimare Bayesiana completa (inclusiv incertitudine) cu sparsitate automata, fara necesitatea de a alege lambda.

Exemplu din Lumea Reala

In procesarea de semnal radar, ARD estimeaza automat numarul si pozitia tintelor (semnale non-zero) dintr-un spectru sparse, fara cunoastere a priori a numarului de tinte.

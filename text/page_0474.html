Metode Proximale de Gradient: Operatorul Proximal

Pagina introduce metodele proximale de gradient pentru optimizarea compusa f(theta) = L(theta) + R(theta), defineste operatorul proximal si prezinta exemple specifice.

Optimizarea compusa (Eq. 13.66): f(theta) = L(theta) + R(theta), unde L este pierderea (convexa, diferentiabila) si R este regularizatorul (convex, posibil ne-diferentiabil). Exemplu: L = RSS, R = lambda||theta||_1.
Operatorul proximal (Eq. 13.68-13.69): prox_R(y) = argmin_z [R(z) + (1/2)||z - y||_2^2]. Gaseste punctul care minimizeaza R fiind cat mai aproape de y. Este generalizarea proiectiei pe o multime convexa.
Operatori proximali specifici (Eq. 13.70-13.72): R = lambda||theta||_1: prox = soft(theta, lambda). R = lambda||theta||_0: prox = hard(theta, sqrt(2*lambda)). R = I_C (indicator multime): prox = proj_C(theta).

Comentariu

Rezumat

Cadrul de optimizare compusa (pierdere smooth + regularizator non-smooth) este extrem de general si acopera LASSO, group LASSO, nuclear norm minimization si multe altele. Operatorul proximal este componenta cheie: transforma problema de optimizare ne-smooth intr-o secventa de sub-probleme cu solutii analitice.

Tipare Cheie

Operatorul proximal generalizeaza trei operatii: soft thresholding (l1), hard thresholding (l0) si proiectie pe multimi convexe (constrangeri). Aceasta unificare permite dezvolarea de algoritme generale care functioneaza cu orice regularizator cu proximal calculabil.

Aplicatii Practice

Operatorii proximali sunt implementati eficient in pachete precum proximal (Python), TFOCS (MATLAB). Permitand separarea pierderii de regularizator, acelasi algoritm poate fi reutilizat cu diferiti regularizatori.

Exemplu din Lumea Reala

In reconstructia de imagini (CT, MRI), operatorul proximal al normei TV (Total Variation) produce denoising care pastreaza marginile ascutite, superior regularizarii l2 care produce blur.

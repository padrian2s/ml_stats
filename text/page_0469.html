Recuperarea Semnalelor Sparse si Compressed Sensing

Pagina prezinta Figura 13.9 (recuperarea unui semnal sparse cu LASSO) si discuta proprietatea caii de regularizare: limitarea la cel mult N variabile si necesitatea elastic net.

Figura 13.9 - Compressed Sensing: Semnal original (D=4096, 160 non-zero). Reconstructia l1 (K=1024 masuratori): MSE=0.0027, recuperare buna. Dupa debiasing: MSE=3.26e-5, recuperare aproape perfecta. Solutia de norma minima (fara sparsitate): MSE=0.029, foarte proasta.
Debiasing: Dupa ce LASSO identifica setul activ (variabilele non-zero), se recalculeaza OLS doar pe aceste variabile, eliminand bias-ul de shrinkage. Produce estimari mult mai precise cand setul activ este corect identificat.
Limitarea LASSO: Daca D > N, LASSO poate selecta cel mult N variabile. Elastic net (l1 + l2 regularization) depaseste aceasta limitare si permite selectarea de grupuri de variabile corelate.

Comentariu

Rezumat

Figura 13.9 demonstreaza puterea compressed sensing: un semnal cu 4096 dimensiuni si 160 componente non-zero poate fi recuperat din doar 1024 masuratori liniare aleatoare, folosind regularizarea l1. Debiasing-ul dupa selectie imbunatateste dramatic precizia. Solutia de norma minima (fara sparsitate) esueaza complet.

Tipare Cheie

Compressed sensing se bazeaza pe doua ingrediente: (1) semnalul este sparse intr-o anumita baza, (2) masuratorile sunt "incoerente" cu baza de sparsitate. Sub aceste conditii, l1 recupereaza semnalul exact din masuratori substochastice.

Aplicatii Practice

Compressed sensing a revolutionat imagistica medicala (MRI rapid), astronomia (reconstructia de imagini din masuratori limitate) si procesarea de semnal (achizitie sub-Nyquist).

Exemplu din Lumea Reala

MRI-ul rapid bazat pe compressed sensing permite scanari de 2-4x mai rapide: se achizitioneaza mai putine masuratori k-space si se reconstruieste imaginea prin regularizare l1 in domeniul wavelet, reducand timpii de asteptare ai pacientilor.

Netezitori liniare si grade de libertate in GP

Aceasta pagina compara GP cu netezitorii liniare, introduce conceptul de kernel echivalent si defineste gradele de libertate efective ale unui GP.

Netezitor liniar: O functie de regresie de forma f_hat(x_*) = sum_i w_i(x_*) y_i, unde w_i(x_*) este functia de ponderare. GP-ul este un netezitor liniar cu w_i(x_*) = [(K + sigma_y^2 I)^{-1} k_*]_i.
Grade de libertate efective: dof = tr(K(K + sigma_y^2 I)^{-1}) = sum_i lambda_i / (lambda_i + sigma_y^2), unde lambda_i sunt valorile proprii ale K. Aceasta specifica cat de 'oscilatorie' este curba ajustata.

Comentariu

Rezumat

GP-ul este un netezitor liniar ale carui ponderi depind de inversul matricei K. Kernelul echivalent w_i(x) este intotdeauna local (cu suport finit efectiv), chiar daca kernelul original nu este local. Interesant, bandwidth-ul efectiv al kernelului echivalent scade automat cu N, spre deosebire de netezirea prin kernel unde h trebuie setat manual. Gradele de libertate efective ofera o masura scalara a complexitatii modelului.

Tipare Cheie

Eigendecompozitia K = sum lambda_i u_i u_i^T arata ca GP-ul actioneaza ca un filtru: componentele cu lambda_i >> sigma_y^2 sunt pastrate (lambda_i/(lambda_i + sigma_y^2) ~ 1), iar cele cu lambda_i << sigma_y^2 sunt atenuate. Aceasta este exact netezirea spectrala.

Aplicatii Practice

Gradele de libertate efective sunt folosite pentru a compara complexitatea modelelor GP cu diferiti hiperparametri sau kerneluri. Sunt analoge numarului de parametri intr-un model parametric si pot fi folosite in criterii de selectie a modelului.

Exemplu din Lumea Reala

In analiza seriilor temporale economice, gradele de libertate efective ajuta la alegerea nivelului de netezire: un dof mic produce un trend neted (util pentru analiza pe termen lung), iar un dof mare captureaza fluctuatii pe termen scurt.

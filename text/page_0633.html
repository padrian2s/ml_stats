Calculul Eficient al Vectorului PageRank (p. 602)

Pagina prezintă exemplul numeric al PageRank-ului, graful web Harvard și definirea matricei de tranziție pentru calculul eficient al vectorului PageRank.

Matricea de tranziție M: M_ij = pG_ij/c_j + δ dacă c_j ≠ 0, sau 1/n dacă c_j = 0, unde p = 0.85 este probabilitatea de a urma un link și δ = (1-p)/n este probabilitatea de salt aleatoriu (Ec. 17.34).
Distribuția staționară exemplu: π = (0.3209, 0.1706, 0.1065, 0.1368, 0.0643, 0.2008) pentru graful din Figura 17.5 (Ec. 17.33).
Figura 17.6: (a) Graful web de 500 de site-uri pornind de la www.harvard.edu. (b) Vectorul PageRank corespunzător, arătând distribuția inegală a importanței.

Comentariu

Rezumat

Se arată cum se construiește matricea de tranziție din graful web și cum distribuția staționară reflectă importanța reală a paginilor. Nodul 1 are cel mai mare PageRank (32%) datorită link-urilor de la noduri influente.

Tipare Cheie

Parametrul de salt aleatoriu δ servește dublu scop: asigură ireductibilitatea lanțului (orice pagină este accesibilă) și regularitatea sa, garantând convergența la o distribuție staționară unică.

Aplicatii Practice

Pentru web-ul real cu miliarde de pagini, stocarea eficientă a matricei rare G este esențială. Matricea M = pGD + 1z^T permite calculul fără stocarea explicită a elementelor mici δ.

Exemplu din Lumea Reala

Graful web Harvard ilustrează cum structura link-urilor unei universități reflectă ierarhia informațională: pagina principală și hub-urile de departamente primesc cele mai mari scoruri PageRank.

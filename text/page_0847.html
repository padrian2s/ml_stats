Capitolul 23: Inferenta Monte Carlo - Esantionarea prin CDF inversa

Aceasta pagina continua capitolul 23 despre inferenta Monte Carlo, prezentand metoda CDF inversa pentru generarea de esantioane din distributii univariate.

Teorema 23.2.1 (CDF inversa): Daca U ~ U(0,1) este o variabila aleatoare uniforma si F este CDF-ul distributiei tinta, atunci F^{-1}(U) ~ F. Demonstratia este directa: Pr(F^{-1}(U) <= x) = Pr(U <= F(x)) = F(x).
Generator pseudo-aleator: Un algoritm determinist care produce secvente de numere care apar aleatorii. Este baza tuturor metodelor de esantionare: genereaza u ~ U(0,1), apoi transforma prin x = F^{-1}(u).
Exemplu - distributia exponentiala: Pentru Expon(lambda), CDF este F(x) = 1 - e^{-lambda*x}, iar inversa este F^{-1}(p) = -ln(1-p)/lambda. Deci x = -ln(u)/lambda cu u ~ U(0,1) produce esantioane din Expon(lambda).

Comentariu

Rezumat

Metoda CDF inversa este fundamentala si eleganta: orice distributie univariata cu CDF inversabila poate fi esantionata printr-o singura transformare a unei variabile uniforme. Limitarea principala este ca necesita calculul analitic al CDF inverse, ceea ce nu este intotdeauna posibil.

Tipare Cheie

Transformarea uniform -> tinta prin CDF inversa este un principiu universal. Pentru distributii cu CDF inversabila analitic (exponentiala, Cauchy, logistica), metoda este cea mai eficienta posibila.

Aplicatii Practice

Metoda CDF inversa este implementata in toate bibliotecile de calcul stiintific (NumPy, SciPy, R) pentru distributii standard. Pentru distributii fara CDF inversa analitica, se folosesc metode alternative.

Exemplu din Lumea Reala

In simularea timpilor de asteptare intr-un call center (distribuiti exponential), se genereaza u uniform si se calculeaza t = -ln(u)/lambda, obtinand direct un timp de asteptare simulat realist.

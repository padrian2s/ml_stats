Capitolul 21. Inferenta Variationala - Aplicatii Motivante pentru Limite Locale (p.757)

Aceasta pagina prezinta functia log-sum-exp (lse), aplicatii motivante (multi-task learning, analiza factoriala discreta, modele de topicuri corelate) si necesitatea limitelor variationala locale.

Verosimilitatea softmax: p(y|X,w) = ∏ᵢ exp[yᵢᵀηᵢ - lse(ηᵢ)], unde ηᵢ = Xᵢwᵢ si lse(ηᵢ) = log(1 + Σₘ exp(ηᵢₘ)).
Problema fundamentala: Verosimilitatea softmax/logistica nu este conjugata cu priorul gaussian, facand actualizarile mean field intractabile. Solutia: derivarea limitelor „de tip gaussian" pe verosimilitate.
Multi-task learning: Cand avem clasificatoare multiple inrudite, vrem sa partajam informatia intre parametri mentinand o distributie posterioara completa, nu doar estimari punctuale.
Analiza factoriala discreta: Model de analiza factoriala pentru date discrete, similar regresiei logistice multinomiale cu factori ascunsi. Se foloseste VBEM cu limita variationala in pasul E.
Modelul de topicuri corelate: Extensie a LDA care foloseste un prior gaussian (in loc de Dirichlet) pentru a modela corelatii intre topicuri, necesitand functia lse si deci limite variationala.

Comentariu

Rezumat

Pagina motiveaza necesitatea limitelor variationala locale prin patru aplicatii importante care implica verosimilitatea softmax/logistica cu priori gaussiene. Toate necesita aproximarea functiei lse cu forme patratice tractabile.

Tipare Cheie

Functia lse este obstacol comun in multe modele probabilistice. Problema centrala este non-conjugarea intre verosimilitatea logistica/softmax si priorul gaussian. Limitele locale transforma aceasta problema intractabila intr-una tractabila.

Aplicatii Practice

Multi-task learning variational este utilizat in NLP, viziune computationala si medicina personalizata. Modelele de topicuri corelate sunt fundamentale in analiza documentelor si recomandari.

Exemplu din Lumea Reala

In analiza stirilor, un model de topicuri corelate poate captura faptul ca articolele despre „economie" si „politica" sunt adesea corelate, iar acest lucru necesita exact limitele variationala discutate aici.

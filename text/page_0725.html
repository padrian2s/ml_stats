SSVM-uri: perspectiva probabilistica

Aceasta pagina dezvolta fundamentul teoretic al SVM-urilor structurale din perspectiva probabilistica, derivand obiectivul prin minimizarea pierderii asteptate a posteriori.

Pierderea asteptata a posteriori: La testare, se alege eticheta care minimizeaza pierderea asteptata: ŷ = argmin_{ŷ} Σ_y L(ŷ, y) p(y|x, w). Aceasta motiveaza incorporarea functiei de pierdere L in antrenare.
Riscul empiric asteptat (R_EL): Obiectivul SSVM: R_EL(w) = -log p(w) + Σ_i log[Σ_y L(y_i, y) p(y|x_i, w)]. Pentru pierderea 0-1, aceasta se reduce la obiectivul MAP standard R_MAP.
Aproximarea SSVM: Folosind marginile log Σ exp(f) ≈ max f (cand functiile sunt dominate de maximum), obiectivul se simplifica la: R_EL ≈ E(w) + Σ_i [max_y {L̃(y_i, y) + w^T φ(x_i, y)} - max_y w^T φ(x_i, y)], ceea ce necesita doar decodare MAP (nu marginale).

Comentariu

Rezumat

SSVM-urile sunt derivate dintr-o perspectiva probabilistica prin doua aproximari: (1) inlocuirea sumei cu maximul (log-sum-exp ≈ max) si (2) ignorarea termenului constant |Y|. Rezultatul este un obiectiv care necesita doar doua decodari MAP la fiecare pas: una augmentata cu pierderea (loss-augmented inference) si una standard. Aceasta evita complet calculul marginalelor si al functiei de partitie.

Tipare Cheie

Aproximarea log-sum-exp ≈ max transforma problema de la calculul marginalelor la calculul MAP-ului. Inferenta calibrata cu pierderea (loss-calibrated inference) este in general intractabila, asa ca se recurge la estimarea MAP calibrata cu pierderea. Distinctia inferenta-decizie este incalcata in SSVM-uri dar functioneaza bine in practica.

Aplicatii Practice

SSVM-urile sunt eficiente in probleme de predicere structurata unde decodarea MAP este rapida: segmentarea imaginilor (graph cuts), parsarea (algoritmi CKY), si etichetarea secventelor (Viterbi). Antrenarea necesita doar un oracol MAP, nu un motor de inferenta complet.

Exemplu din Lumea Reala

In traducerea automata statistica, SSVM-ul antreneaza modelul gasind cea mai buna traducere gresita (loss-augmented MAP) la fiecare pas si ajustand parametrii pentru a o penaliza fata de traducerea corecta. Algoritmul MERT este o varianta populara.

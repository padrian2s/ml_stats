SVD Trunchiata si Aproximarea Low-Rank

Pagina prezinta Figura 12.8 (diagrama SVD completa si trunchiata), legatura intre SVD si PCA, si formulele pentru aproximarea low-rank a matricilor.

Figura 12.8: (a) SVD completa: X = U S V^T cu dimensiunile N x D. (b) SVD trunchiata de rang L: X ≈ U_L S_L V_L^T, retinand doar primele L valori singulare si vectorii asociati.
Descompunerea in rang 1 (Eq. 12.54): X = sigma_1 u_1 v_1^T + ... + sigma_r u_r v_r^T. Fiecare matrice de rang 1 sigma_i u_i v_i^T capteaza un "pattern" din date, ponderat de valoarea singulara.
SVD trunchiata (Eq. 12.55-12.56): X ≈ U_{:,1:L} S_{1:L,1:L} V_{:,1:L}^T. Necesita L(N + D + 1) parametri in loc de ND. Este cea mai buna aproximare de rang L in norma Frobenius (teorema Eckart-Young).

Comentariu

Rezumat

SVD trunchiata ofera cea mai buna aproximare de rang L a oricarei matrice, in sensul normei Frobenius (teorema Eckart-Young). Aceasta este fundatia teoretica a PCA: proiectia pe primele L componente principale este echivalenta cu cea mai buna aproximare de rang L a matricii de date centrate.

Tipare Cheie

Compresia prin rang scazut: o matrice N x D cu ND elemente poate fi aproximata cu doar L(N+D+1) parametri. Daca L << min(N,D), compresia este masiva. Calitatea depinde de cat de repede scad valorile singulare.

Aplicatii Practice

SVD trunchiata este folosita in compresia de date, sisteme de recomandare (factorizarea matricii), reducerea zgomotului in imagini, si Latent Semantic Analysis (LSA) pentru procesarea limbajului natural.

Exemplu din Lumea Reala

In compresia video, SVD trunchiata pe fiecare cadru (sau pe diferente intre cadre) permite stocarea eficienta a filmelor, pastrand doar componentele vizuale semnificative si eliminand zgomotul.

Capitolul 2: Cuantile, Medie si Varianta

Cuantilele distributiilor, intervalele de incredere, media, varianta si deviatia standard.

Cuantila: Daca F este CDF-ul lui X, atunci F⁻¹(α) = x_α este valoarea astfel incat P(X ≤ x_α) = α. Se numeste cuantila α a distributiei.
Mediana: Cuantila F⁻¹(0.5), valoarea care imparte distributia in doua jumatati egale.
Cuartile: F⁻¹(0.25) si F⁻¹(0.75) sunt cuartilele inferioara si superioara.
Media (Valoarea Asteptata): E[X] = Σ_{x∈X} x p(x) (discret) sau E[X] = ∫_X x p(x)dx (continuu). Notata μ.
Varianta: var[X] = E[(X - μ)²] = ∫(x - μ)²p(x)dx = E[X²] - μ² (ecuatiile 2.24-2.25). Notata σ². Masoara "raspandirea" distributiei.
Deviatia Standard: std[X] = √(var[X]) (ecuatia 2.27). Are aceleasi unitati ca X, facand-o mai usor de interpretat decat varianta.
Figura 2.3: (a) CDF-ul distributiei normale standard N(0,1). (b) PDF-ul corespunzator cu regiunile de coada (fiecare cu aria α/2). Intervalul central de 95% este (-1.96, 1.96).
Intervalul Central de 95%: Pentru N(0,1), intervalul (Φ⁻¹(0.025), Φ⁻¹(0.975)) = (-1.96, 1.96). Pentru N(μ, σ²), intervalul este aproximativ μ ± 2σ (ecuatia 2.23).

Comentariu

Rezumat

Sectiunile 2.2.6-2.2.7 introduc statisticile sumarizatoare ale distributiilor. Cuantilele descriu pozitia in distributie, media masoara locatia centrala, iar varianta masoara raspandirea. Relatia E[X²] = μ² + σ² este foarte utila. Intervalul central de 95% (μ ± 1.96σ) este fundamental in statistica.

Tipare Cheie

Media si varianta sunt momente ale distributiei. Pentru distributia Gaussiana, ele specifica complet distributia. Regula "μ ± 2σ" este o aproximare utila: ~95% din masa de probabilitate se afla in acest interval. Cuantilele sunt mai robuste la valori extreme decat media.

Aplicatii Practice

Intervalele de incredere bazate pe cuantile sunt folosite in raportarea rezultatelor experimentale, intervalele de predictie in regresie, si testele de normalitate. Deviatia standard este metrica standard de risc in finante (volatilitate).

Exemplu din Lumea Reala

Un raport meteorologic care spune "temperatura medie de 20°C cu deviatie standard de 3°C" ne spune ca in ~95% din cazuri temperatura va fi intre 14°C si 26°C. Aceasta cuantificare a incertitudinii ajuta la planificarea activitatilor in aer liber.

Comentariu

Rezumat

Capitolele 7 si 8 acopera cele doua modele liniare fundamentale: regresia liniara (pentru probleme de regresie) si regresia logistica (pentru probleme de clasificare). Ambele sunt prezentate atat din perspectiva frequentista (MLE) cat si din cea bayesiana.

Tipare Cheie

Se observa un tipar consistent: fiecare model este prezentat prin specificare → antrenare (MLE) → regularizare → versiunea bayesiana. Aceasta structura se repeta in multe capitole ale cartii.

Aplicatii Practice

Regresia logistica (cap. 8) este unul dintre cele mai utilizate modele in industrie, de la scoring-ul de credit bancar la clasificarea imaginilor medicale. SGD (8.5) este algoritmul de baza pentru antrenarea retelelor neuronale moderne.

Exemplu din Lumea Reala

O banca foloseste regresia logistica regularizata (8.3.6) pentru a prezice probabilitatea de neplata a unui credit. Regularizarea previne supraajustarea pe datele istorice, iar interpretabilitatea modelului este importanta pentru conformitatea cu reglementarile.

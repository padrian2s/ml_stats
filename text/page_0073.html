Cazuri speciale ale distributiei Gamma, distributia Gamma inversa si distributia Beta

Aceasta pagina continua discutia despre distributia Gamma si cazurile sale speciale (exponentiala, Erlang, chi-patrat), introduce distributia Gamma inversa si incepe prezentarea distributiei Beta.

Distributia exponentiala: Expon(x|λ) = Ga(x|1, λ), descrie timpii intre evenimente intr-un proces Poisson cu rata medie λ.
Distributia chi-patrat: χ²(x|ν) = Ga(x|ν/2, 1/2), este distributia sumei patratelor a ν variabile aleatoare Gaussiene standard.
Distributia Gamma inversa: IG(x|a, b) = (b^a / Γ(a)) x^(-(a+1)) e^(-b/x), cu media = b/(a-1) pentru a > 1 si varianta = b²/((a-1)²(a-2)) pentru a > 2.
Distributia Beta: Beta(x|a, b) = (1/B(a,b)) x^(a-1) (1-x)^(b-1), definita pe intervalul [0,1], unde B(a,b) = Γ(a)Γ(b)/Γ(a+b).
Relatia Gamma-Gamma inversa: Daca X ~ Ga(a,b), atunci 1/X ~ IG(a,b). Aceasta relatie este utila in statistica Bayesiana unde Gamma inversa serveste ca prior conjugat pentru varianta distributiei Gaussiene.

Comentariu

Rezumat

Pagina prezinta o familie bogata de distributii derivate din Gamma. Distributia exponentiala modeleaza timpii de asteptare fara memorie, distributia Erlang (Gamma cu parametru intreg) modeleaza suma timpilor de asteptare, iar chi-patrat apare natural in testele statistice. Distributia Gamma inversa este importanta ca prior conjugat in inferenta Bayesiana. Distributia Beta, definita pe [0,1], este fundamentala pentru modelarea probabilitatilor si proportiilor.

Tipare Cheie

1) Familia Gamma formeaza o ierarhie: exponentiala este cel mai simplu caz, Erlang adauga un parametru intreg, iar Gamma generalizeaza la parametru real. 2) Functia Beta B(a,b) este strans legata de functia Gamma. 3) Conditiile de existenta a momentelor (media necesita a > 1 pentru IG, varianta necesita a > 2) sunt cruciale in practica.

Aplicatii Practice

Distributia chi-patrat este omniprezenta in testele de potrivire (goodness-of-fit) si in testele de independenta. Distributia Beta este folosita ca prior pentru parametrii Bernoulli si binomiali in statistica Bayesiana. Gamma inversa serveste ca prior pentru varianta in modele ierarhice.

Exemplu din Lumea Reala

In controlul calitatii industrial, distributia Beta modeleaza rata de defecte a unui proces de fabricatie. Daca avem a succese si b esecuri in inspectii anterioare, Beta(a,b) reprezinta incertitudinea noastra asupra adevaratei rate de defecte, actualizandu-se natural cu fiecare nou lot inspectat.

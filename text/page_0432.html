PCA pentru Date Categoriale

Pagina introduce PCA categorial, extinderea modelului de analiza factoriala la date observate categoriale in loc de continue, cu distributia logistic-normala pe simplex.

PCA Categorial: Un model de analiza factoriala in care datele observate sunt categoriale (yᵢⱼ ∈ {1,...,C}), generate prin trecerea unei variabile latente gaussiene prin functia softmax.
Distributia logistic-normala: Distributia indusa pe simplexul probabilistic cand o variabila gaussiana este trecuta prin functia softmax. Poate genera distributii complexe pe simplex.
Modelul generativ: p(zᵢ) = N(0, I); p(yᵢ|zᵢ, θ) = Π_r Cat(yᵢᵣ | S(Wᵣᵀ zᵢ + w₀ᵣ))

Comentariu

Rezumat

PCA categorial extinde analiza factoriala clasica la date discrete. In loc de o verosimilitate gaussiana, se foloseste o distributie categorica cu parametri generati prin softmax dintr-o variabila latenta gaussiana. Distributia logistic-normala rezultanta pe simplex este foarte flexibila, putand modela distributii multimodale si corelate. Inferenta se face prin EM modificat cu aproximare gaussiana a posteriorului in pasul E.

Tipare Cheie

Trecerea de la date continue la categoriale necesita inlocuirea verosimilitudinii gaussiene cu una categorica, dar structura latenta gaussiana ramane aceeasi. Aceasta modularitate este o caracteristica puternica a modelelor cu variabile latente. Distributia logistic-normala este semnificativ mai flexibila decat distributia Dirichlet, care este limitata la forme unimodale.

Aplicatii Practice

PCA categorial este util in analiza datelor de sondaj (raspunsuri pe scale Likert), modelarea preferintelor consumatorilor, analiza textului (distributii peste vocabular), si in genomica (genotipuri discrete).

Exemplu din Lumea Reala

In cercetarea de piata, raspunsurile consumatorilor la chestionare cu optiuni multiple pot fi modelate cu PCA categorial pentru a descoperi dimensiunile latente ale preferintelor (de exemplu, preferinta pentru calitate vs pret, inovatie vs traditie).

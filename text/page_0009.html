Comentariu

Rezumat

Aceasta parte a cuprinsului acopera fundamente esentiale: distributiile de probabilitate (cap. 2), modelele generative pentru date discrete incluzand clasificatorul Naive Bayes (cap. 3), si modelele Gaussiene cu analiza discriminanta (cap. 4). Este o progresie naturala de la teorie probabilistica la aplicatii concrete de clasificare.

Tipare Cheie

Se observa un tipar recurent: fiecare model este prezentat prin likelihood, prior si posterior - cadrul bayesian clasic. Distributia Gaussiana apare ca element central in multiple contexte.

Aplicatii Practice

Clasificatorul Naive Bayes (sectiunea 3.5) este utilizat pe scara larga in filtrarea spam-ului si clasificarea textelor. Analiza discriminanta (sectiunea 4.2) este folosita in recunoasterea de tipare si biometrie.

Exemplu din Lumea Reala

Clasificarea documentelor folosind "bag of words" (3.5.5) este exact ceea ce fac motoarele de cautare pentru a categoriza paginile web sau ceea ce fac filtrele de email pentru a separa mesajele legitime de spam.

Distributia predictiva Bayesiana - Exemple si ARD pentru retele neurale

Aceasta pagina ilustreaza distributia predictiva Bayesiana pentru un MLP si introduce ARD (Automatic Relevance Determination) aplicat retelelor neurale.

ARD pentru retele neurale: Prior cu hiperparametri separati per intrare si per unitate ascunsa: p(theta) = prod_i N(v_{.,i}|0, 1/alpha_{v,i} I) * prod_j N(w_{.,j}|0, 1/alpha_{w,j} I). Daca alpha_{v,i} -> infinit, intrarea i este eliminata; daca alpha_{w,j} -> infinit, unitatea ascunsa j este eliminata.
Exemplu ARD: Pentru f(x_1,x_2,x_3) = sin(2*pi*x_1) + epsilon, cu x_2 copie zgomotoasa a x_1 si x_3 irelevant, ARD gaseste alpha = [0.2, 21.4, 249001.8], indicand corect ca x_3 este irelevant, x_2 slab relevant si x_1 foarte relevant.

Comentariu

Rezumat

Figura 16.19 arata distributia predictiva pentru un MLP cu 3 unitati ascunse antrenat pe 16 puncte: (a) Laplace cu empirical Bayes produce benzi de eroare care cresc departe de date, (b) HMC produce esantioane mai diverse din posterior. ARD aplicat retelelor neurale permite selectia automata a trasaturilor si a unitatilor ascunse, fiind echivalent cu group lasso din perspectiva penalizarii.

Tipare Cheie

ARD pentru retele neurale functioneaza pe doua niveluri: (1) selectia trasaturilor de intrare prin alpha_{v,i} si (2) selectia numarului de unitati ascunse prin alpha_{w,j}. Aceasta este o forma de cautare automata a arhitecturii (neural architecture search) printr-un prior ierarhic.

Aplicatii Practice

ARD este implementat in NETLAB (Matlab) si in bibliotecile Bayesiene moderne. In practica, dropout poate fi interpretat ca o forma de ARD aproximativ. Neural architecture search (NAS) automatizeaza si mai mult selectia arhitecturii.

In genomica, ARD aplicat unei retele neurale care prezice fenotipul din genotip permite identificarea automata a genelor relevante, oferind atat predictie precisa cat si interpretabilitate biologica.

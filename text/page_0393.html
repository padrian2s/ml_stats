EM pentru Regresia Probit

Pagina prezinta algoritmul EM pentru regresia probit, exploatand interpretarea cu variabile latente si derivand pasii E si M.

Regresia probit: Un model de clasificare unde p(yᵢ = 1|zᵢ) = I(zᵢ > 0), cu zᵢ ~ N(wᵀxᵢ, 1) latent. Variabila latenta zᵢ este trunchiata pentru a produce raspunsul binar.
Gaussiana trunchiata: Posteriorul p(zᵢ|yᵢ, xᵢ, w) este o distributie Gaussiana trunchiata - trunchiata la dreapta (zᵢ > 0) daca yᵢ = 1, sau la stanga (zᵢ < 0) daca yᵢ = 0.
Log-verosimilitatea completa: ℓ(z, w|V₀) = log p(y|z) + log N(z|Xw, I) + log N(w|0, V₀) - include termenul de verosimilitate, modelul latent si priorul pe w.
Pasul E - Media posterioara: E[zᵢ|yᵢ, w, xᵢ] = μᵢ + φ(μᵢ)/Φ(μᵢ) daca yᵢ = 1; μᵢ - φ(μᵢ)/(1 - Φ(μᵢ)) daca yᵢ = 0, unde μᵢ = wᵀxᵢ.
Pasul M: ŵ = (V₀⁻¹ + XᵀX)⁻¹ Xᵀμ - regresie ridge cu tinta μ = E[z], valorile asteptate ale variabilelor latente.
Figura 11.15: Ajustarea unui model de regresie probit 2D folosind metoda quasi-Newton sau EM. EM este mai simplu dar poate fi mult mai lent decat metodele directe bazate pe gradient.

Comentariu

Rezumat

EM pentru regresia probit exploateaza interpretarea cu variabile latente: in pasul E, se calculeaza media posterioara a variabilelor latente zᵢ (care sunt Gaussiene trunchiate), iar in pasul M, se estimeaza w prin regresie ridge cu tintele fiind aceste medii asteptate. Algoritmul este simplu dar poate fi mult mai lent decat metodele directe bazate pe gradient, deoarece entropia posterioara in pasul E este ridicata (observam doar semnul lui z, nu magnitudinea).

Tipare Cheie

Abordarea EM pentru probit generalizeaza la multe alte modele cu variabile latente trunchiate. Viteza de convergenta depinde de cat de informativ este pasul E - in cazul probit, informatia este limitata (doar semnul).

Aplicatii Practice

Regresia probit este utilizata in econometrie (modele de alegere binara), epidemiologie (modele de incidenta a bolilor), si stiinte sociale (modele de decizie). EM ofera o alternativa simpla la metodele Newton.

Exemplu din Lumea Reala

In studiile de marketing, regresia probit modeleaza decizia de cumparare (da/nu) in functie de caracteristicile produsului si ale consumatorului. EM poate fi preferat cand modelul include variabile latente suplimentare.

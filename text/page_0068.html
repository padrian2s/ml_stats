Capitolul 2: Distributia Poisson si Distributia Empirica

Distributia Poisson pentru numararea evenimentelor rare si distributia empirica bazata pe date observate.

Distributia Poisson: Poi(x|λ) = e^(-λ) λ^x / x! (ecuatia 2.39), unde X ∈ {0, 1, 2, ...} si λ > 0 este parametrul de rata. Modeleaza numarul de evenimente rare intr-un interval de timp sau spatiu.
Distributia Empirica: p_emp(A) = (1/N) Σᴺᵢ₌₁ δ_{x_i}(A) (ecuatia 2.40), unde δ_{x_i}(A) este masura Dirac care plaseaza toata masa de probabilitate in punctul x_i.
Masura Dirac: δ_x(A) = 1 daca x ∈ A, 0 daca x ∉ A (ecuatia 2.41). Concentreaza toata probabilitatea intr-un singur punct.
Figura 2.6: Ilustrarea distributiei Poisson pentru λ ∈ {1, 10}. Cu λ = 1, distributia este concentrata la valori mici. Cu λ = 10, distributia seamana cu o Gaussiana (datorita teoremei limitei centrale).
Distributie Empirica Ponderata: p(x) = Σᴺᵢ₌₁ wᵢ δ_{x_i}(x) (ecuatia 2.42), unde 0 ≤ wᵢ ≤ 1 si Σwᵢ = 1. O histograma cu "tepuse" la punctele de date.

Comentariu

Rezumat

Sectiunile 2.3.3-2.3.4 prezinta distributia Poisson si distributia empirica. Poisson este utilizata pentru a modela numarul de evenimente rare (dezintegrare radioactiva, accidente de trafic). Distributia empirica este o aproximare non-parametrica a distributiei reale, bazata pe datele observate, plasand masa egala la fiecare punct de date.

Tipare Cheie

Distributia Poisson devine din ce in ce mai simetrica si mai asemanatoare cu o Gaussiana pe masura ce λ creste (datorita teoremei limitei centrale). Distributia empirica este fundamentala in statistica non-parametrica si sta la baza bootstrap-ului. Masura Dirac face legatura intre distributiile discrete si continue.

Aplicatii Practice

Poisson: modelarea numarului de email-uri primite pe ora, numarul de accidente pe o autostrada, numarul de mutatii genetice pe o regiune a genomului. Distributia empirica: bootstrap pentru estimarea intervalelor de incredere, teste de permutare.

Exemplu din Lumea Reala

Un centru de apeluri modeleaza numarul de apeluri primite pe ora ca o variabila Poisson cu λ = 15. Aceasta permite planificarea personalului: probabilitatea de a primi mai mult de 25 de apeluri intr-o ora poate fi calculata direct din distributia Poisson, ajutand la dimensionarea echipei.

Capitolul 18: Modele spatiu de stare - Introducere

Aceasta pagina introduce Capitolul 18, care trateaza modelele spatiu de stare (SSM), generalizarea HMM-urilor la stari ascunse continue.

Model spatiu de stare (SSM): Similar cu HMM dar cu stari ascunse continue. Forma generica: z_t = g(u_t, z_{t-1}, epsilon_t) (modelul de tranzitie) si y_t = h(z_t, u_t, delta_t) (modelul de observatie), unde z_t este starea ascunsa, u_t este intrarea de control, y_t este observatia, epsilon_t si delta_t sunt zgomotele de sistem si observatie.
SSM linear-Gaussian (LG-SSM / LDS): Cazul special cu CPD-uri linear-Gaussiene: z_t = A_t z_{t-1} + B_t u_t + epsilon_t (tranzitie), y_t = C_t z_t + D_t u_t + delta_t (observatie), epsilon_t ~ N(0, Q_t), delta_t ~ N(0, R_t). Daca parametrii theta_t = (A_t, B_t, C_t, D_t, Q_t, R_t) sunt independenti de timp, modelul este stationar.

Comentariu

Rezumat

Capitolul 18 trateaza SSM-uri, care sunt HMM-uri cu stari ascunse continue z_t in R^n in loc de z_t in {1,...,K}. Forma generala: z_t = g(u_t, z_{t-1}, epsilon_t) pentru tranzitie si y_t = h(z_t, u_t, delta_t) pentru observatie. Scopul principal este estimarea recursiva a starii de credinta p(z_t|y_{1:t}, u_{1:t}, theta). LG-SSM (Linear Gaussian SSM), numit si LDS (Linear Dynamical System), este cazul special cu toate CPD-urile linear-Gaussiene, care suporta inferenta exacta prin filtrul Kalman.

Tipare Cheie

Dualitatea HMM-SSM: HMM are stari discrete si foloseste algoritmul forward-backward; SSM are stari continue si foloseste filtrul Kalman. LG-SSM este important deoarece daca starea initiala este Gaussiana p(z_1) = N(mu_{1|0}, Sigma_{1|0}), atunci toate starile de credinta ulterioare raman Gaussiene: p(z_t|y_{1:t}) = N(mu_{t|t}, Sigma_{t|t}).

Aplicatii Practice

SSM-uri sunt fundamentale in: ingineria de control (modelarea sistemelor dinamice), navigatie (GPS, INS), robotica (estimarea pozitiei si orientarii), econometrie (modele de factori dinamici), procesarea semnalelor (filtrarea zgomotului) si urmarirea obiectelor (tracking).

Exemplu din Lumea Reala

Filtrul Kalman, inventat in 1960, a fost utilizat in programul Apollo al NASA pentru navigatia spre Luna. Astazi, fiecare telefon mobil cu GPS foloseste o varianta a filtrului Kalman pentru a combina masuratorile GPS (zgomotoase) cu datele accelerometrului si giroscopului pentru o estimare precisa a pozitiei.

Regresia Kernel și Modelul Nadaraya-Watson (detalii)

Pagina detaliază derivarea matematică a regresiei kernel prin proprietățile kernelurilor de netezire, formula ponderilor și banda optimală.

Deviația absolută mediană (MAD): MAD = median(|x − median(x)|). Este un estimator robust al dispersiei, mai puțin sensibil la outlieri decât abaterea standard. Relația cu σ: σ̂ = 1.4826 · MAD.
Derivarea regresiei kernel: Pornind de la densitatea comună p(x,y) ≈ (1/N) Σᵢ κ_h(x − xᵢ)κ_h(y − yᵢ), se folosesc proprietățile ∫κ_h(y − yᵢ)dy = 1 și ∫yκ_h(y − yᵢ)dy = yᵢ pentru a obține f(x) = Σᵢ κ_h(x − xᵢ)yᵢ / Σᵢ κ_h(x − xᵢ).
Proprietatea mediei zero: ∫(x + yᵢ)κ_h(x)dx = ∫xκ_h(x)dx + yᵢ∫κ_h(x)dx = 0 + yᵢ = yᵢ, fundamentală pentru derivarea estimatorului Nadaraya-Watson.

Comentariu

Rezumat

Derivarea regresiei kernel se bazează pe aproximarea densității comune p(x,y) prin KDE și apoi calcularea speranței condiționate E[y|x]. Proprietățile kernelurilor de netezire (normalizare și media zero) sunt esențiale pentru simplificarea formulei. Banda h se poate estima separat pe fiecare dimensiune.

Tipare Cheie

Formula Nadaraya-Watson este o medie ponderată unde ponderile sunt normalize (sumă = 1). Kernelul κ nu trebuie neapărat să fie un kernel de netezire pentru formula generală. Metoda are un singur hiperparametru dar funcționează bine în practică.

Aplicatii Practice

Regresia kernel este utilă ca metodă exploratorie pentru a vizualiza tendințe în date înainte de a alege un model parametric. Este implementată în pachetele R (loess) și Python (statsmodels).

Exemplu din Lumea Reala

În analiza piețelor financiare, regresia kernel poate estima relația nelineară între volumul de tranzacționare și volatilitatea prețurilor, fără a impune o formă funcțională presupusă.

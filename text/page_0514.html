Kernelul Matern (continuare), Procesul Ornstein-Uhlenbeck si String Kernels

Pagina continua discutia despre kernelul Matern (cazuri speciale) si introduce Sectiunea 14.2.6 despre string kernels pentru compararea secventelor de lungime variabila.

Cazuri speciale Matern (Eq. 14.21): Cu nu = 1/2: kappa(r) = exp(-r/l) (proces Ornstein-Uhlenbeck). Cu D = 1, aceasta defineste un GP cu traiectorii continue dar nediferentiabile ("jagged"). Pe masura ce nu -> infinit, kernelul Matern converge la SE/RBF kernel (infinit neted).
String kernel (Sectiunea 14.2.6, Eq. 14.22): kappa(x, x') = suma_{s in A*} w_s phi_s(x) phi_s(x'), unde phi_s(x) numara aparitiile substringului s in x, A* este multimea tuturor stringurilor. Este Mercer, calculabil in O(|x| + |x'|) cu suffix trees. Variante: bag-of-characters (w_s = 0 pt |s| > 1), bag-of-words, k-spectrum kernel.
Exemplu secvente proteice: Doua secvente de aminoacizi de lungimi 110 si 153, definite pe alfabetul de 20 litere. Substringul comun "LQE" apare in ambele. k-spectrum kernel (k=3) numara substrings de lungime k comune, folosit pentru clasificarea proteinelor in superfamilii SCOP.

Comentariu

Rezumat

String kernels demonstreaza puterea kernelurilor pentru date structurate: permit compararea secventelor de lungimi diferite fara aliniere, folosind numarul de substrings comune ca masura de similaritate. Aceasta abordare este fundamentala in bioinformatica (secvente de proteine/ADN) si NLP (clasificarea textului).

Tipare Cheie

Ierarhia string kernels: bag-of-characters (cel mai simplu, pierde ordinea) -> bag-of-words (pastreaza cuvinte) -> k-spectrum (substrings de lungime k) -> mismatch kernel (permite erori). Fiecare nivel adauga complexitate si putere expresiva, dar si cost computational.

Aplicatii Practice

String kernels cu SVM au fost state-of-the-art in clasificarea proteinelor (remote homology detection) inainte de deep learning. Raman relevante cand datele sunt limitate. Implementari: shogun library, scikit-learn (cu kernel personalizat).

Exemplu din Lumea Reala

In securitatea cibernetica, string kernels sunt folosite pentru detectia de malware prin compararea secventelor de apeluri de sistem (system calls), identificand programe malitioase pe baza similaritatii comportamentale fara decompilare.

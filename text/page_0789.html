Capitolul 21. Inferenta Variationala - Limita Patratica a lui Bohning pentru Log-Sum-Exp (p.758)

Aceasta pagina deriveaza limita patratica a lui Bohning pentru functia log-sum-exp (lse), o aproximare superioara de tip gaussian care permite inferenta tractabila.

Expansiunea Taylor a lse: lse(ηᵢ) = lse(ψᵢ) + (ηᵢ-ψᵢ)ᵀg(ψᵢ) + 1/2(ηᵢ-ψᵢ)ᵀH(ψᵢ)(ηᵢ-ψᵢ), unde g(ψᵢ) = S(ψᵢ) = exp[ψᵢ-lse(ψᵢ)] (softmax) si H = diag(g) - ggᵀ.
Limita Bohning: Inlocuirea Hessianului H(ψᵢ) cu matricea constanta Aᵢ = 1/2[Iₘ - 1/(M+1)1ₘ1ₘᵀ] care satisface Aᵢ ≻ H(ψᵢ) (Bohning 1992). Avantajul: Aᵢ nu depinde de ψᵢ.
Limita superioara pe lse: lse(ηᵢ) ≤ 1/2 ηᵢᵀAᵢηᵢ - bᵢᵀηᵢ + cᵢ, cu bᵢ = Aᵢψᵢ - g(ψᵢ) si cᵢ = 1/2 ψᵢᵀAᵢψᵢ - g(ψᵢ)ᵀψᵢ + lse(ψᵢ).
Model de observatie „gaussianizat": p(yᵢ|xᵢ,w) ≥ f(xᵢ,ψᵢ) N(ỹᵢ|X,w,Aᵢ⁻¹), unde ỹᵢ = Aᵢ⁻¹(bᵢ+yᵢ) sunt pseudo-masuratori, permitand calcul bayesian standard cu gaussiene.

Comentariu

Rezumat

Limita Bohning transforma problema de inferenta variationala pentru regresie logistica multinomiala intr-o problema de regresie liniara gaussiana echivalenta, prin inlocuirea Hessianului variabil cu o matrice constanta Aᵢ care margineaza superior curbura.

Tipare Cheie

Strategia cheie: inlocuirea unei functii non-conjugate cu o limita patratica care este conjugata cu priorul gaussian. Matricea Aᵢ = 1/2[I - 1/(M+1)11ᵀ] este independenta de parametrii variationali, simplificand enorm calculul. Pseudo-masuratorile ỹᵢ transforma problema in regresie liniara standard.

Aplicatii Practice

Limita Bohning este larg utilizata in clasificarea bayesiana multinomiala, regresie logistica ordinala, si modele de alegere discreta in economie si marketing.

Exemplu din Lumea Reala

In diagnosticul medical automat cu mai multe categorii de boli, limita Bohning permite estimarea incertitudinii in clasificare, informand medicul nu doar despre diagnosticul cel mai probabil, ci si despre cat de sigur este modelul.

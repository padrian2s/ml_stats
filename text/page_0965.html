Rezolvarea paradoxului Simpson si invatarea structurilor DAG cauzale

Aceasta pagina completeaza demonstratia matematica a rezolvarii paradoxului lui Simpson si prezinta doua modele cauzale alternative (Figura 26.16) si consecintele lor diferite.

DAG cauzal cu confunder (Figura 26.16a): F (gen) -> C si F -> E. Trebuie ajustat pe F. Concluzia: p(E|do(C)) < p(E|do(not C)), medicamentul dauneaza.
DAG cauzal cu mediator (Figura 26.16b): C -> F (tensiune) si F -> E. Nu trebuie ajustat pe F (blocheaza calea cauzala). Efectul poate fi pozitiv.
Concluzie fundamentala: Presupuneri cauzale diferite duc la concluzii diferite, chiar cu aceleasi date. Aceasta ridica intrebarea: putem invata structura cauzala din date?

Comentariu

Rezumat

Demonstratia formala arata ca daca modelul 26.16(a) este corect, p(E|do(C)) < p(E|do(not C)) (medicamentul dauneaza), deoarece fiecare termen in suma este mai mic. Dar daca modelul 26.16(b) este corect, concluzia se poate inversa. Aceasta motiveaza sectiunea urmatoare: invatarea structurilor DAG cauzale din date.

Tipare Cheie

Paradoxul lui Simpson demonstreaza ca statistica descriptiva (chiar conditionata) nu este suficienta pentru inferenta cauzala. Este nevoie de un model cauzal explicit, iar alegerea modelului afecteaza direct deciziile practice.

Aplicatii Practice

Aceasta lectie este critica in politica de sanatate publica: de exemplu, relatia intre un factor de risc si o boala poate parea diferita la nivel de populatie fata de nivel de subgrup, iar doar rationamentul cauzal poate ghida interventia corecta.

Exemplu din Lumea Reala

In analiza efectului exercitiului fizic asupra bolilor de inima, varsta este un confunder (afecteaza atat exercitiul cat si boala). Fara ajustare cauzala, putem concluziona gresit ca exercitiul este ineficient daca persoanele active sunt mai in varsta in esantion.

Trucul Kernel - Kernel PCA (Algoritmul 14.2)

Pagina prezintă algoritmul Kernel PCA și vizualizarea datelor 2D folosind PCA și kPCA, demonstrând diferențele dintre proiecțiile liniare și cele neliniare.

Kernel PCA: Algoritm care aplică PCA în spațiul de caracteristici indus de o funcție kernel, permițând reducerea dimensionalității neliniare. Se primește matricea kernel K de dimensiune N×N și se calculează vectorii proprii ai matricei kernel centrate.
Algoritmul 14.2: K̃ = K − OK − KO + OKO, unde O = 1_N · 1_N^T / N este matricea de centrare. Se calculează descompunerea spectrală [U, Λ] = eig(K̃), iar vectorii de proiecție sunt v_i = u_i / √Λ_i.

Comentariu

Rezumat

Algoritmul Kernel PCA extinde PCA clasic folosind trucul kernel pentru a opera în spații de caracteristici de dimensiune înaltă. Centrarea matricei kernel este esențială și se realizează prin formula K̃ = K − OK − KO + OKO. Deși kPCA poate fi utilă pentru clasificare, nu este întotdeauna optimă pentru vizualizare.

Tipare Cheie

Centrarea în spațiul de caracteristici se realizează implicit prin operații pe matricea kernel. PCA clasic poate fi superior kPCA pentru vizualizare când datele au o structură liniară naturală. Primele două componente principale separă cele trei clustere în ambele metode.

Aplicatii Practice

Kernel PCA se utilizează în recunoașterea facială, analiza imaginilor medicale și orice domeniu unde reducerea dimensionalității neliniare este necesară. Este frecvent utilizat ca etapă de preprocesare pentru clasificatori.

Exemplu din Lumea Reala

Într-un sistem de detectare a anomaliilor în rețele de calculatoare, kPCA poate proiecta traficul de rețea de înaltă dimensionalitate într-un spațiu de dimensiune redusă, unde tiparele anormale devin mai vizibile decât cu PCA liniar.

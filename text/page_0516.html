Kerneluri Derivate din Modele Generative: Probability Product si Fisher Kernels

Pagina prezinta Sectiunea 14.2.8 despre kerneluri derivate din modele generative probabilistice, cu doua abordari: probability product kernels si Fisher kernels.

Probability product kernel (Sectiunea 14.2.8.1, Eq. 14.23-14.24): kappa(x_i, x_j) = integral p(x|x_i)^rho p(x|x_j)^rho dx, unde p(x|x_i) = p(x|theta_hat(x_i)) cu theta_hat estimat dintr-un singur punct. Cu model Gaussian N(mu, sigma^2 I) si rho=1, mu(x_i)=x_i: kappa reduce la RBF kernel (cu factor constant).
Fisher kernel (Sectiunea 14.2.8.2, Eq. 14.25-14.27): kappa(x, x') = g(x)^T F^{-1} g(x'), unde g(x) = nabla_theta log p(x|theta)|_{theta_hat} (score vector) si F = nabla nabla log p(x|theta)|_{theta_hat} (informatia Fisher). Intuitie: doua puncte sunt similare daca ar muta parametrii in aceeasi directie.
Proprietati Fisher kernel: theta_hat depinde de toate datele, deci similaritatea este calculata in context global. Se antreneaza un singur model generativ. String kernel de la Sectiunea 14.2.6 este echivalent cu Fisher kernel al unui lant Markov de ordin L. TF-IDF kernel este aproximativ Fisher kernel al modelului compound Dirichlet multinomial.

Comentariu

Rezumat

Kernelurile derivate din modele generative sunt o punte eleganta intre abordarea generativa si cea discriminativa: se antreneaza un model generativ p(x|theta), apoi se foloseste geometria spatiului parametrilor (score, informatia Fisher) pentru a defini similaritati discriminative. Fisher kernel este deosebit de puternic deoarece captureaza informatia de curbura a likelihood-ului.

Tipare Cheie

Unificarea surprinzatoare: string kernel = Fisher kernel al lantului Markov, TF-IDF = Fisher kernel al compound Dirichlet multinomial. Aceasta sugereaza ca multe kerneluri "ad-hoc" au de fapt o justificare probabilistica profunda prin modele generative implicite.

Aplicatii Practice

Fisher vectors (extensia Fisher kernel) au fost state-of-the-art in clasificarea de imagini (ImageNet 2013) inainte de CNNs. Raman utile in NLP: Fisher kernel cu HMM pentru clasificarea secventelor biologice, sau cu GMM pentru recunoasterea vorbitorului.

Exemplu din Lumea Reala

In recunoasterea vorbitorului (speaker verification), Fisher kernel cu model GMM (Gaussian Mixture Model) al vocii transforma inregistrari audio de lungimi variabile in vectori de dimensiune fixa (i-vectors/Fisher vectors), permitand compararea eficienta a vorbitorilor.

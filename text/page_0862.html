Filtrarea particulelor Rao-Blackwellizata (RBPF)

Aceasta pagina introduce RBPF, o metoda care reduce varianta filtrului cu particule prin marginalizarea analitica a unor variabile latente.

RBPF (Rao-Blackwellised Particle Filter): Se partitioneaza variabilele ascunse in q_t (discrete, esantionate) si z_t (continue, marginalizate analitic). Fiecare particula contine o valoare q_{1:t}^s si o distributie p(z_t|y_{1:t}, q_{1:t}^s) reprezentata parametric. Se numesc particule distributionale sau collapsed particles.
RBPF pentru SLDS (Switching Linear Dynamical Systems): Fiecare particula esantioneaza starea discreta q_t, iar starea continua z_t este urmarita de un filtru Kalman. Ponderea w_t^s = w_{t-1}^s * L_{tk}^s, unde L_{tk}^s este verosimilitatea predictiva a filtrului Kalman.
Reducerea dimensionalitatii: Prin marginalizarea analitica a z_t, se reduce dimensionalitatea spatiului de esantionare, reducand varianta estimatorului conform teoremei Rao-Blackwell.

Comentariu

Rezumat

RBPF combina puterea filtrarii particulelor (pentru variabile discrete/nelineari) cu eficienta filtrarii Kalman (pentru variabile liniare-Gaussiene). Fiecare particula mentine un filtru Kalman, obtinand o aproximare "amestec de Gaussiene" a posteriorului.

Tipare Cheie

Principiul Rao-Blackwell: inlocuirea esantionarii cu calcul analitic reduce intotdeauna varianta. Deci cu cat mai multe variabile pot fi marginalizate analitic, cu atat RBPF este mai eficient.

Aplicatii Practice

RBPF este standard in SLAM (Simultaneous Localization and Mapping), unde starea discreta este pozitia robotului iar starea continua este harta mediului.

Exemplu din Lumea Reala

Algoritmul FastSLAM foloseste RBPF: fiecare particula reprezinta o traiectorie posibila a robotului, iar harta este mentinuta analitic (ca set de filtre Kalman) conditionat de traiectorie. Aceasta permite cartografierea simultana cu localizarea.

Capitolul 2: Distributiile Multinomiala si Multinoulli

Generalizarea distributiei binomiale la mai mult de doua rezultate: distributia multinomiala si varianta sa categorica.

Distributia Multinomiala: Mu(x|n, θ) = C(n; x₁...x_K) ∏ⱼ₌₁ᴷ θⱼ^xⱼ (ecuatia 2.33), unde x = (x₁,...,x_K) este vectorul de contorizari, θⱼ este probabilitatea fetei j si Σxⱼ = n.
Coeficientul Multinomial: C(n; x₁...x_K) = n! / (x₁!x₂!...x_K!) (ecuatia 2.34) - numarul de moduri de a imparti n elemente in K subseturi.
Distributia Multinoulli (Categorica): Cat(x|θ) = Mu(x|1, θ) (ecuatia 2.36) - cazul special cu n = 1, echivalent cu aruncarea unui zar cu K fete o singura data. Daca x ~ Cat(θ), atunci p(x = j|θ) = θⱼ.
Codificare One-Hot: Reprezentarea unei variabile categorice cu K stari ca un vector binar x ∈ {0,1}^K cu Σx_k = 1. Doar un singur element este 1 ("hot"), restul sunt 0.

Comentariu

Rezumat

Sectiunea 2.3.2 introduce distributia multinomiala, generalizarea distributiei binomiale la K > 2 rezultate posibile. Distributia multinoulli (categorica) este cazul special cu n = 1, modeland o singura observatie categorica. Codificarea one-hot transforma variabile categorice in vectori binari, facilitand calculele.

Tipare Cheie

Ierarhia Bernoulli → Binomiala → Multinoulli → Multinomiala formeaza o familie coerenta de distributii. Codificarea one-hot este omniprezenta in ML modern: retele neuronale, word embeddings si clasificarea multi-clasa o folosesc extensiv. Distributia categorica este baza clasificarii multi-clasa.

Aplicatii Practice

Distributia multinomiala modeleaza frecventele cuvintelor in documente (bag-of-words), distributia nucleotidelor in secvente ADN, si distributia categoriilor de produse in cosuri de cumparaturi. Codificarea one-hot este standard in preprocesarea datelor categorice.

Exemplu din Lumea Reala

Un model de limbaj simplu care estimeaza probabilitatea fiecarui cuvant intr-un vocabular de 50.000 de cuvinte foloseste distributia categorica: p(cuvant = j|context) = θⱼ. Fiecare cuvant generat este o singura extragere dintr-o distributie multinoulli cu 50.000 de categorii.

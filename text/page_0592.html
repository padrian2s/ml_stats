Algoritmul Gradient Boosting si Sparse Boosting

Pagina prezinta algoritmul complet de gradient boosting (Algoritm 16.4), legatura cu BinomialBoost, si introduce sparse boosting ca metoda de selectie a variabilelor.

Gradient Boosting (Algoritm 16.4): Algoritm general care: (1) initializeaza f₀, (2) la fiecare pas calculeaza reziduurile gradientului r_{im}, (3) potriveste clasificatorul slab pe reziduuri, (4) actualizeaza f_m = f_{m-1} + ν φ(x; γ_m).
BinomialBoost: Gradient boosting aplicat pe logloss, avand avantajul ca nu necesita potrivire ponderata - aplica un model de regresie black-box pe vectorul gradientului.
Sparse Boosting: Varianta in care clasificatorul slab cauta peste toate variabilele j = 1:D si alege variabila j(m) care prezice cel mai bine vectorul rezidual.
Selectia variabilei optime: j(m) = argmin_j Σ_{i=1}^{N} (r_{im} - β̂_{jm} x_{ij})² (Ec. 16.54), cu β̂_{jm} = Σ x_{ij} r_{im} / Σ x_{ij}² (Ec. 16.55)
Functia de baza: φ_m(x) = β̂_{j(m),m} x_{j(m)} (Ec. 16.56)

Comentariu

Rezumat

Algoritmul de gradient boosting este generic si functioneaza cu orice functie de pierdere diferentiabila. Cu pierdere patrata se obtine L2Boosting, cu logloss se obtine BinomialBoost. Sparse boosting selecteaza la fiecare pas o singura variabila, fiind identic cu algoritmul matching pursuit si echivalent cu LAR cand ν → 0.

Tipare Cheie

Avantajul BinomialBoost fata de LogitBoost: nu necesita potrivire ponderata, doar aplica un model de regresie pe vectorul gradientului. Sparse boosting poate folosi spline-uri de netezire in loc de regresie liniara, producand un model aditiv generalizat rar.

Aplicatii Practice

Sparse boosting este utilizat in genomica si proteomice pentru selectia genelor relevante dintr-un numar foarte mare de variabile (mii de gene, zeci de esantioane).

Exemplu din Lumea Reala

In analiza expresiei genice pentru diagnosticul cancerului, sparse boosting selecteaza iterativ genele cele mai informative din mii de gene candidate, construind un model de diagnostic precis si interpretabil.

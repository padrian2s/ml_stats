Teoria esantioanelor mari pentru MLE: normalitatea asimptotica si informatia Fisher

Aceasta pagina prezinta teoria asimptotica a MLE, demonstrand ca distributia de esantionare a MLE devine Gaussiana pentru esantioane mari, cu varianta determinata de informatia Fisher.

Functia scor: s(θ_hat) = ∇ log p(D|θ)|_θ_hat, gradientul log-verosimilitatii evaluat la estimarea θ_hat.
Matricea de informatie observata: J(θ_hat(D)) = -∇² log p(D|θ)|_θ_hat, Hessianul negativ al log-verosimilitatii. Masoara curbura suprafetei log-verosimilitatii.
Matricea de informatie Fisher: I_N(θ|θ*) = E_θ*[J(θ|D)], valoarea asteptata a informatiei observate. Se factorizeaza ca I_N(θ) = N I_1(θ), deci informatia creste liniar cu N.
Normalitatea asimptotica a MLE: θ_hat → N(θ*, I_N(θ*)^(-1)) cand N → ∞. MLE este asimptotic nedeplasat si eficient (atinge marginea inferioara Cramer-Rao). Varianta de esantionare este invers proportionala cu curbura log-verosimilitatii.

Comentariu

Rezumat

Pagina stabileste fundamentele teoretice ale statisticii frequentiste: pentru esantioane mari, MLE este aproximativ Gaussian cu media la valoarea adevarata si varianta data de inversul informatiei Fisher. Erorile standard se calculeaza din diagonala inversei matricii de informatie.

Tipare Cheie

1) Informatia Fisher = varianta functiei scor = valoarea asteptata a curbure log-verosimilitatii. 2) Informatia creste liniar cu N, deci eroarea standard scade cu 1/sqrt(N). 3) Normalitatea asimptotica necesita conditii de regularitate care nu sunt intotdeauna satisfacute (ex: modele de amestec). 4) Intuitia: o verosimilitate cu varf ascutit inseamna estimare precisa.

Aplicatii Practice

Erorile standard bazate pe informatia Fisher sunt raportate standard in regresia logistica, modelele de supravietuire (Cox), si modelele economice. Informatia Fisher este folosita in designul optimal de experimente (D-optimal design), in calculul marimii esantionului necesare, si in algoritmul de gradient natural.

Exemplu din Lumea Reala

Pentru un model Bernoulli cu parametru θ, informatia Fisher este I(θ) = 1/(θ(1-θ)). Aceasta este maxima cand θ = 0.5 (informatia per observatie este maxima cand evenimentul este cel mai incert) si tinde la infinit cand θ se apropie de 0 sau 1. Eroarea standard a MLE este sqrt(θ(1-θ)/N), exact formula din sondajele de opinie: cu N=1000 si θ=0.5, marja de eroare este ~1.6%.

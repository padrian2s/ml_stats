Regresia liniara Bayesiana: calculul posteriorului

Aceasta pagina prezinta calculul posteriorului Bayesian pentru coeficientii regresiei liniare cu varianta cunoscuta, aratand ca posteriorul este Gaussian si ca estimarea ridge este un caz special.

Verosimilitatea in regresia liniara: p(y | X, w, σ²) = N(y | Xw, σ² I_N). Datele centrate (y inlocuit cu y - y_bar) permit integrarea termenului de intercept.
Posteriorul Bayesian: p(w | X, y, σ²) = N(w | w_N, V_N), unde w_N = V_N(V_0^(-1) w_0 + (1/σ²) X^T y) si V_N^(-1) = V_0^(-1) + (1/σ²) X^T X.
Legatura cu regresia ridge: Daca w_0 = 0 si V_0 = τ² I, atunci media posterioara w_N devine identica cu estimarea ridge, cu λ = σ²/τ². Regularizarea ridge are deci o interpretare Bayesiana naturala: este MAP-ul sub un prior Gaussian cu medie zero.

Comentariu

Rezumat

Pagina derivese posterior Gaussian pentru coeficientii regresiei liniare. Media posterioara w_N este o combinatie ponderata intre priorul w_0 si estimarea OLS, cu ponderi date de matricea de precizie prior V_0^(-1) si matricea de precizie a datelor (1/σ²)X^T X. Covarianta posterioara V_N scade pe masura ce se acumuleaza date (inversul creste).

Tipare Cheie

1) Posteriorul este Gaussian deoarece atat priorul cat si verosimilitatea sunt Gaussiene (conjugare). 2) Media posterioara interpoleaza intre prior si date. 3) Cu date suficiente, posteriorul converge la MLE (OLS) indiferent de prior. 4) Parametrul de regularizare ridge λ = σ²/τ² arata compromisul: zgomot mare sau prior puternic = mai multa regularizare.

Aplicatii Practice

Regresia liniara Bayesiana este folosita in geostatistica (kriging), in calibrarea instrumentelor, in previziuni economice cu priori din teorie economica, si in meta-analiza (combinarea rezultatelor din studii diferite cu priori informative).

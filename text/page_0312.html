Capitolul 9: Modele liniare generalizate si familia exponentiala - Introducere

Aceasta pagina deschide Capitolul 9, care unifica multe distributii de probabilitate sub umbrela familiei exponentiale si introduce modelele liniare generalizate (GLM).

Familia exponentiala: Clasa de distributii de probabilitate care include Gaussiana, Bernoulli, Poisson, Gamma, Beta, si multe altele. Este importanta deoarece permite derivarea de teoreme si algoritmi cu aplicabilitate foarte larga.
De ce este importanta familia exponentiala: 1) Este singura familie cu statistici suficiente de dimensiune finita (Pitman-Koopman-Darmois). 2) Singura familie pentru care exista priori conjugate. 3) Face cele mai putine presupuneri sub constrangeri date (entropie maxima). 4) Este la baza GLM-urilor. 5) Este la baza inferentei variationlae.

Comentariu

Rezumat

Capitolul 9 este unul dintre cele mai fundamentale din carte, unificand o varietate mare de modele sub un cadru teoretic comun. Familia exponentiala nu este doar o curiozitate matematica, ci are consecinte practice profunde: permite invatare online eficienta (prin statistici suficiente), inferenta Bayesiana exacta (prin priori conjugate), si modele de regresie flexibile (prin GLM-uri).

Aplicatii Practice

Familia exponentiala fundamenteaza aproape toate modelele statistice standard: regresie liniara, logistica, Poisson, modele de supravietuire, si modele de amestec. GLM-urile sunt implementate in toate pachetele statistice majore (R glm(), Python statsmodels, SAS PROC GENMOD).

Exemplu din Lumea Reala

In epidemiologie, se folosesc diferite modele din familia exponentiala in functie de tipul de date: Poisson pentru numarul de cazuri de boala, binomial pentru rata de infectare, Gamma pentru durata spitalizarii. Toate acestea pot fi unificate si estimate in cadrul GLM, folosind aceeasi procedura de optimizare (IRLS) cu diferite functii de legatura.

Derivarea ecuatiilor filtrului Kalman si introducerea Kalman smoother

Aceasta pagina prezinta derivarea matematica detaliata a ecuatiilor filtrului Kalman si introduce algoritmul de netezire Kalman (Kalman smoother).

Derivarea filtrului Kalman: Din regula Bayes pentru Gaussiene (Ecuatia 4.125): precizia posterioara Sigma_t^{-1} = Sigma_{t|t-1}^{-1} + C_t^T R_t^{-1} C_t. Prin lema de inversare matriciala: Sigma_t = (I - K_t C_t) Sigma_{t|t-1}. Media posterioara: mu_t = Sigma_t C_t R_t^{-1} y_t + Sigma_t Sigma_{t|t-1}^{-1} mu_{t|t-1}, care se simplifica la mu_t = mu_{t|t-1} + K_t(y_t - C_t mu_{t|t-1}).
Kalman smoothing (netezire): Filtrul Kalman calculeaza p(z_t|y_{1:t}) - estimarea online. In setari offline, dorim p(z_t|y_{1:T}) - estimarea folosind toate datele. Netezirea reduce semnificativ incertitudinea (elipsoidele de covarianta sunt mai mici decat cele filtrate), ca ilustrat in Figura 18.1(c). Elipsoidele sunt mai mari la marginile traiectoriei, unde sunt mai putini vecini informativi.

Comentariu

Rezumat

Derivarea ecuatiilor filtrului Kalman foloseste regula Bayes pentru Gaussiene: priorul p(z_t|y_{1:t-1}) = N(mu_{t|t-1}, Sigma_{t|t-1}) se combina cu verosimilitatea p(y_t|z_t) = N(C_t z_t, R_t). Precizia posterioara este suma preciziilor (Ecuatia 18.44). Prin lema de inversare matriciala se obtine forma standard cu K_t. Derivarea mediei (Ecuatiile 18.47-18.55) arata cum se obtine formula clasica mu_t = mu_{t|t-1} + K_t(y_t - C_t mu_{t|t-1}). Sectiunea 18.3.2 introduce Kalman smoother-ul pentru estimari offline mai precise.

Tipare Cheie

Doua forme echivalente ale filtrului Kalman: (1) forma covariantei (standard): actualizeaza Sigma_t, robusta numeric pentru |z| mic; (2) forma informatiei: actualizeaza Lambda_t = Sigma_t^{-1}, mai naturala pentru fuziunea senzoriala (precizia se aduna). Lema de inversare matriciala (Woodbury) este instrumentul matematic central, permitand trecerea intre cele doua forme.

Aplicatii Practice

Derivarea arata ca filtrul Kalman este forma optima de actualizare Bayesiana pentru sisteme liniare Gaussiene. In practica, se alege forma (covarianta vs informatie) in functie de dimensiunile relative ale starii si observatiei. Kalman smoother-ul este folosit in: procesarea offline a datelor de navigatie, reconstructia traiectoriei, si analiza post-hoc a seriilor temporale.

Exemplu din Lumea Reala

In analiza datelor de zbor dupa un incident aviatic (black box analysis), Kalman smoother-ul este folosit pentru a reconstitui traiectoria exacta a avionului din inregistrarile senzorilor. Spre deosebire de filtrarea in timp real din cockpit, netezirea offline foloseste toate datele (inclusiv cele de dupa momentul de interes) pentru estimari mult mai precise ale pozitiei, vitezei si atitudinii avionului.

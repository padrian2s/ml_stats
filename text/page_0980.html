LDA, PLSI si modelul GaP / Factorizare matriciala non-negativa

Aceasta pagina prezinta relatia intre LDA si PLSI, introduce modelul Gamma-Poisson (GaP) si factorizarea matriciala non-negativa (NMF).

User Rating Profile (URP): Extensia LDA la raspunsuri categorice multiple: p(y_{i,1:R}|pi_i) = produs Cat(y_{il}|B^(r)*pi_i). Numit si "simplex factor model" (Bhattacharya si Dunson 2011).
Modelul Gamma-Poisson (GaP): z_i^+ ~ produs Ga(alpha_k, beta_k), p(n_i|z_i^+) = produs Poi(n_{iv}|b_v^T z_i^+). Variabilele latente sunt non-negative (dar nu suma la 1). Conditionat pe L_i fix, se reduce la mPCA.
NMF (Non-negative Matrix Factorization): Cazul special al GaP cu alpha_k = beta_k = 0 (fara prior propriu pe z_i^+). Nu este un model generativ propriu-zis, si algoritmul original (Lee si Seung 2001) este un EM degenerat predispus la overfitting.
Prior spike-and-gamma: p(z_{ik}^+) = rho_k * I(z_{ik}^+=0) + (1-rho_k)*Ga(z_{ik}^+|alpha_k, beta_k), incurajand raritate in reprezentarea latenta. Numit "conditional Gamma Poisson model".

Comentariu

Rezumat

Pagina conecteaza mai multe modele aparent diferite: LDA, NMF si GaP sunt toate forme de factorizare matriciala non-negativa cu priori diferite. NMF (Lee si Seung 2001) este popular dar nu are un model probabilistic complet. GaP (Canny 2004) adauga un prior Gamma propriu, iar conditionat pe lungimea fixa se reduce la mPCA. Priorul spike-and-gamma adauga raritate, incurajand reprezentari cu multi zero-uri.

Tipare Cheie

Unificarea LDA, NMF si GaP sub cadrul factorizarii non-negative cu priori diferite ofera o perspectiva clara: alegerea priorului determina proprietatile modelului (raritate, normalizare, flexibilitate). NMF fara prior este predispus la overfitting.

Aplicatii Practice

NMF este popular in procesarea semnalelor audio (separarea surselor sonore), analiza imaginilor (extragerea trasaturilor non-negative), si bioinformatica (identificarea semnaturilor mutationale in cancer).

Exemplu din Lumea Reala

In analiza genomica a cancerului, NMF este folosit standard pentru identificarea semnaturilor mutationale: fiecare semnatura (topic) este un profil de mutatii, iar fiecare tumora este o combinatie non-negativa de semnaturi, reflectand procesele mutagene active.

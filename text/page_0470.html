Selectia Modelului LASSO: Debiasing si Stabilitate

Pagina discuta utilizarea l1 pentru selectia modelului (nu doar regularizare), conceptul de debiasing, consistenta de selectie, Bootstrap Lasso (Bolasso) si selectia de stabilitate.

Consistenta de selectie (Model selection consistency): Proprietatea ca metoda recupereaza setul adevarat de variabile cand N -> infinit. LASSO nu este intotdeauna consistent in selectie; conditii restrictive pe matricea de design sunt necesare.
Bootstrap Lasso (Bolasso): Se ruleaza LASSO pe B subseturi bootstrap ale datelor, se intersecteaza seturile de variabile selectate, si se face OLS pe intersectie. Produce selectie consistenta sub conditii mai slabe decat LASSO simplu.
Selectia de stabilitate (Stability selection): Se ruleaza LASSO pe subseturi aleatoare si se calculeaza probabilitatile de selectie per variabila. Se selecteaza variabilele cu probabilitate peste un prag. Controleaza rata de fals pozitivi. Abordare Bayesiana aproximativa.

Comentariu

Rezumat

LASSO nu este ideal pentru selectia de variabile: bias-ul de shrinkage si lipsa consistentei de selectie sunt limitari cunoscute. Bolasso si selectia de stabilitate rezolva partial aceste probleme prin randomizarea procedurii de selectie si agregarea rezultatelor, producand selectii mai robuste.

Tipare Cheie

Distinctia selectie vs predictie: lambda optim pentru predictie (CV) nu este acelasi cu lambda optim pentru selectia corecta a variabilelor. Metodele de stabilitate adreseaza explicit obiectivul de selectie.

Aplicatii Practice

In practica, se recomanda: (1) LASSO cu CV pentru predictie, (2) Debiased LASSO sau Bolasso pentru selectia variabilelor, (3) Selectia de stabilitate pentru controlul fals pozitivilor.

Exemplu din Lumea Reala

In descoperirea de medicamente, selectia de stabilitate identifica tintele moleculare cu probabilitate ridicata de a fi relevante, reducand numarul de experimente costisitoare de validare necesare.

Analiza Corelatiei Canonice (CCA) si Introducere in Analiza Componentelor Independente (ICA)

Pagina prezinta CCA ca versiune simetrica nesupervizata a PLS si introduce ICA prin problema petrecerii cocktail.

Analiza Corelatiei Canonice (CCA): Un model simetric care permite fiecarei vizualizari sa aiba propriul subspatiu „privat", pe langa un subspatiu partajat comun. Variabilele latente sunt: zₛ (partajat), zₓ (privat pentru x), zᵧ (privat pentru y).
Analiza Componentelor Independente (ICA): O metoda de separare oarba a surselor care descompune semnale observate amestecate in surse independente non-gaussiene, rezolvand problema petrecerii cocktail.
Problema petrecerii cocktail (Blind Source Separation): Deconvoluarea semnalelor mixate liniar in componentele lor sursa, fara cunoasterea matricei de amestecare W.
Modelul CCA: p(zᵢ) = N(zₛ|0,I_L₀)N(zₓ|0,I_Lx)N(zᵧ|0,I_Ly); p(xᵢ|zᵢ) = N(Bₓzₓ + Wₓzₛ + μₓ, σ²I); p(yᵢ|zᵢ) = N(Bᵧzᵧ + Wᵧzₛ + μᵧ, σ²I)
Modelul ICA: xₜ = Wzₜ + εₜ, unde W este matricea de amestecare si zₜ sunt sursele independente.

Comentariu

Rezumat

CCA generalizeaza PLS adaugand subspatii private pentru ambele vizualizari, creand o descompunere simetrica a informatiei in componente partajate si private. Avantajele perspectivei probabilistice includ: generalizarea la M > 2 vizualizari, mixturi CCA, versiuni sparse cu ARD, extensii la familia exponentiala si inferenta bayesiana. ICA abordeaza o problema complementara: in loc sa caute dimensiuni de maxima varianta (PCA), cauta surse statistice independente.

Tipare Cheie

CCA si PLS difera in simetrie: CCA trateaza ambele vizualizari egal, PLS este asimetric. ICA difera fundamental de PCA prin priorul pe sursele latente: PCA foloseste gaussiana (care nu poate identifica unic sursele), ICA impune non-gaussianitate si independenta.

Aplicatii Practice

CCA este utilizata in integrarea datelor multi-omice, alinierea limbilor in traducere automata, si analiza datelor neuroimagistice multi-modale. ICA este esentiala in procesarea semnalelor EEG/MEG, analiza imaginilor fMRI si separarea surselor audio.

Exemplu din Lumea Reala

Intr-o sala de conferinte cu mai multe microfoane, ICA poate separa vocile vorbitorilor individuali din inregistrarile mixate, permitand transcrierea automata a fiecarui vorbitor separat - exact problema petrecerii cocktail.

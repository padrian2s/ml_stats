MLE pentru Distributia Gaussiana Multivariata

Pagina 99 prezinta estimarea prin maximum likelihood (MLE) a parametrilor distributiei Gaussiene multivariate, inclusiv demonstratia bazata pe calculul matriceal.

Teorema 4.1.1 (MLE pentru Gaussiana): Pentru N esantioane i.i.d. x_i ~ N(μ, Σ), estimarile MLE sunt: μ̂_mle = (1/N) Σᵢ xᵢ = x̄ (media empirica) si Σ̂_mle = (1/N) Σᵢ (xᵢ - x̄)(xᵢ - x̄)ᵀ (covarianta empirica).
Cazul univariat (Ecuatiile 4.8-4.9): μ̂ = (1/N) Σᵢ xᵢ = x̄ si σ̂² = (1/N) Σᵢ (xᵢ - x̄)² = (1/N) Σᵢ xᵢ² - x̄²
Identitati de calcul matriceal (Ecuatia 4.10): ∂(bᵀa)/∂a = b; ∂(aᵀAa)/∂a = (A + Aᵀ)a; ∂tr(BA)/∂A = Bᵀ; ∂log|A|/∂A = A⁻ᵀ; tr(ABC) = tr(CAB) = tr(BCA) (proprietatea de permutare ciclica)
Trucul urmei (trace trick): xᵀAx = tr(xᵀAx) = tr(xxᵀA) = tr(Axxᵀ), transformand un produs scalar intr-o urma de matrice, util pentru derivarea MLE.

Comentariu

Rezumat

Pagina prezinta rezultatul fundamental ca MLE pentru o distributie Gaussiana multivariata este pur si simplu media si covarianta empirica. Demonstratia necesita identitati de algebra liniara si calcul matriceal, incluzand derivate de urme, determinanti si forme patratice.

Tipare Cheie

Trucul urmei (trace trick) este o tehnica eleganta care permite reordonarea termenilor intr-un produs scalar prin transformarea in operatia de urma. Proprietatea de permutare ciclica a urmei (tr(ABC) = tr(CAB)) este fundamentala in multe derivari din invatarea automata.

Aplicatii Practice

MLE pentru Gaussiana este utilizat in orice model care presupune date distribuite normal: regresie liniara, analiza discriminanta, modele de amestec Gaussiene. Estimatorul MLE al covariantei este biased (se prefera impartirea la N-1 pentru estimatorul nebiased).

Exemplu din Lumea Reala

In procesarea semnalelor, cand estimam spectrul de putere al unui semnal, calculam media si covarianta empirica din esantioanele colectate. Aceste estimari MLE sunt apoi folosite pentru filtrarea optima Wiener sau pentru detectia semnalelor.

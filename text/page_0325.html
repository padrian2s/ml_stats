Regresia probit: estimare ML/MAP si interpretarea cu variabile latente

Aceasta pagina detaliaza estimarea MLE/MAP pentru regresia probit si introduce interpretarea profunda prin modelul de utilitate aleatoare (Random Utility Model - RUM).

Gradientul regresiei probit: g_i = x_i y_tilde_i φ(μ_i)/Φ(y_tilde_i μ_i), unde μ_i = w^T x_i, φ este PDF-ul normal standard, Φ este CDF-ul, si y_tilde_i ∈ {-1, +1}.
Modelul de utilitate aleatoare (RUM): u_{0i} = w_0^T x_i + δ_{0i} si u_{1i} = w_1^T x_i + δ_{1i} sunt utilitatile celor doua actiuni. y_i = I(u_{1i} > u_{0i}). Daca δ ~ N(0,1), se obtine modelul probit. Daca δ ~ Logistic, se obtine regresia logistica.
Echivalenta RUM-Probit: Definind z_i = u_{1i} - u_{0i} = w^T x_i + ε_i cu ε_i ~ N(0,1), avem y_i = I(z_i ≥ 0). Marginalizand peste z_i: p(y_i=1|x_i, w) = P(w^T x_i + ε ≥ 0) = P(ε ≥ -w^T x_i) = Φ(w^T x_i). Exact modelul probit.

Comentariu

Rezumat

Pagina completeaza prezentarea regresiei probit cu doua perspective. Prima este computationala: gradientul si Hessianul se calculeaza prin regula lantului, iar estimarea MAP se obtine adaugand termenul de penalizare Gaussiana. A doua este conceptuala: modelul de utilitate aleatoare (RUM) ofera o interpretare economica profunda, unde alegerea observata reflecta compararea a doua utilitati perturbate de zgomot Gaussian.

Tipare Cheie

1) In RUM, zgomotul δ reprezinta factori neobservati care influenteaza decizia. 2) Doar diferenta de utilitati conteaza, reducand 2K parametri la K. 3) Distributia zgomotului determina modelul: Gaussian = probit, logistic = logit, Gumbel = log-log. 4) Interpretarea cu variabile latente faciliteaza extensii: modele multinomiale, modele ierarhice, modele cu panouri.

Aplicatii Practice

RUM este fundamentul economiei alegerilor discrete (Daniel McFadden, premiul Nobel 2000). Se aplica in transporturi (alegerea modului de transport), marketing (alegerea produsului), economie imobiliara (alegerea locuintei), si politici publice (evaluarea proiectelor de infrastructura).

Exemplu din Lumea Reala

Daniel McFadden a folosit modelul de utilitate aleatoare pentru a prezice cererea pentru sistemul de transport rapid BART din San Francisco, inainte de constructia sa. Modelul a estimat utilitatile diferitelor moduri de transport (masina, autobuz, BART) in functie de timp, cost si confort, si a prezis corect proportia calatorilor care vor folosi BART - o realizare remarcabila care a demonstrat puterea cadrului RUM.

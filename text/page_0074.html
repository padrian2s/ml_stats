Distributia Beta (continuare) si distributia Pareto

Aceasta pagina completeaza discutia despre distributia Beta cu proprietatile sale si diferitele forme, apoi introduce distributia Pareto pentru modelarea cozilor grele si a legilor de putere.

Proprietatile distributiei Beta: media = a/(a+b), modul = (a-1)/(a+b-2), varianta = ab/((a+b)²(a+b+1)). Daca a, b < 1, distributia este bimodala cu varfuri la 0 si 1. Daca a, b > 1, este unimodala. Daca a = b = 1, se obtine distributia uniforma.
Distributia Pareto: Pareto(x|k, m) = k m^k x^(-(k+1)) I(x ≥ m), unde k controleaza cat de grea este coada, iar m este valoarea minima. Pe scala log-log, densitatea formeaza o linie dreapta (lege de putere).
Legea lui Zipf: Daca plotam frecventa cuvintelor dintr-un text in functie de rangul lor, obtinem o lege de putere (distributie Pareto). Acelasi tipar apare in distributia averii, dimensiunea oraselor si multe alte fenomene naturale si sociale.

Comentariu

Rezumat

Pagina prezinta proprietatile complete ale distributiei Beta si diversele sale forme in functie de parametri, ilustrate in Figura 2.10. Apoi introduce distributia Pareto, fundamentala pentru modelarea fenomenelor cu cozi grele si legi de putere. Proprietatile Pareto includ media = km/(k-1) pentru k > 1 si varianta = m²k/((k-1)²(k-2)) pentru k > 2.

Tipare Cheie

1) Flexibilitatea distributiei Beta: poate fi uniforma, unimodala, bimodala, sau concentrata la extreme, in functie de parametri. 2) Distributia Pareto este echivalentul pentru cozi grele al distributiei exponentiale. 3) Legile de putere apar in mod surprinzator in multe domenii diferite (lingvistica, economie, retele sociale). 4) Pe scala log-log, Pareto produce o linie dreapta - un test vizual simplu pentru legi de putere.

Aplicatii Practice

Distributia Pareto este esentiala in modelarea riscului financiar (Value-at-Risk), in dimensionarea retelelor (traficul internet urmeaza legi de putere), si in stiintele sociale (distributia veniturilor). Distributia Beta este utilizata in testarea A/B (modelarea ratelor de conversie) si in planificarea de proiecte (estimarea PERT).

Exemplu din Lumea Reala

Distributia averii in societate urmeaza o distributie Pareto: un numar mic de persoane detin o proportie disproportionat de mare din averea totala.

Graphcuts pentru MRF-uri Metrice Non-Binare (Sectiunea 22.6.3.3)

Aceasta pagina prezinta constructia ponderilor muchiilor pentru MRF-uri binare submodulare si extinderea graphcuts la MRF-uri cu stari multiple folosind expansiunea alfa.

Ponderi muchii submodulare: E'_u,v(0,1) = E_u,v(1,0) + E_u,v(0,1) - E_u,v(0,0) - E_u,v(1,1) (Ec. 22.156)
Actualizari energetice: E'_u(1) = E'_u(1) + (E_u,v(1,0) - E_u,v(0,0)) si E'_v(1) = E'_v(1) + (E_u,v(1,1) - E_u,v(1,0)) (Ec. 22.157-22.158)
MRF Metric: Un MRF in care energiile de perechi formeaza o metrica. Se poate defini E(x_s, x_t) = min(delta, |x_s - x_t|) sau semi-metrica E(x_s, x_t) = min(delta, (x_s - x_t)^2) pentru o constanta delta > 0.
Expansiunea Alfa (Alpha Expansion): O versiune a graphcuts care la fiecare pas selecteaza o eticheta alfa si rezolva o sub-problema binara: fiecare variabila poate ramane in starea curenta sau sa devina starea alfa. (Ec. 22.159-22.160)
Conditia de submodularitate pentru expansiune: E_u,v(x_u, x_v) + E_u,v(alfa, alfa) <= E_u,v(x_u, alfa) + E_u,v(alfa, x_v) (Ec. 22.161) - satisfacuta de orice functie de distanta datorita inegalitatii triunghiului.

Comentariu

Rezumat

Pagina arata cum se construiesc ponderile muchiilor pentru graphcuts in cazul submodular si introduce expansiunea alfa pentru MRF-uri cu stari multiple. Expansiunea alfa transforma problema multi-eticheta intr-o serie de sub-probleme binare, fiecare rezolvabila exact prin graphcuts.

Tipare Cheie

Tranzitia de la MRF-uri binare la cele cu stari multiple necesita o descompunere iterativa. Expansiunea alfa este aplicabila oricarui MRF metric datorita inegalitatii triunghiului. Termenul delta previne supra-netezirea prin limitarea penalizarii pentru diferente mari.

Aplicatii Practice

MRF-urile metrice sunt esentiale in estimarea adancimii stereo, segmentarea semantica si reconstrucerea scenelor 3D. Expansiunea alfa este unul dintre cele mai utilizate algoritmi in viziunea computationala datorita eficientei si calitatii solutiilor.

Exemplu din Lumea Reala

In estimarea adancimii stereo, fiecare pixel primeste o eticheta de disparitate (de exemplu, 0-255 nivele). Expansiunea alfa itereaza prin fiecare nivel de disparitate, permitand pixelilor sa-si schimbe eticheta la nivelul curent daca aceasta reduce energia globala, producand harti de adancime netede si precise.

HMM factorial si HMM cuplat

Aceasta pagina introduce HMM-urile factoriale si cuplate, care folosesc reprezentari distribuite ale starii ascunse pentru a creste expresivitatea.

HMM factorial: Starea ascunsa este reprezentata ca mai multi lanturi Markov paralele: z_t = (z_{1,t}, z_{2,t},...,z_{C,t}), fiecare z_{c,t} in {1,...,K_c}. Figura 17.19(a): C lanturi independente, fiecare influentand observatia x_t.
Reprezentare distribuita: Cu C lanturi binare, se pot reprezenta 2^C stari cu doar 2C parametri de tranzitie, in loc de 2^{2C} pentru un HMM standard cu acelasi numar de stari. Inferenta exacta este O(K^{2C} T), necesitand aproximari.

Comentariu

Rezumat

HMM-ul standard codifica starea ca un singur numar z_t in {1,...,K}; 10 biti de informatie necesita K=1024 stari. HMM factorial foloseste C lanturi paralele, fiecare cu K_c stari. Figura 17.19(a) arata 3 lanturi independente care influeteaza observatia. HMM cuplat (Figura 17.19b) permite interactiuni intre lanturi: z_{c,t} depinde de z_{c',t-1}. Inferenta exacta este intractabila pentru C mare, necesitand aproximari (variational, particle filtering).

Tipare Cheie

Eficienta parametrica: C lanturi cu K_c stari fiecare necesita sum(K_c^2) parametri de tranzitie vs (prod K_c)^2 pentru HMM aplatizat. Dezavantajul: inferenta exacta este exponentiala in C. Aproximarile variatioanale (mean field) decupleaza lanturile, facand inferenta O(sum K_c^2 * T).

Aplicatii Practice

HMM factorial este folosit in: separarea surselor audio (fiecare lant modeleaza o sursa sonora), modelarea video (lanturi separate pentru fundal, obiecte, iluminare), si bioinformatica (lanturi separate pentru diferite proprietati genomice).

Exemplu din Lumea Reala

In separarea muzicala (demixarea unui mix audio in instrumente individuale), fiecare lant al HMM-ului factorial modeleaza un instrument, permitand separarea si re-mixarea independenta a vocii, chitarei, basului si tobelor.

Exercitii Capitolul 13: Derivata RSS, Pasul M pentru EB si Punct Fix

Pagina incepe sectiunea de exercitii pentru Capitolul 13, cu exercitiile 13.1-13.3 care acopera derivata partiala a RSS, pasul M pentru EB si actualizarile cu punct fix.

Exercitiul 13.1 - Derivata partiala a RSS (Eq. 13.180-13.184): RSS(w) = ||Xw - y||_2^2. (a) d RSS/d w_k = a_k w_k - c_k, cu a_k = 2||x_{:,k}||^2 si c_k = 2 x_{:,k}^T r_k. (b) Cand d RSS/d w_k = 0: w_hat_k = x_{:,k}^T r_k / ||x_{:,k}||^2 (proiectia ortogonala pe residual).
Exercitiul 13.2 - Derivarea pasului M pentru EB (Eq. 13.185-13.188): Derivarea ecuatiilor 13.166 si 13.168. Identitate utila: Sigma X^T X = (I - A Sigma) beta^{-1}. Aceasta permite simplificarea expresiilor pentru actualizarile alpha si beta in algoritmul EM.
Exercitiul 13.3 - Derivarea actualizarilor cu punct fix (Eq. 13.189): Derivarea ecuatiilor 13.169 si 13.170. Indiciu: rescrierea log p(D|alpha, beta) ca (N/2) log beta - (beta/2)||y - Xw||^2 + (1/2) suma log alpha_j - (1/2) m^T A m + (1/2) log|Sigma| - (D/2) log(2 pi).

Comentariu

Rezumat

Exercitiile 13.1-13.3 solidifica intelegerea mecanismelor computationale din capitolul 13. Exercitiul 13.1 deriveaza formulele de baza pentru coordinate descent (proiectia ortogonala pe residual). Exercitiile 13.2-13.3 deriveaza algoritmii EM si punct fix pentru ARD, demonstrand echivalenta lor la convergenta.

Tipare Cheie

Formula w_hat_k = x_{:,k}^T r_k / ||x_{:,k}||^2 din exercitiul 13.1 este exact pasul de coordinate descent pentru LASSO (inainte de soft thresholding). Identitatea Sigma X^T X = (I - A Sigma) beta^{-1} este fundamentala pentru derivarea eficienta a actualizarilor Bayesiene.

Aplicatii Practice

Aceste derivari sunt esentiale pentru implementarea corecta a algoritmilor. Formula derivatei partiale a RSS (Ex. 13.1) este baza ISTA/FISTA. Identitatea matriceala din Ex. 13.2 permite implementari eficiente fara inversari de matrice repetate.

Exemplu din Lumea Reala

In implementarea eficienta a LASSO pentru date genomice (milioane de SNP-uri), formula de proiectie ortogonala din Ex. 13.1 permite actualizarea incrementala a fiecarui coeficient in O(N), fara recalcularea intregului model.

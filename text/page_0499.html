Penalitatea Reweighted l1, ARD pentru Regresie Logistica si Introducere in Sparse Coding

Pagina prezinta formula penalitatii reweighted l1 (Eq. 13.173), ARD pentru regresie logistica (Sectiunea 13.7.5) si incepe Sectiunea 13.8 despre sparse coding.

Penalitatea reweighted l1 (Eq. 13.173): lambda_j <- [X_{:,j}^T (sigma^2 I + X diag(1/lambda_j) diag(|w_j^{(t+1)}|))^{-1} X^T)^{-1} X_{:,j}]^{1/2}. Penalitatea depinde de toate ponderile curente (nu doar w_j), spre deosebire de adaptive LASSO.
ARD pentru regresie logistica (Sectiunea 13.7.5): p(y|x, w) = Ber(y|sigm(w^T x)), p(w) = N(w|0, A^{-1}). Priorul Gaussian nu este conjugat cu logistic likelihood, deci pasul E necesita aproximare variationala sau Laplace (Sectiunea 8.4.1). Alternativ, reweighted l1 evita aproximari.
Sparse coding (Sectiunea 13.8): Extensie a ICA cu prior sparse (Laplace) pe factorii latenti z_i si dictionar W non-ortogonal. Fiecare observatie x_i este aproximata ca combinatie liniara sparsa de coloane (atomi) din dictionarul W. Reprezentarea este de obicei supracompleta (L > D).

Comentariu

Rezumat

Pagina face tranzitia de la ARD (regularizare sparse pentru modele supervizate) la sparse coding (regularizare sparse pentru modele nesupervizate). ARD pentru clasificare necesita aproximari (Laplace sau variationala) datorita non-conjugarii, dar algoritmul reweighted l1 ofera o alternativa exacta. Sparse coding generalizeaza ICA relaxand constrangerea de ortogonalitate a dictionarului.

Tipare Cheie

Diferenta cheie intre adaptive LASSO si reweighted l1 ARD: adaptive LASSO actualizeaza lambda_j bazat doar pe |w_j|, in timp ce ARD foloseste informatia globala din toate ponderile. Aceasta face ARD mai robust in cazul coeficientilor corelati, dar mai costisitor computational.

Aplicatii Practice

Sparse coding este fundamental in procesarea de imagini si semnale: dictionarele invatate din date (data-driven) depasesc in performanta bazele fixe (wavelet, DCT) pentru sarcini de denoising, inpainting si super-rezolutie.

Exemplu din Lumea Reala

In recunoasterea faciala, sparse coding reprezinta fiecare fata de test ca combinatie sparsa a fetelor de antrenare, clasificand-o bazat pe clasa care produce cea mai mica eroare de reconstructie (Wright et al., 2009).

Exemple: Marginale si Conditionale ale unei Gaussiene 2D

Pagina 112 ilustreaza cu un exemplu concret 2D cum functioneaza marginalele si conditionalele Gaussiene, inclusiv efectul corelatiei asupra incertitudinii.

Figura 4.9: (a) Distributia Gaussiana comuna p(x₁, x₂) cu coeficient de corelatie ρ = 0.8, conturul de 95% si axele principale. (b) Marginala neconditionata p(x₁) - o Gaussiana 1D. (c) Conditionala p(x₁|x₂ = 1) = N(x₁|0.8, 0.36) - mai ingusta decat marginala, deoarece observarea lui x₂ reduce incertitudinea.
Conditionala 2D (Ecuatia 4.72): p(x₁|x₂) = N(x₁|μ₁ + ρσ₁σ₂/σ₂² (x₂ - μ₂), σ₁² - (ρσ₁σ₂)²/σ₂²)
Cazul σ₁ = σ₂ = σ (Ecuatia 4.73): p(x₁|x₂) = N(x₁|μ₁ + ρ(x₂ - μ₂), σ²(1 - ρ²)). Cu ρ = 0.8 si x₂ = 1: E[x₁|x₂=1] = 0.8, var[x₁|x₂=1] = 1 - 0.64 = 0.36.
Interpolarea datelor fara zgomot (Sectiunea 4.3.2.2): Se doreste estimarea unei functii 1D f(t) din N observatii exacte. Se discretizeaza functia in D puncte si se defineste xⱼ = f(sⱼ), sⱼ = jh, h = T/D.

Comentariu

Rezumat

Exemplul 2D face formulele abstracte tangibile. Cand ρ = 0.8, observarea ca x₂ = 1 deplaseaza media lui x₁ de la 0 la 0.8 (regresie catre media) si reduce varianta de la 1 la 0.36. Cu cat corelatia este mai mare, cu atat observatia reduce mai mult incertitudinea. Sectiunea de interpolare introduce aplicarea acestor formule la estimarea functiilor.

Tipare Cheie

Factorul de reducere a variantei este (1 - ρ²): pentru ρ = 0 nu se reduce deloc, pentru ρ = 1 varianta devine zero (determinare perfecta). Aceasta este esenta "invatarii" intr-un model Gaussian: observatiile corelate reduc incertitudinea.

Aplicatii Practice

Interpolarea Gaussiana este precursorul proceselor Gaussiene, utilizate in optimizarea bayesiana (tuning hiperparametri ML), geostatistica (kriging) si modelarea suprafetelor 3D din scanari laser.

Exemplu din Lumea Reala

In monitorizarea calitatii aerului, senzori plasati in diferite locatii masoara poluarea. Datorita corelatiei spatiale, masurarea ridicata la un senzor (x₂ = mare) deplaseaza estimarea la senzorul vecin (E[x₁|x₂]) in sus, cu incertitudine redusa proportional cu ρ².

Filtrul Kalman - Verosimilitatea marginala, predictia posterioara si aspecte computationale

Aceasta pagina accopera calculul verosimilitatii marginale, distributia predictiva posterioara si problemele computationale ale filtrului Kalman.

Verosimilitatea marginala: log p(y_{1:T}|u_{1:T}) = sum_t log p(y_t|y_{1:t-1}, u_{1:t}), unde fiecare termen este Gaussian: p(y_t|y_{1:t-1}, u_{1:t}) = N(y_t|C_t mu_{t|t-1}, S_t). Aceasta permite estimarea hiperparametrilor prin maximizare (ex: Q, R, parametrii modelului de tranzitie).
Aspecte computationale: Doua costuri dominante: (1) inversarea matriciala pentru K_t: O(|y_t|^3), (2) inmultirea matriciala pentru Sigma_t: O(|z_t|^2). Pentru sisteme invariante in timp (theta_t = theta), ecuatiile Ricatti pentru Sigma_t converg la un punct fix, permitand precomputarea castigului K. Variante numerice stabile: filtrul informational (Lambda_t = Sigma_t^{-1}, eta_t = Lambda_t mu_t) si filtrul radacina patrata (descompunere Cholesky).

Comentariu

Rezumat

Verosimilitatea marginala log p(y_{1:T}) se descompune ca o suma de log-verosimilitudini predictive one-step-ahead, fiecare fiind Gaussiana cu media C mu_{t|t-1} si covarianta S_t. Aceasta permite: (1) estimarea hiperparametrilor prin MLE, (2) selectia de modele prin compararea verosimilitatilor. Distributia predictiva posterioara p(y_{t+1}|y_{1:t}) = N(y_{t+1}|C mu_{t|t-1}, C Sigma_{t|t-1} C^T + R) este utila pentru prognoza. Computational, filtrul Kalman are complexitate O(|z|^3) per pas. Pentru sisteme stationare, ecuatiile Ricatti converg si K devine constant (steady-state Kalman gain).

Tipare Cheie

Proprietate remarcabila a LG-SSM: K_t nu depinde de observatiile y_{1:t}, ci doar de modelul (A, C, Q, R) si de incertitudinea initiala. Pentru sisteme stationare, K converge la o valoare fixa independenta de date. Aceasta permite precomputarea offline a castigului, reducand costul online la O(|z|^2). Ecuatiile Ricatti sunt analoge ecuatiilor de recurenta pentru probabilitatile forward in HMM.

Aplicatii Practice

Filtrul informational este preferat cand observatiile sunt de dimensiune mai mica decat starea (|y| < |z|), cum e in SLAM. Filtrul radacina patrata este preferat pentru stabilitate numerica in sisteme de navigatie cu precizie inalta (avioane, rachete). Verosimilitatea marginala este folosita in: selectia ordinului modelului, estimarea zgomotelor Q si R, si validarea modelului.

Exemplu din Lumea Reala

In controlul de zbor al avioanelor comerciale, filtrul Kalman ruleaza in timp real cu castig Kalman pre-calculat (steady-state), deoarece dinamica avionului este bine cunoscuta si stationara in regim de croaziera. Castigul K este calculat offline pe baza modelului aerodinamic certificat, iar filtrul online necesita doar inmultiri matriciale simple, asigurand conformitatea cu cerintele de timp real ale sistemelor avionice.

Algoritmul RTS Smoother si derivarea sa

Aceasta pagina prezinta algoritmul de netezire Rauch-Tung-Striebel (RTS) si derivarea sa matematica.

RTS Smoother: Calculeaza p(z_t|y_{1:T}) = N(mu_{t|T}, Sigma_{t|T}) prin trecere backward: mu_{t|T} = mu_{t|t} + J_t(mu_{t+1|T} - mu_{t+1|t}), Sigma_{t|T} = Sigma_{t|t} + J_t(Sigma_{t+1|T} - Sigma_{t+1|t})J_t^T, unde J_t = Sigma_{t|t} A_{t+1}^T Sigma_{t+1|t}^{-1} este matricea de castig backward. Se initializeaza cu mu_{T|T} si Sigma_{T|T} din filtrul Kalman.
Derivarea RTS: Se exploateaza proprietatea Markov: z_t este independent de y_{t+1:T} conditionat pe z_{t+1}. Se conditioneaza pe z_{t+1} si se integreaza: p(z_t|y_{1:T}) = integral p(z_t|y_{1:t}, z_{t+1}) p(z_{t+1}|y_{1:T}) dz_{t+1}. Distributia filtrata bi-felie p(z_t, z_{t+1}|y_{1:t}) este Gaussiana, permitand conditionarea in forma inchisa.

Comentariu

Rezumat

RTS smoother-ul (Rauch, Tung, Striebel 1965) este analogul continuu al algoritmului backward din HMM. Opereaza in doua faze: (1) Forward pass: ruleaza filtrul Kalman standard, salvand mu_{t|t} si Sigma_{t|t} pentru fiecare t. (2) Backward pass: pornind de la T, actualizeaza estimarile folosind informatia din viitor prin J_t. Trecerea backward nu necesita acces la observatiile y_{1:T} - foloseste doar statisticile filtrate salvate. Aceasta permite 'aruncarea' vectorilor de observatie de dimensiune mare dupa pasul forward, reducand memoria necesara.

Tipare Cheie

Analogia cu HMM: algoritmul forward -> filtrul Kalman; algoritmul backward -> RTS backward pass; gamma_t (posterior marginal in HMM) -> N(mu_{t|T}, Sigma_{t|T}). Diferenta subtila: in HMM, backward pass calculeaza beta_t(i) = p(y_{t+1:T}|z_t=i), iar in RTS se calculeaza direct posteriorul smoothed. J_t este analogul castigului Kalman dar pentru directia backward.

Aplicatii Practice

RTS smoother-ul este folosit in: algoritmul EM pentru estimarea parametrilor SSM (pasul E necesita statistici smoothed), analiza de serii temporale offline, reconstructia de traiectorii, si interpolarea datelor lipsa in serii temporale. Este implementat in Kalman filter libraries (FilterPy, pykalman, statsmodels).

Exemplu din Lumea Reala

In oceanografie, RTS smoother-ul este folosit pentru a reconstitui traiectoriile curentilor oceanici din masuratorile rare ale plutitorelor Argo (care transmit pozitia prin satelit la intervale de zile). Netezirea produce traiectorii mult mai precise decat filtrarea, deoarece fiecare estimare beneficiaza de informatia din masuratorile anterioare si ulterioare, reducand gapurile temporale mari intre transmisii.

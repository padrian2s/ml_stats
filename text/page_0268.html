Echivalenta Bayesian-frequentista in regresia liniara si exemplul Caterpillar

Aceasta pagina demonstreaza ca, cu prior neinformativ, intervalele de credibilitate Bayesiene sunt identice cu intervalele de incredere frequentiste, si prezinta un exemplu numeric pe datele Caterpillar.

Marginala posterioara pentru w_j: p(w_j | D) = T(w_j | w_hat_j, C_jj s²/(N-D), N-D), unde C = (X^T X)^(-1) si s_j = sqrt(s² C_jj/(N-D)) este eroarea standard a coeficientului j.
Semnificatia statistica: Un coeficient este „semnificativ" daca intervalul sau de credibilitate de 95% nu include 0. Din Tabelul 7.2 (datele Caterpillar), coeficientii w0, w1, w2, w4, w5 sunt semnificativi.
Echivalenta exacta: (w_j - w_hat_j)/s_j ~ t_{N-D}, identic cu distributia de esantionare frequentista. Intervalele de credibilitate Bayesiene de 95% sunt identice cu intervalele de incredere frequentiste la nivelul de 5%. Aceasta echivalenta tine doar cu prior neinformativ.

Comentariu

Rezumat

Pagina stabileste un rezultat remarcabil: inferenta Bayesiana cu prior neinformativ reproduce exact inferenta frequentista in regresia liniara. Tabelul 7.2 arata rezultatele pentru 11 coeficienti pe datele Caterpillar: mediile posterioare, deviatiile standard, intervalele de credibilitate si semnificatia. Se noteaza ca aceasta echivalenta nu mai tine cand N < D (unde MLE nici nu exista), dar inferenta Bayesiana functioneaza cu priori proprii.

Tipare Cheie

1) Echivalenta Bayesian-frequentista este exacta in regresia liniara cu prior neinformativ. 2) Testul de semnificativa „CI nu include 0" este echivalent cu p-value < 0.05. 3) Cand N < D, abordarea frequentista esueaza complet, dar cea Bayesiana functioneaza cu un prior propriu. 4) Eroarea standard s_j depinde de pozitia variabilei j in spatiul corelat al predictorilor (prin C_jj).

Aplicatii Practice

Aceasta echivalenta inseamna ca rezultatele standard din regresie (tabelul ANOVA, p-values, intervale de incredere) au o interpretare Bayesiana valida. Dar avantajul Bayesian apare cand N < D (genomica, imagistica), cand se doresc priori informative, sau cand se doreste medierea peste modele.

Exemplu din Lumea Reala

Tabelul 7.2 arata ca w3, w6, w7, w8, w9, w10 nu sunt semnificativi (intervalele lor de credibilitate includ 0). Un practician ar putea decide sa le elimine din model, obtinand un model mai parcimonios. Aceasta este exact ceea ce fac procedurile standard de selectie a variabilelor in software-urile statistice (SPSS, Stata, R).

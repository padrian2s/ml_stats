Conditionarea Gaussiana pentru Interpolare si Imputarea Datelor

Pagina 114 aplica formulele de conditionare Gaussiana la interpolarea functiilor si introduce imputarea datelor lipsa.

Partitionarea matricei L (Ecuatia 4.79): L = [L₁, L₂], unde L₁ ∈ ℝ^((D-2)x(D-N)) corespunde punctelor necunoscute si L₂ ∈ ℝ^((D-2)xN) corespunde punctelor observate.
Distributia conditionala (Ecuatiile 4.81-4.83): p(x₁|x₂) = N(μ₁|₂, Σ₁|₂), unde μ₁|₂ = -Λ₁₁⁻¹ Λ₁₂ x₂ = -L₁ᵀ⁻¹ L₁ᵀ L₂ x₂ si Σ₁|₂ = Λ₁₁⁻¹. Media se calculeaza rezolvand L₁ μ₁|₂ = -L₂ x₂.
Intervale de credibilitate marginale (95%): μⱼ ± 2√(Σ₁|₂,ⱼⱼ). Varianta creste pe masura ce ne departam de punctele observate. λ nu afecteaza media posterioara (se anuleaza), dar afecteaza netezimea functiilor esantionate din posterior.
Imputarea datelor lipsa (Sectiunea 4.3.2.3): Cand lipsesc valori dintr-o matrice de design, se calculeaza p(x_hᵢ|x_vᵢ, θ) pentru fiecare rand i, unde h_i si v_i sunt indicii variabilelor ascunse si vizibile. Estimarea: x̂ᵢⱼ = E[xⱼ|x_vᵢ, θ].

Comentariu

Rezumat

Pagina demonstreaza doua aplicatii practice ale conditionarii Gaussiene. In interpolare, media posterioara trece exact prin punctele observate si interpoleaza neted intre ele, cu incertitudine crescanda departe de date. In imputarea datelor, fiecare valoare lipsa este estimata ca media conditionala, data fiind valorile observate din acelasi rand.

Tipare Cheie

Sistemul tridiagonal L₁ μ₁|₂ = -L₂ x₂ se rezolva eficient in O(D) operatii. Proprietatea ca λ nu afecteaza media posterioara (dar afecteaza varianta si esantioanele din posterior) este surprinzatoare si specifica acestui prior particular.

Aplicatii Practice

Imputarea datelor lipsa este esentiala in sondaje (raspunsuri lipsa), date medicale (teste nerealizate), senzori defecti si orice set de date real care contine valori lipsa. Metoda Gaussiana ofera atat estimari cat si masuri de incertitudine.

Exemplu din Lumea Reala

In studii clinice, pacientii adesea nu completeaza toate testele de laborator. Daca temperatura, tensiunea si glicemia sunt modelate ca MVN, putem estima glicemia lipsa din temperatura si tensiunea observate, impreuna cu un interval de incredere care cuantifica incertitudinea estimarii.

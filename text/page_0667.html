Invatarea online a parametrilor cu cele mai mici patrate recursive (RLS)

Aceasta pagina prezinta utilizarea SSM-urilor pentru invatarea online a parametrilor prin RLS (Recursive Least Squares) si introduce modelul liniar dinamic.

Recursive Least Squares (RLS): Starea ascunsa z_t = theta (parametrii regresiei), modelul de tranzitie: p(theta_t|theta_{t-1}) = delta(theta_t - theta_{t-1}) (parametri constanti), modelul de observatie: p(y_t|x_t, theta) = N(y_t|x_t^T theta, sigma^2). Aplicarea filtrului Kalman produce actualizarea: theta_hat_t = theta_hat_{t-1} + (1/sigma^2) Sigma_{t|t} (y_t - x_t^T theta_hat_{t-1}) x_t.
RLS vs LMS: Daca se aproximeaza (1/sigma^2) Sigma_{t|t-1} cu eta_t I, se obtine algoritmul Least Mean Squares (LMS). LMS necesita ajustarea manuala a ratei de invatare eta_t si poate necesita mai multe treceri prin date. RLS adapteaza automat pasul si converge la posteriorul optimal intr-o singura trecere (Figura 18.4b).

Comentariu

Rezumat

Ideea cheie: regresia liniara poate fi formulata ca un SSM unde starea ascunsa sunt parametrii theta, tranzitia este identitate (parametri constanti) cu Q_t = 0, si modelul de observatie este N(y_t|x_t^T theta, sigma^2) cu C_t = x_t^T variind in timp. Aplicarea filtrului Kalman produce algoritmul RLS care actualizeaza posterior beliefs despre theta pe masura ce datele vin. Daca parametrii se schimba in timp (Q_t > 0), se obtine modelul liniar dinamic (dynamic linear model). Figura 18.4 arata convergenta RLS la solutia batch (offline).

Tipare Cheie

Dualitatea offline-online: regresia liniara batch (OLS) proceseaza toate datele simultan; RLS proceseaza datele secvential, convergand la aceeasi solutie. Avantajele RLS: O(D^2) per pas de timp (vs O(TD^2) pentru batch), necesita O(D^2) memorie (vs O(TD) pentru batch), si se adapteaza la date noi fara recalculare completa.

Aplicatii Practice

RLS este folosit in: filtrarea adaptiva (anularea ecoului in telecomunicatii), egalizarea canalelor de comunicatie, controlul adaptiv (robotica, procesare industriala), si antrenarea online a modelelor de machine learning cand datele vin in flux (streaming).

Exemplu din Lumea Reala

In apelurile telefonice si videoconferintele (Zoom, Teams), RLS este folosit pentru anularea ecoului acustic: filtrul adaptiv invata in timp real transferul acustic dintre difuzor si microfon, permitand anularea ecoului fara a cunoaste in avans acustica camerei. Parametrii filtrului se actualizeaza continuu pe masura ce acustica se schimba (de ex. cand cineva se misca).

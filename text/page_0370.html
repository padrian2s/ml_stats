Amestecuri de Gaussiene: proprietati, amestec de Bernoulli si clustering

Aceasta pagina prezinta amestecul de Bernoulli (multinomiale) pentru date binare, demonstreaza ca amestecurile pot capta corelatii, si introduce utilizarea modelelor de amestec pentru clustering.

Amestec de Bernoulli: p(x_i | z_i = k) = ∏_{j=1}^D Ber(x_{ij} | μ_{jk}), un produs de Bernoulli independente per cluster. Desi componentele sunt factorizate, distributia de amestec poate capta corelatii intre variabile.
Responsabilitati (soft clustering): r_{ik} = p(z_i = k | x_i, θ) = p(z_i=k|θ) p(x_i|z_i=k, θ) / Σ_{k'} p(z_i=k'|θ) p(x_i|z_i=k', θ). Probabilitatea posterioara ca observatia i sa apartina clusterului k.
Soft vs. hard clustering: Soft clustering: fiecare punct are probabilitati de apartinenta la fiecare cluster (r_{ik}). Hard clustering: z_i* = argmax_k r_{ik}, fiecare punct este atribuit unui singur cluster. Soft clustering este mai informativ dar hard clustering este mai usor de interpretat.

Comentariu

Rezumat

Pagina arata ca amestecurile pot modela corelatii complexe chiar daca componentele individuale presupun independenta: cov[x] = Σ_k π_k[Σ_k + μ_k μ_k^T] - E[x]E[x]^T. Amestecul de Bernoulli este util pentru date binare (documente, imagini binarizate). Clusteringul cu modele de amestec se reduce la calculul responsabilitatilor posterioare.

Tipare Cheie

1) Independenta intra-cluster + amestec = corelatie inter-cluster. 2) Soft clustering este identic cu clasificarea generativa (aceeasi formula Bayes). 3) Diferenta: in clasificare stim y_i (supervizat), in clustering nu stim z_i (nesupervizat). 4) Problema label switching: clusterele pot fi renumite arbitrar fara a afecta modelul.

Aplicatii Practice

Amestecul de Bernoulli este folosit in modelarea textului (documente ca vectori binari bag-of-words), in genomica (haplotipuri), si in segmentarea de imagini binarizate. Soft clustering este preferat in marketingul personalizat (un client poate apartine partial mai multor segmente).

Exemplu din Lumea Reala

In segmentarea de clienti pentru o banca, un client poate fi 60% „tanar profesionist" si 40% „investitor conservator" (soft clustering). Aceasta informatie este mai utila pentru personalizarea ofertelor decat o atribuire dura la un singur segment, deoarece permite recomandari nuantate care reflecta multiplele aspecte ale profilului clientului.

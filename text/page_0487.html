Elastic Net: Versiunea Vanilla si Algoritmi (p.456)

Aceasta pagina detaliaza elastic net: functia obiectiv, efectul de grupare, algoritmul LARS-EN si versiunea imbunatatita.

Obiectivul Elastic Net: J(w, lambda_1, lambda_2) = ||y - Xw||^2 + lambda_2 * ||w||_2^2 + lambda_1 * ||w||_1. Functia este strict convexa (daca lambda_2 > 0), garantand un minim global unic.
Efect de Grupare: Orice penalizare strict convexa pe w produce un efect de grupare: coeficientii variabilelor foarte corelate tind sa fie egali. Daca X_{.,j} = X_{.,k}, atunci w_hat_j = w_hat_k, spre deosebire de lasso unde unul poate fi zero.
Algoritmul LARS-EN: Elastic net poate fi redus la o problema lasso pe date modificate: X_tilde = c * [X; sqrt(lambda_2) * I_D], y_tilde = [y; 0]. Se rezolva cu LARS si apoi w = c * w_tilde. Costul este O(m^3 + Dm^2).
Versiunea Imbunatatita: Elastic net vanilla produce dubla micsorare (shrinkage). Solutia: w_hat = sqrt(1 + lambda_2) * w_tilde, care anuleaza micsorarea l2 si pastreaza doar efectul de selectie.

Comentariu

Rezumat

Elastic net combina avantajele ambelor regularizari: selectia de variabile de la l1 si stabilitatea de la l2. Efectul de grupare asigura ca variabilele corelate sunt tratate similar. Versiunea imbunatatita corecteaza problema dublei micsorari.

Tipare Cheie

Reducerea elastic net la o problema lasso pe date augmentate este o idee eleganta care permite reutilizarea algoritmilor existenti (LARS). Matricea X_tilde are rang D chiar daca X nu, permitand selectia a mai mult de N variabile.

Aplicatii Practice

Elastic net este metoda standard in pachetele software (glmnet in R, scikit-learn in Python) si este una dintre cele mai utilizate metode de regularizare in practica, datorita robustetei sale la multi-coliniaritate.

Exemplu din Lumea Reala

In predictia preturilor imobiliare, trasaturile (suprafata, numar camere, locatie) sunt adesea corelate. Elastic net selecteaza toate trasaturile relevante si le trateaza echitabil, in loc sa aleaga arbitrar intre ele ca lasso.

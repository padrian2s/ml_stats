Capitolul 1: Completarea Matricelor, Inpainting si Filtrare Colaborativa

Probleme de imputare a datelor lipsa: completarea matricelor, restaurarea imaginilor si sisteme de recomandare.

Completarea Matricelor (Matrix Completion): Inferarea valorilor plauzibile pentru intrarile lipsa dintr-o matrice de date. Valorile lipsa sunt reprezentate ca NaN ("not a number").
Imputare (Imputation): Procesul de completare a valorilor lipsa din date, inferand valori plauzibile din datele disponibile.
Image Inpainting: Completarea "gaurilor" dintr-o imagine (cauzate de zgarieturi sau ocluziuni) cu textura realista, construind un model probabilistic conjunct al pixelilor.
Filtrare Colaborativa (Collaborative Filtering): Prezicerea preferintelor utilizatorilor pe baza unei matrice de evaluari X(m, u), unde m este filmul si u este utilizatorul. Cheia: predictia se bazeaza pe matricea de evaluari, nu pe caracteristici.
Figura 1.12: (a) O imagine zgomotoasa cu un element de ocluziune. (b) Estimarea pixelilor originali folosind un model MRF (Markov Random Field) pe perechi.
Matricea de Evaluari: X(m, u) este evaluarea data de utilizatorul u filmului m. Majoritatea intrarilor sunt lipsa (NaN), iar scopul este de a prezice evaluarile lipsa.

Comentariu

Rezumat

Sectiunea 1.3.4 introduce completarea matricelor ca problema generala de imputare. Image inpainting foloseste modele probabilistice pentru a completa portiuni lipsa din imagini. Filtrarea colaborativa prezice preferintele utilizatorilor pe baza evaluarilor altor utilizatori, fara a necesita informatii despre continutul filmelor.

Tipare Cheie

Toate aceste probleme implica inferenta valorilor lipsa din date observate. Abordarea probabilistica permite modelarea incertitudinii asociate cu valorile imputate. Filtrarea colaborativa este un caz special elegant unde nu se folosesc caracteristici ale filmelor sau utilizatorilor, ci doar patternuri in matricea de evaluari.

Aplicatii Practice

Image inpainting este utilizat in restaurarea fotografiilor vechi, stergerea obiectelor nedorite din imagini si reconstructia imaginilor medicale. Filtrarea colaborativa este baza sistemelor de recomandare folosite de Netflix, Amazon si Spotify.

Exemplu din Lumea Reala

Netflix foloseste filtrare colaborativa pentru a recomanda filme. Din milioane de utilizatori si mii de filme, matricea de evaluari are peste 99% valori lipsa. Sistemul identifica patternuri (utilizatori cu gusturi similare) si prezice evaluarile lipsa, generand recomandari personalizate.

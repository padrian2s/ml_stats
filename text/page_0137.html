LDA cu Doua Clase - d-prime si MLE pentru Analiza Discriminanta

Pagina 106 continua cu regula finala de decizie LDA cu doua clase, introduce masura d-prime din teoria detectiei semnalelor si prezinta estimarea MLE pentru analiza discriminanta.

Regula de decizie (Ecuatia 4.50): p(y=1|x, θ) = sigm(wᵀ(x - x₀)), unde se deplaseaza x cu x₀, se proiecteaza pe directia w si se aplica functia sigmoid.
d-prime (Ecuatia 4.51): d' = (μ₁ - μ₀) / σ, o masura din psihologie si teoria detectiei semnalelor care cuantifica discriminabilitatea intre semnal si zgomot. d' mare inseamna separare usoara.
Log-verosimilitatea pentru GDA (Ecuatia 4.52): log p(D|θ) = [Σᵢ Σ_c I(yᵢ=c) log π_c] + Σ_c [Σᵢ:yᵢ=c log N(x|μ_c, Σ_c)]. Se factorizeaza intr-un termen pentru π si cate un termen pentru fiecare (μ_c, Σ_c).
MLE pentru GDA (Ecuatia 4.53): π̂_c = N_c/N, μ̂_c = (1/N_c) Σᵢ:yᵢ=c xᵢ, Σ̂_c = (1/N_c) Σᵢ:yᵢ=c (xᵢ - μ̂_c)(xᵢ - μ̂_c)ᵀ

Comentariu

Rezumat

Regula de decizie LDA se reduce la: deplaseaza, proiecteaza, aplica sigmoid. Masura d-prime ofera o metrica intuitiva de dificultate a clasificarii. MLE-ul pentru GDA este simplu: se partitioneaza datele pe clase si se calculeaza media si covarianta empirica pentru fiecare clasa separat.

Tipare Cheie

Factorizarea log-verosimilitatii permite estimarea independenta a parametrilor fiecarei clase. Acest lucru face ca antrenarea GDA sa fie foarte eficienta computational - nu necesita optimizare iterativa, ci doar calcule statistice directe.

Aplicatii Practice

Masura d-prime este utilizata pe scara larga in psihofizica (cat de bine poate un subiect distinge doua stimuluri), in evaluarea sistemelor biometrice (separarea intre scoruri de potrivire si nepotrivire) si in radiologie (detectabilitatea leziunilor).

Exemplu din Lumea Reala

In evaluarea unui sistem de recunoastere a amprentelor digitale, d-prime masoara separarea intre distributia scorurilor pentru amprente care se potrivesc vs. cele care nu se potrivesc. Un d' = 4 indica performanta excelenta, in timp ce d' = 1 indica suprapunere semnificativa intre distributii.

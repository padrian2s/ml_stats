Functii de disimilaritate si evaluarea clustering-ului

Aceasta pagina detaliaza functiile de disimilaritate comune (distanta euclidiana patrata, distanta L1/city block, distanta Hamming) si introduce evaluarea rezultatelor clustering-ului.

Distanta euclidiana patrata: Delta_j(x_{ij}, x_{i'j}) = (x_{ij} - x_{i'j})^2, aplicabila doar atributelor cu valori reale.
Distanta city block (L1): Delta_j(x_{ij}, x_{i'j}) = |x_{ij} - x_{i'j}|, mai robusta la diferente mari decat distanta patrata.
Distanta Hamming: Delta(x_i, x_i') = suma I(x_{ij} != x_{i'j}), folosita pentru variabile categorice, numara cate trasaturi difera.
Echivalenta clustering-corelatie: Pentru date standardizate, clustering-ul bazat pe corelatie este echivalent cu clustering-ul bazat pe distanta euclidiana patrata, deoarece suma(x_{ij} - x_{i'j})^2 = 2(1 - corr[x_i, x_i']).

Comentariu

Rezumat

Pagina prezinta functiile de disimilaritate specifice diferitelor tipuri de date: distanta euclidiana pentru date continue, distanta L1 pentru robustete, coeficientul de corelatie pentru serii temporale, codificarea ordinala pentru variabile ordinale, si distanta Hamming pentru variabile categorice. Se evidentiaza echivalenta importanta intre corelatie si distanta euclidiana pentru date standardizate.

Tipare Cheie

Alegerea functiei de disimilaritate este critica si depinde de tipul datelor. Distanta euclidiana patrata amplifica diferentele mari, in timp ce L1 este mai robusta. Pentru date mixte (continue si categorice), se pot combina diferite masuri pe atribute.

Aplicatii Practice

In bioinformatica, coeficientul de corelatie este preferat pentru clustering-ul profilurilor de expresie genica (serii temporale). In text mining, distanta cosinus (legata de corelatie) este standard. In analiza datelor mixte din anchete, combinatia de masuri este esentiala.

Exemplu din Lumea Reala

In sistemele de recomandare, alegerea metricii de similaritate intre utilizatori (corelatie Pearson vs. distanta cosinus vs. Jaccard) afecteaza semnificativ calitatea recomandarilor, fiecare metrica fiind potrivita pentru tipuri diferite de date de rating.

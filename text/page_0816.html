Propagarea increderii convexa (Convex BP)

Aceasta pagina introduce propagarea increderii convexa, care combina o margine superioara convexa pe entropie cu o aproximare exterioara convexa a politopului marginal.

Propagarea increderii convexa: Se lucreaza cu o multime F de submodele tractabile (arbori, grafuri planare). Pentru fiecare submodel F, entropia pe F este >= entropia pe G, deoarece F are mai putine constrangeri. O combinatie convexa a acestor entropii ofera o margine superioara convexa pe entropia reala.
Energia libera convexa: F_Convex(mu, rho) = -[mu^T*theta + H(mu, rho)], unde H(mu, rho) = sum_F rho(F)*H(mu(F)) cu rho(F) >= 0, sum rho(F) = 1. Obiectivul este concav si multimea L(G;F) este convexa, deci exista un maxim global unic.
Avantaj principal: Spre deosebire de energia Bethe (neconcava) si mean field (multime neconvexa), problema convexa are un optim global unic, eliminand dependenta de initializare.

Comentariu

Rezumat

Pagina prezinta o abordare eleganta pentru a face inferenta variationala convexa: se combina o margine superioara pe entropie (obtenuta prin medierea peste submodele tractabile) cu multimea convexa L(G;F). Rezultatul este o problema de optimizare convexa cu solutie unica.

Tipare Cheie

Convexitatea este obtinuta prin doua ingrediente: (1) margine superioara convexa pe entropie (medierea peste arbori), (2) multimea L(G;F) definita prin intersectia multimilor M(F) (convexa, deoarece intersectia multimilor convexe este convexa).

Aplicatii Practice

Convex BP garanteaza un optim global unic, ceea ce il face atractiv cand reproducibilitatea rezultatelor este critica. Dezavantajul este costul computational mai mare.

Exemplu din Lumea Reala

In sisteme de diagnosticare automata a defectelor in productie, unde consistenta rezultatelor este critica, convex BP ofera garantia ca analiza va produce mereu aceleasi concluzii, indiferent de ordinea procesarii datelor.

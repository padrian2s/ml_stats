Exercitii: optiuni multiple de respingere, problema newsvendor, factori Bayes si curbe ROC

Aceasta pagina contine exercitii avansate despre decizii cu optiuni multiple de respingere, problema clasica a newsvendorului din teoria economica a deciziei, si relatia intre factori Bayes si curbe ROC.

Problema newsvendor: Se doreste determinarea cantitatii optime Q de produs de cumparat cand cererea D este incerta. Profitul asteptat este Eπ(Q) = ∫(P-C)Q f(D)dD + ... Cantitatea optima satisface F(Q*) = (P-C)/P, unde F este CDF-ul cererii.
Medierea Bayesiana a modelelor (BMA): p^BMA(Δ) = Σ_m p(Δ|m, D) p(m|D). Predictia BMA este o medie ponderata peste toate modelele, cu ponderi date de probabilitatile posterioare ale modelelor.
Optimalitatea BMA: BMA produce pierdere asteptata mai mica sau egala cu orice model individual: E[L(Δ, p^BMA)] ≤ E[L(Δ, p^m)] pentru orice m. Demonstratia foloseste non-negativitatea divergentei KL.

Comentariu

Rezumat

Pagina prezinta exercitii care conecteaza teoria deciziei Bayesiene cu probleme practice din economie si invatare automata. Problema newsvendor demonstreaza ca decizia optima sub incertitudine depinde de raportul cost-pret (fractila critica). BMA este demonstrata a fi optima comparativ cu orice model individual, motivand medierea peste modele in loc de selectia unui singur model.

Tipare Cheie

1) Formula fractilei critice F(Q*) = (P-C)/P: daca marja de profit este mare (P >> C), comanda mai mult; daca costul este aproape de pret, comanda mai putin. 2) BMA domina orice model individual sub pierderea logaritmica. 3) Factorii Bayes si curbele ROC ofera informatii complementare dar diferite despre performanta modelelor.

Aplicatii Practice

Problema newsvendor se aplica in managementul inventarelor (retail, farmacii), in rezervarea de capacitate (hoteluri, aviatie - overbooking), si in planificarea productiei. BMA este utilizata in prognoza meteo (combinarea modelelor climatice), in epidemiologie, si in previziunile macroeconomice.

Exemplu din Lumea Reala

Un magazin de flori trebuie sa comande trandafiri pentru Ziua Indragostitilor. Costul per trandafir este C = 2 lei, pretul de vanzare P = 8 lei. Formula newsvendor da F(Q*) = 6/8 = 0.75: trebuie comandata cantitatea corespunzatoare percentilei 75 a distributiei cererii. Daca cererea este N(100, 20), Q* = 100 + 0.675 x 20 = 113.5 trandafiri. Intuitiv: marja mare justifica un risc mai mare de supra-stoc.

MLE pentru familia exponentiala si teorema Pitman-Koopman-Darmois

Aceasta pagina deriveaza MLE-ul pentru familia exponentiala si introduce teorema fundamentala despre statisticile suficiente.

Teorema Pitman-Koopman-Darmois: Sub conditii de regularitate, familia exponentiala este singura familie de distributii cu statistici suficiente de dimensiune finita (independente de marimea esantionului N).
MLE pentru familia exponentiala (moment matching): La MLE, media empirica a statisticilor suficiente trebuie sa egaleze media teoretica: E[φ(X)] = (1/N) Σ φ(x_i). Aceasta conditie se numeste potrivirea momentelor (moment matching).
Concavitatea log-verosimilitatii: log p(D|θ) = θ^T φ(D) - NA(θ). Deoarece -A(θ) este concava si θ^T φ(D) este liniara, log-verosimilitatea este concava in θ, garantand un unic maxim global.

Comentariu

Rezumat

Pagina stabileste doua rezultate majore. Primul: MLE pentru familia exponentiala se reduce la potrivirea momentelor - media teoretica a statisticilor suficiente trebuie sa egaleze media empirica. Al doilea: teorema Pitman-Koopman-Darmois arata ca familia exponentiala este unica in a permite compresia datelor in statistici de dimensiune finita. Exemplul distributiei uniforme arata de ce aceasta nu e in familia exponentiala: suportul depinde de parametri.

Tipare Cheie

1) MLE = moment matching: o conditie simpla si intuitiva. 2) Statisticile suficiente φ(D) = Σ φ(x_i) cresc liniar cu N, dar dimensiunea lor este fixa. 3) Concavitatea garanteaza unicitatea MLE si convergenta algoritmilor de optimizare. 4) Distributia uniforma are statistica suficienta max(x_i), dar aceasta depinde de parametru prin suportul distributiei.

Aplicatii Practice

Moment matching este folosit pe scara larga in estimarea parametrilor: Gaussiana (media si varianta), Poisson (media), Gamma (media si media logaritmului). In procesarea de limbaj natural, moment matching apare in estimarea modelelor MaxEnt (entropie maxima), unde constrangerile sunt date de statisticile empirice ale trasaturilor.

Exemplu din Lumea Reala

In modelarea duratei apelurilor intr-un call center (distributie Gamma), MLE se obtine prin egalizarea mediei si variantei teoretice cu cele empirice. Cu N = 10.000 de apeluri, intreaga informatie este capturata de doar 3 numere: N, Σ x_i si Σ log(x_i). Aceasta permite analiza rapida chiar si pe seturi de date masive.

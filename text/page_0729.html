Metode Cutting Plane pentru Antrenarea SSVM-urilor

Pagina prezinta algoritmul cutting plane pentru antrenarea eficienta a SSVM-urilor, ilustrat cu Figura 19.21 si inceputul sectiunii despre decodarea cu pierdere augmentata.

Figura 19.21: Ilustrarea algoritmului cutting plane in 2D. (a) Se incepe cu w = w₀ = 0, se adauga prima constrangere, regiunea umbrita este noul set fezabil, solutia cu norma minima este w₁. (b) Se adauga o alta constrangere, regiunea intunecata este noul set fezabil. (c) Se adauga o a treia constrangere, rafinand si mai mult setul fezabil.
Metoda cutting plane: Un algoritm iterativ din optimizarea convexa (Kelley, 1960) care incepe cu w si fara constrangeri, apoi adauga iterativ constrangerile cele mai violate pana la convergenta.
Pachetul SVMstruct: Implementarea populara a algoritmului cutting plane pentru SSVM-uri, care poate gestiona functii de pierdere generale.
Decodare cu pierdere augmentata: argmax_{y∈Y} L(yᵢ,y) - wᵀδᵢ(y) = argmax_{y∈Y} L(yᵢ,y) + wᵀφ(xᵢ,y) (Ecuatia 19.107) - gasirea constrangerii cel mai violate, un pas cheie in eficienta algoritmului.

Comentariu

Rezumat

Algoritmul cutting plane rezolva problema numarului exponential de constrangeri prin adaugarea iterativa doar a constrangerilor cel mai violate. La fiecare iteratie, se gaseste exemplul cu cea mai mare violare a marginii augmentate cu pierdere, se adauga constrangerea corespunzatoare, si se rezolva QP-ul rezultat. Doar un numar polinomial de constrangeri trebuie adaugate.

Tipare Cheie

Metoda cutting plane exploateaza faptul ca din numarul exponential de constrangeri, doar un numar mic sunt active la optim. Figura 19.21 ilustreaza vizual cum setul fezabil este rafinat iterativ prin adaugarea de hiperplane care taie regiuni nefezabile. Eficienta depinde de capacitatea de a rezolva subproblema de decodare augmentata.

Aplicatii Practice

SVMstruct este unul dintre cele mai utilizate pachete software pentru antrenarea SSVM-urilor si este disponibil public. Este folosit in cercetarea academica si in industrie pentru probleme de clasificare structurata, de la NLP la viziune computationala.

Exemplu din Lumea Reala

In detectia obiectelor, SSVM-ul trebuie sa prezica atat categoria obiectului cat si cadrul de delimitare (bounding box). Algoritmul cutting plane permite antrenarea eficienta chiar daca spatiul iesirilor (toate combinatiile posibile de categorii si pozitii) este imens.

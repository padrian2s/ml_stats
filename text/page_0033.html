Capitolul 1: Tipuri de Invatare Automata

Prezentarea principalelor tipuri de invatare automata: supervizata, nesupervizata si prin recompensa.

Invatare Supervizata (Supervised Learning): Scopul este de a invata o mapare de la intrari x la iesiri y, folosind un set de antrenament de perechi etichetate D = {(x_i, y_i)}ᴺᵢ₌₁.
Clasificare (Classification): Problema in care variabila de iesire y_i este categorica, apartinand unui set finit y_i ∈ {1, ..., C}. Se mai numeste si recunoastere de tipare (pattern recognition).
Regresie (Regression): Problema in care variabila de iesire y_i este un scalar cu valori reale (de exemplu, nivelul venitului).
Invatare Nesupervizata (Unsupervised Learning): Scopul este de a gasi "tipare interesante" in date, avand doar intrari D = {x_i}ᴺᵢ₌₁, fara etichete. Se mai numeste si descoperire de cunostinte (knowledge discovery).
Invatare prin Recompensa (Reinforcement Learning): Invatarea modului de a actiona pe baza semnalelor ocazionale de recompensa sau pedeapsa.
Caracteristici (Features): Fiecare intrare x_i este un vector D-dimensional de numere, reprezentand atribute precum inaltimea si greutatea. Se mai numesc atribute sau covariate.

Comentariu

Rezumat

Sectiunea 1.1.1 imparte invatarea automata in trei categorii principale. Invatarea supervizata invata din exemple etichetate si include clasificarea (iesiri categorice) si regresia (iesiri continue). Invatarea nesupervizata cauta tipare fara etichete. Invatarea prin recompensa invata prin interactiune cu mediul. Cartea se concentreaza pe primele doua tipuri.

Tipare Cheie

Distinctia intre clasificare si regresie este fundamentala si determina alegerea modelului si a metricii de evaluare. Regresia ordinala (de exemplu, note A-F) este un caz intermediar interesant. Clasificarea multi-eticheta difera de clasificarea multi-clasa: in prima, un obiect poate avea mai multe etichete simultan.

Aplicatii Practice

Clasificarea: detectarea spam-ului, diagnosticul medical, recunoasterea obiectelor in imagini. Regresia: prezicerea preturilor, estimarea temperaturii. Invatare nesupervizata: segmentarea clientilor, detectarea anomaliilor. Invatare prin recompensa: roboti care invata sa mearga, jocuri video.

Exemplu din Lumea Reala

Un filtru de spam pentru email este un clasificator binar (C = 2): primeste ca intrare caracteristici ale email-ului (cuvinte, expeditor, formatare) si produce ca iesire o eticheta y ∈ {0, 1} indicand daca mesajul este spam sau nu.

Analiza Componentelor Principale (PCA) - Aproximare de Rang Scazut si PCA Probabilistic

Pagina continua discutia despre PCA ca aproximare de rang scazut prin SVD trunchiat si introduce PCA probabilistic (PPCA).

Aproximare SVD trunchiat: Daca X = USV^T este descompunerea SVD, atunci reconstructia optima de rang L este X̂ = USV^T, identica cu proiectia PCA.
PCA Probabilistic (PPCA): Un model de analiza factoriala in care Ψ = σ²I, adica zgomotul este izotrop, permitand o interpretare probabilistica a PCA clasic.
Log-verosimilitatea datelor observate: log p(X|W, σ²) = -N/2 ln|C| - 1/2 Σ xᵢᵀ C⁻¹ xᵢ, unde C = WW^T + σ²I
Estimatorul MLE al zgomotului: σ̂² = 1/(D-L) Σⱼ₌L₊₁ᴰ λⱼ, media valorilor proprii asociate dimensiunilor eliminate.
Posteriorul factorilor latenti: p(zᵢ|xᵢ, θ̂) = N(zᵢ | F̂⁻¹Ŵ^T xᵢ, σ²F̂⁻¹), unde F̂ = Ŵ^T Ŵ + σ̂²I

Comentariu

Rezumat

Aceasta pagina stabileste legatura fundamentala intre PCA clasic si descompunerea SVD trunchiat, demonstrand ca PCA ofera cea mai buna aproximare de rang scazut. Apoi introduce PCA probabilistic (PPCA) ca un caz special al analizei factoriale cu zgomot izotrop, derivand log-verosimilitatea, estimatorii MLE pentru matricea de greutati W si varianta zgomotului σ², precum si distributia posterioara a factorilor latenti.

Tipare Cheie

Cand σ² → 0, PPCA converge la PCA clasic: Ŵ → V (eigenvectorii), iar posteriorul devine o proiectie ortogonala. Varianta zgomotului estimata σ̂² este media valorilor proprii eliminate, ceea ce are o interpretare intuitiva: zgomotul captureaza varianta medie in directiile ignorate de model.

Aplicatii Practice

PPCA este utilizat in reducerea dimensionalitatii cu un cadru probabilistic solid, permitand selectia modelului prin verosimilitate marginala, gestionarea datelor lipsa si combinarea cu alte modele generative. Este preferat PCA clasic cand se doreste cuantificarea incertitudinii.

Exemplu din Lumea Reala

In procesarea imaginilor medicale, PPCA poate fi utilizat pentru a reduce dimensionalitatea scanurilor RMN, modeland explicit zgomotul senzorului ca σ²I si recuperand componentele principale care captureaza variabilitatea anatomica reala.

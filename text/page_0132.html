Derivarea de Entropie Maxima si Analiza Discriminanta Gaussiana

Pagina 101 demonstreaza ca distributia Gaussiana multivariata are entropie maxima printre toate distributiile cu aceeasi covarianta si introduce analiza discriminanta Gaussiana (GDA).

Entropia diferentiala a MVN (Ecuatia 4.25): h(N(μ, Σ)) = (1/2) ln[(2πe)^D |Σ|]. Aceasta este entropia maxima posibila pentru orice distributie cu covarianta Σ specificata.
Teorema 4.1.2 (Entropie maxima): Fie q(x) orice densitate cu ∫ q(x) xᵢxⱼ = Σᵢⱼ. Fie p = N(0, Σ). Atunci h(q) ≤ h(p). Demonstratia foloseste KL(q||p) ≥ 0.
Analiza Discriminanta Gaussiana (GDA): O metoda de clasificare generativa unde densitatile conditionale pe clasa sunt Gaussiene: p(x|y=c, θ) = N(x|μ_c, Σ_c). Daca Σ_c este diagonala, GDA se reduce la Naive Bayes.

Comentariu

Rezumat

Sectiunea 4.1.4 ofera o justificare profunda pentru utilizarea distributiei Gaussiene: este distributia de entropie maxima sub constrangerile de medie si covarianta fixate. Aceasta proprietate face Gaussiana cea mai "conservatoare" alegere cand stim doar primele doua momente. Sectiunea 4.2 introduce GDA ca aplicatie directa a MVN in clasificare.

Tipare Cheie

Demonstratia prin divergenta KL este eleganta: KL(q||p) ≥ 0 implica -h(q) - ∫q log p ≥ 0, iar pasul cheie este ca q si p au aceleasi momente de ordinul doi, deci ∫q log p = ∫p log p. Aceasta legatura intre entropie maxima si KL este fundamentala in teoria informatiei.

Aplicatii Practice

GDA este utilizata in recunoasterea vocii (fiecare vorbitor are propria distributie Gaussiana), clasificarea textelor si diagnosticul medical. Principiul entropiei maxime justifica utilizarea Gaussienei in fizica statistica si inferenta Bayesiana.

Exemplu din Lumea Reala

In telecomunicatii, zgomotul termic este modelat ca Gaussian tocmai pentru ca, dat fiind ca stim doar puterea zgomotului (varianta), distributia Gaussiana este cea mai pesimista presupunere (entropie maxima), garantand performanta in cel mai rau caz.

Capitolul 3 - Modele Generative pentru Date Discrete: Introducere

Pagina deschide capitolul 3, care tratează modele generative pentru date discrete, și introduce învățarea conceptelor Bayesiene ca exemplu motivator.

Clasificator generativ: p(y=c|x,θ) ∝ p(x|y=c,θ)·p(y=c|θ), unde p(x|y=c,θ) este densitatea condiționată de clasă care definește ce date așteptăm să vedem în fiecare clasă.
Învățarea conceptelor: Echivalentă cu clasificarea binară: f(x) = 1 dacă x ∈ C (concept), f(x) = 0 altfel. Scopul: învățarea funcției indicator f din exemple pozitive.
Jocul numerelor (Tenenbaum): Se alege un concept C (ex: „număr prim", „putere a lui 2") și se dau exemple pozitive D = {x₁,...,x_N}. Sarcina: clasificarea unui nou număr x̃ ca aparținând sau nu conceptului C.
Teoria mulțimilor vagi: Permițând incertitudine în definiția conceptului C, putem emula teoria mulțimilor vagi (fuzzy set theory) folosind calcul probabilistic standard.

Comentariu

Rezumat

Capitolul 3 abordează modelarea generativă pentru date discrete. Jocul numerelor al lui Tenenbaum demonstrează cum oamenii învață concepte din exemple pozitive, un comportament pe care modelele Bayesiene îl pot reproduce.

Tipare Cheie

Modelele generative modelează distribuția datelor p(x|y), în contrast cu modelele discriminative care modelează direct p(y|x). Avantajul: pot genera date noi și gestionează mai bine datele lipsă.

Aplicatii Practice

Învățarea conceptelor din puține exemple (few-shot learning) este esențială în robotică, unde robotul trebuie să învețe noi obiecte din doar câteva demonstrații.

Exemplu din Lumea Reala

Un copil care învață cuvântul „câine" din câteva exemple pozitive (fără exemple negative explicite) realizează exact acest tip de învățare a conceptelor descris de Tenenbaum.

Constructia stick-breaking si posteriorul DP

Aceasta pagina prezinta constructia stick-breaking (ruperea bastonului) pentru procesul Dirichlet si posteriorul DP dupa observarea datelor.

Constructia stick-breaking: Ponderile pi_k sunt generate prin: beta_k ~ Beta(1, alpha), pi_k = beta_k * produs(1-beta_l) pentru l=1..k-1. Aceasta se noteaza pi ~ GEM(alpha) si genereaza o secventa infinita descrescatoare in medie.
Posteriorul DP: Dupa observarea esantioanelor theta_1,...,theta_N ~ G, posteriorul este G|theta_1,...,theta_N ~ DP(alpha+N, 1/(alpha+N) * (alpha*H + suma(delta_{theta_i}))). DP-ul este prior conjugat pentru el insusi.
Proprietatea de conjugare: Alpha joaca rolul unei "dimensiuni efective a esantionului" pentru masura de baza H, iar posteriorul este un amestec ponderat intre prior (H) si datele observate (suma de delta-uri).

Comentariu

Rezumat

Constructia stick-breaking ofera o procedura constructiva pentru generarea ponderilor: se rupe un baston de lungime 1 la un punct aleator (beta_1), se pastreaza bucata (pi_1), apoi se repeta pe restul bastonului. Figura 25.5 arata atat procesul vizual cat si esantioane pentru alpha=2 si alpha=5. Cu alpha mic, primele componente domina; cu alpha mare, ponderile sunt mai uniform distribuite.

Tipare Cheie

Constructia stick-breaking este esentiala pentru implementarea practica a DP-ului, deoarece ofera o reprezentare parametrica explicita (desi infinita) a masurii aleatoare G. Posteriorul DP pastreaza forma DP, demonstrand conjugarea perfecta.

Aplicatii Practice

Constructia stick-breaking este baza algoritmilor de inferenta variationala pentru modele DP (de exemplu, variational inference truncata), unde se trunchiaza seria la K componente si se optimizeaza parametrii variationale pentru beta_k.

Exemplu din Lumea Reala

In sistemele de dialog conversational, constructia stick-breaking este folosita in modele DP pentru a descoperi automat numarul de intentii ale utilizatorilor, cu alpha controlat astfel incat intentiile rare dar importante (precum urgentele) sa fie detectate.

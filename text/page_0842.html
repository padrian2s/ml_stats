Recuperarea Asignarii MAP si Marginale Max (Sectiunea 22.6.5.5)

Aceasta pagina discuta calculul marginalelor max, convergenta coborarii pe coordonate si metode de recuperare a asignarii MAP din solutia duala optima.

Mesajul de iesire de la factor: delta_{f->2}(x_2) = -delta_{2->f}(x_2) + (1/2) max_{x_1} [theta_f(x_1, x_2) + delta_{1->f}(x_1) + delta_{2->f}(x_2)] (Ec. 22.181)
Mesajul de intrare: delta_{2->f}(x_2) = theta_2(x_2) + delta_{h,2}(x_2) (Ec. 22.182)
Marginale max: max_{x_{f\i}} h(x_{f\i}, x_i) unde h(x_{f\i}, x_i) = theta_f(x_f) + SUMA_{j in f} delta_{jf}(x_j) (Ec. 22.183)
Decodabil Local: Daca fiecare theta_i^{delta*} are un maxim unic x_i*, atunci delta* este decodabil local, relaxarea LP este unica si solutia sa este x*. Variabilele decodabil local pot fi "fixate" la valoarea lor MAP.
Convergenta: Coborarea pe coordonate este adesea mai rapida decat coborarea subgradientului in iteratiile initiale, reduce obiectivul monoton si nu necesita parametri de pas. Dezavantajul: nu este garantata convergenta la optimul global deoarece L(delta) este convexa dar nu strict convexa.

Comentariu

Rezumat

Pagina abordeaza problema practica cruciala: cum obtinem asignarea MAP din solutia duala optima. Calculul marginalelor max este bottleneck-ul computational principal. Se prezinta conditiile in care solutia este decodabila local si strategii pentru cazurile problematice (asignari fractionale cu aceeasi valoare optima).

Tipare Cheie

Diferenta cheie fata de coborarea subgradientului: coborarea pe coordonate reduce obiectivul monoton dar poate ramane blocata. O solutie este inlocuirea functiei max cu soft-max pentru a face obiectivul strict convex. Decodarea locala functioneaza surprinzator de bine in practica.

Aplicatii Practice

Decodarea locala a fost folosita cu succes in estimarea CRF-urilor pentru viziune stereo si predictia structurii laterale a proteinelor. Tehnica de "clamping" reduce dimensiunea problemei ramase.

Exemplu din Lumea Reala

In predictia structurii proteinelor, fiecare aminoacid poate adopta diverse conformatii laterale. Decodarea locala identifica rapid aminoacizii cu conformatie unica (de obicei cei ingropati in interiorul proteinei), reducand problema la cei de la suprafata cu conformatii ambigue.

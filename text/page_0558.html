Verosimilitatea marginala si stabilitate numerica in clasificarea GP

Aceasta pagina prezinta calculul verosimilitatii marginale pentru clasificare GP si formularea stabil numeric folosind matricea B.

Verosimilitatea marginala aproximata: log p(y|X) ~ log p(y|f_hat) - 1/2 * f_hat^T K^{-1} f_hat - 1/2 * log|K| - 1/2 * log|K^{-1} + W|, obtinuta prin aproximarea Laplace.
Formulare stabil numerica: Definind B = I_N + W^{1/2} K W^{1/2}, cu valori proprii marginite inferior de 1, se evita inversarea directa a K sau W. Actualizarea IRLS devine f^{new} = K(b - W^{1/2} L^T \ (L \ (W^{1/2} Kb))), unde b = Wf + nabla log p(y|f).

Comentariu

Rezumat

Calculul verosimilitatii marginale in clasificarea GP este mai complex decat in regresie, deoarece f_hat, W si K depind toate de hiperparametrii theta. Se foloseste aproximarea Laplace: log p(y|X) ~ ell(f_hat) - 1/2 log|H|. Pentru stabilitate numerica, se introduce matricea B = I + W^{1/2} K W^{1/2}, ale carei valori proprii sunt bine conditionate.

Tipare Cheie

Matricea B are valori proprii >= 1 (deoarece w_ii = pi_i(1-pi_i) <= 0.25 pentru regresie logistica), ceea ce o face usor de inversat. Lema de inversare matriciala permite exprimarea (K^{-1} + W)^{-1} = K - KW^{1/2} B^{-1} W^{1/2} K, evitand inversarea directa a matricelor mari.

Aplicatii Practice

Stabilitatea numerica este critica in implementarile practice ale GP. Bibliotecile GPy, GPflow si GPML folosesc aceste formulari stabilizate. Fara ele, inversarea K poate duce la instabilitati numerice, special cand datele sunt aproape colineare.

Exemplu din Lumea Reala

In clasificarea imaginilor medicale cu GP, unde numarul de caracteristici poate fi mare si matricea K poate deveni slab conditionata, formularea prin B asigura predictii stabile si fiabile.

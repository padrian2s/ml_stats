Compressed Sensing si Image Inpainting

Pagina introduce Sectiunile 13.8.3 (compressed sensing) si 13.8.4 (image inpainting/denoising), aplicatii practice ale sparse coding.

Compressed sensing (Sectiunea 13.8.3): Se observa y = Rx + e, unde x in R^D, y in R^M (M < D), R este matricea de masurare (cunoscuta), e este zgomot. Se presupune ca x = Wz cu z sparse. Scopul: reconstituirea lui x din mai putine masuratori decat dimensiunile sale, exploatand sparsitatea.
Conditii pentru compressed sensing: R poate fi aleator Gaussian sau alta matrice cu proprietatea RIP (Restricted Isometry Property). Dictionarul W poate fi fix (wavelet, DCT) sau invatat. In aplicatii CS, matricea de masurare R este aleatorea; in aplicatii clasice, R selecteaza componente observate.
Image inpainting (Sectiunea 13.8.4): Imaginea este corupta (text suprapus, zgomot, pixeli lipsa). Se partitioneaza in patch-uri y_i, se defineste R pentru a selecta pixelii vizibili V, si se calculeaza p(y_H|y_V, theta) cu dictionar W invatat. Comentariu

Rezumat

Compressed sensing si image inpainting sunt doua aplicatii cheie ale sparse coding. CS permite reconstructia semnalelor din masuratori insuficiente (sub-Nyquist), exploatand sparsitatea in baza corecta. Inpainting-ul restaureaza regiunile corupte ale imaginilor folosind dictionare sparse invatate din portiunile intacte sau din baze de date externe.

Tipare Cheie

Ambele probleme pot fi formulate ca inferinta Bayesiana in modelul grafic din Figura 13.22: z -> x (sparse coding) -> y (masurare/observare). Diferenta: in CS, R este aleatorea si cunoscuta; in inpainting, R selecteaza pixelii vizibili. In ambele cazuri, sparsitatea lui z este esentiala pentru reconstructie.

Aplicatii Practice

Compressed sensing a revolutionat imagistica medicala (MRI accelerat cu factor 4-8x), radar si seismologie. Image inpainting este standard in editarea foto (Content-Aware Fill in Photoshop) si restaurarea fotografiilor vechi deteriorate.

Exemplu din Lumea Reala

In MRI clinic, compressed sensing reduce timpul de achizitie de la 45 minute la 10 minute pentru scanari 3D, imbunatatind confortul pacientului si capacitatea departamentului de radiologie, fara pierdere semnificativa de calitate diagnostica.

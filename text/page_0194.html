Factorii Bayes si Scala lui Jeffreys

Pagina introduce factorii Bayes ca alternativa bayesiana la testarea ipotezelor si prezinta scala lui Jeffreys pentru interpretarea magnitudinii lor.

Factorul Bayes (Bayes factor): Raportul verosimilitatilor marginale pentru doua modele: BF₁,₀ = p(D|M₁)/p(D|M₀) · p(M₁)/p(M₀) (Ecuatia 5.39). Este un raport de verosimilitate dar cu parametrii integrati.
Ipoteza nula (M₀): Modelul simplu (de exemplu, moneda este corecta, θ=0.5). Ipoteza alternativa (M₁): Modelul complex (de exemplu, moneda poate fi incorecta, θ ∈ [0,1]).
Scala lui Jeffreys: BF < 1/100: evidenta decisiva pentru M₀; 1/10 < BF < 1/3: evidenta moderata pentru M₀; 1 < BF < 3: evidenta slaba pentru M₁; BF > 100: evidenta decisiva pentru M₁.
Conversia BF la posterioara: p(M₀|D) = BF₀,₁/(1 + BF₀,₁) = 1/(BF₁,₀ + 1) (Ecuatia 5.40)

Comentariu

Rezumat

Factorii Bayes sunt alternativa bayesiana la p-value-urile frecventiste. Daca BF₁,₀ > 1, preferem modelul M₁; altfel M₀. Scala lui Jeffreys ofera o interpretare calitativa: BF > 100 este "evidenta decisiva", BF intre 3 si 10 este "evidenta moderata". Cu prior uniform p(M₁) = p(M₀) = 0.5, factorul Bayes se converteste direct in probabilitate posterioara.

Tipare Cheie

Spre deosebire de p-value-uri, factorii Bayes au o interpretare directa in termeni de probabilitate si nu depind de "spatiul esantionului" sau de intentia experimentatorului. Sunt simetrici: pot oferi evidenta atat pentru cat si impotriva ipotezei nule.

Aplicatii Practice

Factorii Bayes sunt din ce in ce mai folositi in stiintele sociale si medicina ca alternativa la testele de semnificatie clasice (p < 0.05). Avantajul lor principal este ca pot cuantifica evidenta in favoarea ipotezei nule, nu doar impotriva ei.

Exemplu din Lumea Reala

In testarea daca o moneda este corecta: M₀ presupune θ=0.5, M₁ presupune θ ∈ [0,1]. Verosimilitatea marginala sub M₀ este simpla: p(D|M₀) = (1/2)^N. Sub M₁ cu prior Beta(1,1), p(D|M₁) = B(α₁+N₁, α₀+N₀)/B(α₁,α₀).

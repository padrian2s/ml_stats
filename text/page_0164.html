Distributia Normal-Inverse-Wishart (NIW) ca prior conjugat

Aceasta pagina introduce distributia Normal-Inverse-Wishart (NIW) ca prior conjugat pentru inferenta parametrilor unei distributii Gaussiene multivariate cand atat media cat si matricea de covarianta sunt necunoscute.

Distributia NIW: NIW(μ, Σ | m_0, κ_0, ν_0, S_0) = N(μ | m_0, Σ/κ_0) x IW(Σ | S_0, ν_0). Combina o distributie Gaussiana pentru medie (conditionata de covarianta) cu o distributie Inverse-Wishart pentru matricea de covarianta.
Interpretarea parametrilor NIW: m_0 este media prior pentru μ; κ_0 masoara cat de puternic credem in aceasta medie; S_0 este (proportional cu) priorul pentru Σ; ν_0 masoara cat de puternic credem in acest prior pentru covarianta.

Comentariu

Rezumat

Distributia NIW este priorul conjugat natural pentru modelul Gaussian multivariat cu parametri necunoscuti. Pagina prezinta formulele detaliate ale densitatii NIW si constanta de normalizare Z_NIW. Se discuta si priorul neinformativ improper, care are forma NIW(μ, Σ | 0, 0, 0, 0I), si recomandarea practica de a utiliza priori slab informative bazate pe date.

Tipare Cheie

1) Priorul NIW are patru parametri, dar in realitate doar trei grade de libertate, deoarece incertitudinea asupra mediei este proportionala cu varianta. 2) Priorul neinformativ improper se obtine cand κ_0 tinde la 0. 3) O alegere practica comuna pentru priorul slab informativ: S_0 = diag(S_x)/N, ν_0 = D + 2, μ_0 = media esantionului, κ_0 mic (ex: 0.01).

Aplicatii Practice

Priorul NIW este esential in modelele de amestec Gaussian (GMM) cu inferenta Bayesiana, in analiza discriminanta Bayesiana, si in orice model unde trebuie estimata o distributie Gaussiana multivariata cu incertitudine asupra parametrilor.

Exemplu din Lumea Reala

In segmentarea de imagini medicale, fiecare tip de tesut este modelat ca o distributie Gaussiana multivariata (peste intensitatile de culoare). Priorul NIW permite incorporarea cunostintelor anterioare despre distributia tipica a intensitatilor pentru fiecare tip de tesut, fiind crucial cand avem putine date de antrenament etichetate.

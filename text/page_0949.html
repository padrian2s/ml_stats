Sectiunea 27.2.4 - Modelul GaP si Factorizarea Matriceala Non-negativa

Pagina introduce modelul Gamma-Poisson (GaP) si conexiunea sa cu factorizarea matriceala non-negativa (NMF), precum si priorul spike-and-Gamma pentru raritate.

Modelul Gamma-Poisson (GaP): Variabilele latente z_i⁺ sunt non-negative cu prior Gamma: p(z_i⁺) = Π_k Ga(z_ik⁺|α_k, β_k). Verosimilitatea: p(n_i|z_i⁺) = Π_v Poi(n_iv|b_v^T z_i⁺) (Ecuatii 27.15-27.16). (Canny 2004)
Factorizarea Matriceala Non-negativa (NMF): Se obtine din GaP cand α_k = β_k = 0 (Lee si Seung 2001). NMF nu este un model generativ probabilistic propriu-zis deoarece nu specifica un prior corect pentru z_i⁺, si sufera de supraajustare.
Priorul spike-and-Gamma: p(z_ik⁺) = ρ_k I(z_ik⁺ = 0) + (1 − ρ_k)Ga(z_ik⁺|α_k, β_k) (Ecuatia 27.17), unde ρ_k este probabilitatea de spike la zero. Aceasta incurajeaza raritatea in variabilele latente.
Conexiunea GaP-mPCA: Cand GaP este conditionat pe o lungime fixa L_i, se reduce la modelul mPCA. Aceasta deoarece o suma de variabile aleatoare Poisson, conditionata pe suma lor, devine o distributie multinomiala.

Comentariu

Rezumat

GaP ofera un cadru probabilistic riguros pentru factorizarea matriceala non-negativa, inlocuind abordarea ad-hoc a NMF cu un model generativ complet. Priorul Gamma asigura non-negativitatea variabilelor latente, iar verosimilitatea Poisson este naturala pentru date de contorizare.

Tipare Cheie

NMF poate fi vazut ca o versiune degenerata a GaP, fara prior propriu-zis. Modelul GaP rezolva problemele de supraajustare ale NMF prin specificarea unui prior corect. Priorul spike-and-Gamma adauga o componenta de selectie a caracteristicilor, permitand ca unele componente latente sa fie exact zero.

Aplicatii Practice

NMF este extrem de popular in: analiza de imagini (decomponerea fetelor in parti componente: ochi, nas, gura), analiza audio (separarea surselor sonore), bioinformatica (analiza profilelor de expresie genica) si text mining (descoperirea temelor).

Exemplu din Lumea Reala

In analiza fetelor, NMF descompune imaginile in parti semnificative (ochi, nas, gura), spre deosebire de PCA care produce "eigenfaces" holiste. GaP imbunatateste aceasta decomponere prin incertitudine probabilistica si evitarea supraajustarii.

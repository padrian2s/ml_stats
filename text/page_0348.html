10.2.4-10.2.5 Linkaj Genetic (continuare) si Modele Grafice Gaussiene Directionate

Pagina termina discutia despre analiza de linkaj genetic si introduce modelele grafice gaussiene directionate (Gaussian Bayes nets), derivand distributia comuna gaussiana din CPD-urile liniare gaussiene.

Problema inversa in genetica: Maparea de la fenotip la genotip este ambigua (many-to-one). Informatia de la rude poate fi folosita pentru dezambiguizare prin inferenta pe DGM-ul genealogic.
Sarcina de linkaj genetic: Determinarea locatiei unei gene care cauzeaza o boala. Se construiesc L+1 modele (cate unul pentru fiecare pozitie posibila intre markeri), se calculeaza verosimilitatea fiecaruia si se alege modelul cu verosimilitate maxima.
Model grafic gaussian directionat (GGM directionat): DGM in care toate variabilele sunt continue si CPD-urile sunt gaussiene liniare: p(x_t|x_{pa(t)}) = N(x_t|μ_t + w_t^T x_{pa(t)}, σ_t²). Produsul tuturor CPD-urilor rezulta intr-o distributie comuna gaussiana multivariata p(x|μ, Σ).
Retea Bayes gaussiana: Alt nume pentru GGM directionat. Distributia comuna rezultanta este N(x|μ, Σ), unde μ este concatenarea mediilor locale si Σ se deriva din ponderile de regresie.
CPD liniar gaussian (10.14): p(x_t|x_{pa(t)}) = N(x_t|μ_t + w_t^T x_{pa(t)}, σ_t²).
Forma vectoriala (10.15): x_t = μ_t + Σ_{s∈pa(t)} w_{ts}(x_s - μ_s) + σ_t z_t, unde z_t ~ N(0, 1).
Forma matriceala (10.16): (x - μ) = W(x - μ) + Sz, unde S = diag(σ) si W este matricea de ponderi.

Comentariu

Rezumat

Analiza de linkaj genetic este computational intensiva deoarece necesita marginalizarea tuturor variabilelor ascunse Z si G. Se folosesc metode exacte (eliminarea variabilelor) sau aproximative (inferenta variationala). Modelele grafice gaussiene directionate demonstreaza o proprietate remarcabila: produsul CPD-urilor gaussiene liniare rezulta exact intr-o distributie comuna gaussiana multivariata, legand modelele grafice de algebra liniara.

Tipare Cheie

CPD-ul liniar gaussian are o forma naturala: media conditionata este o functie liniara a parintilor, iar varianta conditionata este constanta. Matricea W este inferior triunghiulara (datorita ordonarii topologice), facand inversarea triviala. Ponderile de regresie w_{ts} corespund descompunerii Cholesky a matricei de covarianta Σ.

Aplicatii Practice

Retelele Bayes gaussiene sunt folosite in modelarea financiara (dependente intre active), inginerie (analiza de fiabilitate) si stiintele climei (modelarea dependentelor spatiale intre variabile meteorologice).

Exemplu din Lumea Reala

Un model de risc financiar poate fi reprezentat ca retea Bayes gaussiana: variabilele macroeconomice (rata dobanzii, inflatie) sunt parintii variabilelor de performanta a companiilor (venituri, profit), care la randul lor sunt parintii preturilor actiunilor. CPD-urile liniare gaussiene captureaza aceste dependente.

Prior-ul Jeffreys pentru Bernoulli, Multinomial si Parametri de Locatie/Scala

Pagina calculeaza explicit prior-ul Jeffreys pentru distributiile Bernoulli si multinomiale si introduce prior-urile Jeffreys pentru parametri de locatie si scala.

Jeffreys pentru Bernoulli: I(θ) = 1/(θ(1-θ)), deci p(θ) ∝ θ^(-1/2)(1-θ)^(-1/2) = Beta(1/2, 1/2) (Ecuatia 5.61)
Jeffreys pentru Multinomial: p(θ) ∝ Dir(1/2, ..., 1/2) (Ecuatia 5.62) - diferit de prior-ul uniform Dir(1,...,1) sau Dir(1/K,...,1/K).
Prior invariant la translatie: p(μ) ∝ 1 pentru un parametru de locatie (media gaussiana). Satisface proprietatea ca masa probabilitatii atribuita oricarui interval [A,B] este aceeasi ca cea atribuita [A-c, B-c] (Ecuatia 5.63).
Prior invariant la scala: p(s) ∝ 1/s pentru un parametru de scala (varianta gaussiana). Satisface proprietatea ca masa probabilitatii atribuita [A,B] este aceeasi ca cea atribuita [A/c, B/c] (Ecuatia 5.64).

Comentariu

Rezumat

Se demonstreaza explicit ca prior-ul Jeffreys pentru Bernoulli este Beta(1/2, 1/2), nu prior-ul uniform Beta(1,1). Pentru multinomiale cu K stari, este Dir(1/2,...,1/2). Pentru parametri de locatie, prior-ul Jeffreys este uniform p(μ) ∝ 1 (impropriu dar acceptabil). Pentru parametri de scala, este p(σ²) ∝ 1/σ², care este si impropriu dar da posterioare proprii cu N ≥ 2.

Tipare Cheie

Exista o simetrie eleganta: prior-ul invariant la translatie este uniform pe scala naturala, iar prior-ul invariant la scala este uniform pe scala logaritmica (deoarece p(s) ∝ 1/s inseamna p(log s) = constant).

Aplicatii Practice

In practica bayesiana, prior-urile Jeffreys sunt adesea "puncte de plecare" care sunt apoi ajustate pe baza cunostintelor de domeniu. In MCMC si Stan, se folosesc frecvent prior-uri vag informative (de exemplu, N(0, 100²)) in locul prior-urilor improprii.

Exemplu din Lumea Reala

Cand estimam varianta zgomotului de masurare in fizica, prior-ul Jeffreys p(σ²) ∝ 1/σ² reflecta faptul ca nu avem preferinta pentru nicio scala particulara: nu stim daca zgomotul este de ordinul milimetrilor sau metrilor.

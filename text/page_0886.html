Ajustarea unui model de amestec DP

Pagina descrie esantionatorul Gibbs colaps pentru ajustarea unui model de amestec cu procese Dirichlet (DPMM), derivand distributia conditionala completa pentru atribuirile de cluster.

Esantionator Gibbs colaps pentru DPMM: Modificarea esantionatorului Gibbs colaps pentru amestecuri finite, adaptata pentru un numar potential infinit de clustere, unde z_i poate fi atribuit unui cluster nou k*.
Distributia conditionala: p(z_i = k|z_{-i}, α) = N_{k,-i}/(α+N-1) pentru clustere existente, sau α/(α+N-1) pentru un cluster nou (Ecuatia 25.35)
Verosimilitatea marginala: p(x_i, x_{-i,k}|λ) = ∫ p(x_i|θ_k) [∏_{j≠i:z_j=k} p(x_j|θ_k)] H(θ_k|λ)dθ_k (Ecuatia 25.37)
Distributie predictiva priori: p(x_i|x_{-i}, z_{-i}, z_i = k*, λ) = ∫ p(x_i|θ)H(θ|λ)dθ (Ecuatia 25.38)

Comentariu

Rezumat

Esantionatorul Gibbs colaps pentru DPMM este similar cu cel pentru amestecuri finite, cu diferenta ca trebuie considerat cazul z_i = k* (cluster nou). Probabilitatea conditionala combina doua componente: priorul din procesul restaurantului chinezesc si verosimilitatea predictiva posterioara pentru fiecare cluster.

Tipare Cheie

Formularea este echivalenta cu ecuatia Polya urn in limita K → ∞. Schimbabilitatea permite tratarea fiecarui z_i ca ultimul client care intra in restaurant. Distributia predictiva pentru un cluster nou se reduce la priorul predictiv evaluat la x_i.

Aplicatii Practice

Algoritmul este implementat in pachete software pentru clustering Bayesian nonparametric, fiind folosit in analiza de imagini medicale, segmentarea secventelor genetice si modelarea topicilor in colectii de documente.

Exemplu din Lumea Reala

In analiza expresiei genice, fiecare gena este un punct de date. Esantionatorul Gibbs colaps poate grupa genele in clustere functionale fara a specifica numarul de grupuri, descoperind automat module de co-expresie relevante biologic.

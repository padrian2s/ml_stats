Asocierea Datelor, Urmarirea Multi-Tinta si Corectia Expectatiei

Pagina prezinta formulele de colapsare GPB2, modele multiple interactive (IMM), asocierea datelor si urmarirea multi-tinta ca aplicatii ale SLDS.

Formulele de colapsare GPB2: π_j = Σ_i π_{ij} (ec. 18.139), π_{j|i} = π_{ij} / Σ_{j'} π_{ij'} (ec. 18.140), μ_j = Σ_i π_{j|i} μ_{ij} (ec. 18.141), Σ_j = Σ_i π_{j|i} (Σ_{ij} + (μ_{ij} − μ_j)(μ_{ij} − μ_j)^T) (ec. 18.142).
Corectia expectatiei (Expectation Correction): Extensie offline a ADF care imbunatateste aproximarile iterativ, descrisa de Barber si colaboratorii.
Asocierea datelor (Data Association): Problema determinarii corespondentei intre K' detectii y_{tk} si K obiecte z_{tj}. Poate exista K' < K (ocluziune), K' > K (alarme false) sau K' = K.
Euristica celui mai apropiat vecin: Se calculeaza o matrice de ponderi K × K' bazata pe compatibilitatea intre obiectul j si masurarea k, apoi se rezolva problema de potrivire bipartita maximala folosind algoritmul Ungar in O(N³).

Comentariu

Rezumat

Pagina extinde discutia despre SLDS la doua aplicatii importante: asocierea datelor si urmarirea multi-tinta. Problema asocierii datelor apare cand nu se stie care detectie corespunde carui obiect, iar urmarirea multi-tinta adauga complexitatea unui numar variabil de obiecte.

Tipare Cheie

Asocierea datelor este o problema de potrivire combinatorie care poate fi formulata ca SSM hibrid. Algoritmul Ungar ofera o solutie polinomiala pentru potrivirea bipartita. Urmarirea multi-tinta necesita un spatiu de stari cu dimensionalitate variabila.

Aplicatii Practice

Asocierea datelor si urmarirea multi-tinta sunt fundamentale in: supravegherea video, conducerea autonoma, monitorizarea traficului naval, urmarirea celulelor in microscopie si detectia si urmarirea obiectelor in imagini satelitare.

Exemplu din Lumea Reala

Intr-un sistem de conducere autonoma, camerele si lidar-ul detecteaza multiple obiecte (masini, pietoni, biciclisti). Asocierea datelor determina care detectie din cadrul curent corespunde carui obiect urmarit din cadrul anterior, chiar si cand obiecte se ocluzioneza reciproc.

Estimarea starii MAP in modele grafice discrete

Aceasta pagina introduce problema estimarii MAP (Maximum A Posteriori) in modele grafice discrete si prezinta relaxarea prin programare liniara (LP relaxation).

Estimarea MAP: x* = arg max_{x in X^m} p(x|theta) = arg max sum theta_i(x_i) + sum theta_f(x_f) = arg max theta^T*phi(x). Se cauta configuratia de variabile cu probabilitatea maxima.
Relaxarea LP: In loc de a optimiza peste politopul marginal M(G) (exponential de multe fete), se relaxeaza la L(G) (polinomial de multe constrangeri): max theta^T*mu <= max theta^T*tau. Daca solutia este integrala, este exacta; daca este fractionara, este o aproximare.
Relaxare de ordinul intai: Constrangerile impuse corespund consistentei pe arbori (treewidth 1). Se pot impune constrangeri de ordin mai mare pentru treewidth mai mare, obtinand aproximari mai stranse dar mai costisitoare.

Comentariu

Rezumat

Pagina face tranzitia de la inferenta marginala la estimarea MAP, formuland-o ca problema de optimizare liniara peste politopul marginal. Relaxarea LP inlocuieste M(G) cu L(G), transformand o problema NP-hard intr-o problema polinomiala (LP), cu garantia ca solutia este o margine superioara pe obiectivul MAP.

Tipare Cheie

Estimarea MAP difera de inferenta marginala prin absenta termenului de entropie (sau echivalent, limita de temperatura zero). Aceasta simplificare face obiectivul liniar (nu concav), dar face optimizarea combinatorie (nu continua).

Aplicatii Practice

Estimarea MAP este esentiala in etichetarea secventelor (NER, POS tagging), segmentarea imaginilor si configurarea optima in sisteme complexe.

Exemplu din Lumea Reala

In etichetarea gramaticala a textului (POS tagging), estimarea MAP gaseste secventa de etichete (substantiv, verb, adjectiv...) cu probabilitatea maxima data secventa de cuvinte, folosind algoritmul Viterbi pe CRF-uri.

Dependenta MAP de Parametrizare si Intervale Credibile

Pagina demonstreaza prin exemplul distributiei Bernoulli ca MAP depinde de parametrizare si introduce conceptul de interval credibil bayesian.

Interval credibil (Credible interval): Un interval C = (l, u) care contine 100(1-α)% din masa probabilitatii posterioare: C_α(D) = (l, u) : P(l ≤ θ ≤ u|D) = 1 - α (Ecuatia 5.8).
Interval central: Un interval credibil ales astfel incat exista (1-α)/2 masa probabilitatii in fiecare coada a distributiei.
Exemplu Bernoulli: Cu μ ~ Uniform(0,1), reparametrizand θ = √μ obtinem p_θ(θ) = 2θ cu θ_MAP = 1, iar cu φ = 1-√(1-μ) obtinem p_φ(φ) = 2(1-φ) cu φ_MAP = 0. MAP-ul depinde complet de parametrizare!
Estimare independenta de parametrizare: θ_hat = argmax_θ p(D|θ)p(θ)|I(θ)|^(-1/2) (Ecuatia 5.7), unde I(θ) este matricea de informatie Fisher.

Comentariu

Rezumat

Exemplul lui Michael Jordan demonstreaza dramatic dependenta MAP de parametrizare: pentru aceeasi distributie Bernoulli, trei parametrizari diferite dau trei MAP-uri diferite (mode de la 0 la 1). Solutia propusa (Ecuatia 5.7) foloseste matricea de informatie Fisher pentru independenta de parametrizare, dar este dificil de optimizat. Sectiunea 5.2.2 introduce intervalele credibile ca alternativa la estimarile punctuale.

Tipare Cheie

Intervalul central se calculeaza folosind l = F⁻¹(α/2) si u = F⁻¹(1-α/2), unde F este CDF-ul posterioarei. Pentru o posterioara gaussiana N(0,1) cu α=0.05, obtinem intervalul (-1.96, 1.96), justificand conventia μ ± 2σ.

Aplicatii Practice

Intervalele credibile bayesiene sunt mult mai intuitive decat intervalele de incredere frecventiste: un interval credibil de 95% inseamna ca exista o probabilitate de 95% ca parametrul sa fie in acel interval, conditionat de datele observate. Aceasta interpretare directa le face foarte utile in comunicarea rezultatelor.

Exemplu din Lumea Reala

Daca observam 47 de capete din 100 de aruncari de moneda, posterioara este Beta(48,54), iar intervalul credibil de 95% este aproximativ (0.3749, 0.5673). Aceasta ne spune direct ca suntem 95% siguri ca probabilitatea de cap este in acest interval.

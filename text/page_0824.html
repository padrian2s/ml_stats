EP si LBP pe grafuri factor - Derivarea echivalentei

Aceasta pagina demonstreaza formal echivalenta intre EP si LBP pe grafuri factor, aratand ca mesajele EP corespund mesajelor BP si introduce tree EP.

EP pe grafuri factor: Multimea convexa L(phi, Phi) peste care EP optimizeaza este identica cu L(G) din LBP. Multiplicatorii Lagrange lambda_uv corespund mesajelor BP: M_uv(x_v) = exp(lambda_uv(x_v)).
Potrivirea momentelor ca transmitere de mesaje: Constrangerile de potrivire a momentelor (E_q[x_s] = E_{q_uv}[x_s]) se rezolva prin transmiterea mesajelor sum-product standard de-a lungul muchiei u-v.
Tree EP: O extensie naturala in care distributia de baza este un arbore (nu un produs). Se adauga cate o muchie la un moment dat, se absorbe efectul si se aproximeaza distributia rezultanta cu un nou arbore. Este mai precis si uneori mai rapid decat LBP.
Generalizare: Prin considerarea altor distributii de baza structurate, se pot deriva algoritmi care depasesc performanta generalized BP (Welling et al. 2005).

Comentariu

Rezumat

Pagina completeaza demonstratia formala ca EP pe modele grafice pairwise este echivalent cu LBP, si introduce tree EP ca extensie directa. Perspectiva EP ofera un cadru principial pentru proiectarea de noi algoritmi de inferenta.

Tipare Cheie

Echivalenta EP-LBP demonstreaza ca transmiterea mesajelor BP poate fi inteleasa ca potrivire iterativa a momentelor, o perspectiva care permite extensii elegante la distributii continue si modele cu factori complecsi.

Aplicatii Practice

Tree EP este implementat in biblioteci precum Infer.NET si oferape o alternativa directa la LBP cu acuratete imbunatatita la cost computational similar.

Exemplu din Lumea Reala

In sistemele de recomandare ale Microsoft (Xbox, Bing), Infer.NET foloseste EP si tree EP pentru a face inferenta rapida in modele probabilistice complexe cu milioane de utilizatori si itemi.

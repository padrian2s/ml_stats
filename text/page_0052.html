Capitolul 1: Regresia Logistica

Generalizarea regresiei liniare la clasificare binara prin functia sigmoid si distributia Bernoulli.

Regresia Logistica: Model de clasificare binara care combina o combinatie liniara a intrarilor cu functia sigmoid: p(y|x, w) = Ber(y|sigm(w^T x)) (ecuatia 1.11). Desi se numeste "regresie", este de fapt un model de clasificare.
Distributia Bernoulli: Distributie de probabilitate pentru variabile binare y ∈ {0, 1}, mai potrivita decat Gaussiana pentru clasificare.
Functia Sigmoid (Logistica): sigm(η) = 1/(1 + exp(-η)) = e^η/(e^η + 1) (ecuatia 1.10). Functie in forma de S care mapeaza intreaga linie reala in intervalul [0, 1], permitand interpretarea ca probabilitate.
Functie de Strivire (Squashing Function): Alt nume pentru functia sigmoid, deoarece "striveste" intreaga linie reala in intervalul [0, 1].
Figura 1.19: (a) Functia sigmoid: sigm(-∞) = 0, sigm(0) = 0.5, sigm(∞) = 1. (b) Regresia logistica aplicata la scoruri SAT: punctele negre sunt datele de antrenament, cercurile rosii sunt probabilitatile prezise. Crucile verzi arata doi studenti cu acelasi scor SAT (525) dar rezultate diferite.
Formule Cheie: p(y|x, w) = Ber(y|μ(x)) (ec. 1.8); μ(x) = sigm(w^T x) (ec. 1.9); sigm(η) = 1/(1+exp(-η)) (ec. 1.10).

Comentariu

Rezumat

Sectiunea 1.4.6 introduce regresia logistica, generalizand regresia liniara la clasificare binara. Doua modificari cheie: (1) distributia Gaussiana este inlocuita cu Bernoulli, (2) combinatia liniara este trecuta prin functia sigmoid pentru a obtine o probabilitate in [0,1]. Exemplul cu scoruri SAT arata ca datele nu sunt perfect separabile liniar.

Tipare Cheie

Functia sigmoid este fundamentala in ML: apare in regresia logistica, retele neuronale si multe alte modele. Regresia logistica produce probabilitati calibrate, nu doar etichete, permitand cuantificarea incertitudinii. Numele "regresie logistica" este inselator - este de fapt un model de clasificare.

Aplicatii Practice

Regresia logistica este unul dintre cele mai utilizate modele de clasificare: scoring de credit bancar, predictia riscului medical, analiza sentimentelor, prezicerea click-urilor pe reclame (CTR) si multe altele.

Exemplu din Lumea Reala

O banca foloseste regresia logistica pentru scoring de credit: probabilitatea de neplata p(y=1|x) este modelata ca sigm(w^T x), unde x include venitul, istoricul de credit, varsta si alte caracteristici. Pragul de 0.5 determina decizia de aprobare sau respingere a creditului.

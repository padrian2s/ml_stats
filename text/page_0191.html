Verosimilitatea Marginala: Modelele Beta-Binomial si Dirichlet-Multinomial

Pagina prezinta calculul explicit al verosimilitatii marginale pentru modelele Beta-Binomial si Dirichlet-Multinomial, si introduce modelul Gaussiana-Gaussiana-Wishart.

Modelul Beta-Binomial: Verosimilitatea marginala se obtine combinand prior-ul Beta(a,b) cu verosimilitatea binomiala, rezultand p(D) = C(N,N₁) · B(a+N₁, b+N₀) / B(a,b) (Ecuatia 5.23).
Modelul Dirichlet-Multinomial: Generalizarea la K clase: p(D) = B(N+α) / B(α) (Ecuatia 5.24), unde B(α) = ∏ᵏ Γ(αₖ) / Γ(Σₖ αₖ).
Formula explicita Dirichlet-Multinomial: p(D) = Γ(Σₖ αₖ) / Γ(N + Σₖ αₖ) · ∏ₖ Γ(Nₖ + αₖ) / Γ(αₖ) (Ecuatia 5.26)

Comentariu

Rezumat

Se deriveaza verosimilitatea marginala pentru doua modele fundamentale cu prior conjugat. Pentru Beta-Binomial, rezultatul este un raport de functii Beta. Pentru Dirichlet-Multinomial, formula este analogica dar generalizata la K clase. Se introduce si modelul Gaussiana-Gaussiana-Wishart (NIW) pentru date multivariate continue.

Tipare Cheie

In toate cazurile, verosimilitatea marginala se reduce la raportul constantelor de normalizare ale posterioarei si prior-ului. Aceasta structura comuna reflecta proprietatea priori-lor conjugate: posterioara are aceeasi forma functionala ca prior-ul.

Aplicatii Practice

Formula Dirichlet-Multinomial (Ecuatia 5.26) este folosita extensiv in modelarea textelor (LDA - Latent Dirichlet Allocation), clasificarea documentelor si modelarea limbajului. Permite calculul exact al verosimilitatii marginale fara aproximari.

Exemplu din Lumea Reala

In clasificarea textelor cu Naive Bayes multinomial, verosimilitatea marginala Dirichlet-Multinomial permite compararea directa a modelelor cu vocabulare de dimensiuni diferite, selectand automat nivelul optim de complexitate.

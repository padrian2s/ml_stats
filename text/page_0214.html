Rata Falselor Descoperiri si Alte Topici in Teoria Deciziei

Pagina introduce rata falselor descoperiri (FDR) pentru testarea ipotezelor multiple si prezinta pe scurt banditi contextuali si teoria utilitatii.

Rata falselor descoperiri (False Discovery Rate - FDR): FDR(τ,D) = FD(τ,D)/N(τ,D) (Ecuatia 5.119), unde FD este numarul asteptat de false descoperiri si N este numarul total de descoperiri.
Testarea ipotezelor multiple: Cand facem N decizii binare simultan (de exemplu, care gene sunt exprimate diferential), trebuie sa controlam rata globala de erori, nu doar erorile individuale.
Banditi multi-brat (Multi-armed bandits): Problema de decizie secventiala cu K actiuni posibile, fiecare cu recompensa necunoscuta r_k. Indicii Gittins rezolva optim compromisul explorare-exploatare.
FD asteptat bayesian: FD(τ,D) = Σᵢ (1-pᵢ) · I(pᵢ > τ) (Ecuatia 5.118), unde pᵢ = p(yᵢ=1|D) este credinta ca obiectul i prezinta fenomenul.
Abordarea probabilitatii posterioare directe: Se ajusteaza pragul τ pentru a atinge FDR-ul dorit (de exemplu, α = 0.05).

Comentariu

Rezumat

FDR este esential cand se fac mii de teste simultane (genomica, neuroimagistica). Abordarea bayesiana calculeaza probabilitatile posterioare pᵢ = p(yᵢ=1|D) si ajusteaza pragul τ pentru a controla FDR-ul. Estimarea conjuncta a pᵢ-urilor prin modele ierarhice permite imprumutul de forta statistica, reducand FDR-ul. Banditi contextuali extind teoria deciziei la situatii secventiale cu incertitudine.

Tipare Cheie

Diferenta fata de corectia Bonferroni frecventista: abordarea bayesiana a FDR-ului estimeaza probabilitatile posterioare direct, nu ajusteaza p-value-urile. Aceasta este mai eficienta statistic, mai ales cu modele ierarhice care permit partajarea informatiei intre teste.

Aplicatii Practice

In genomica, se testeaza simultan ~20000 gene pentru expresie diferentiala. Controlul FDR-ului la 5% inseamna ca ne asteptam ca doar 5% din genele "descoperite" sa fie false descoperiri. In publicitate online, banditi contextuali selecteaza optim reclamele de afisat.

Exemplu din Lumea Reala

In studii GWAS (Genome-Wide Association Studies), se testeaza milioane de variante genetice. Fara controlul FDR, mii de asocieri false ar fi raportate. Abordarea bayesiana ierarhica permite descoperiri mai precise prin modelarea structurii genetice.

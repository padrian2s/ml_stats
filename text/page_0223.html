Distributia de esantionare a unui estimator si metoda bootstrap

Aceasta pagina prezinta conceptul de distributie de esantionare a unui estimator, metoda bootstrap (parametric si non-parametric) pentru aproximarea distributiei de esantionare, si comparatia cu posteriorul Bayesian.

Bootstrap parametric: Se genereaza S seturi de date artificiale din modelul estimat p(.|θ_hat(D)), se aplica estimatorul pe fiecare set, si se foloseste distributia empirica a estimarilor ca aproximare a distributiei de esantionare.
Bootstrap non-parametric: Se genereaza seturi de date prin reeesantionare cu inlocuire din datele originale D (nu dintr-un model parametric). Nu necesita specificarea unui model, dar poate fi mai lent.

Comentariu

Rezumat

Pagina explica cum se construieste si aproximeaza distributia de esantionare. Figura 6.1 arata distributia bootstrap a MLE pentru un model Bernoulli: cu N=10 este asimetrica, cu N=100 devine aproape Gaussiana (conform teoriei esantioanelor mari). Bootstrap-ul este prezentat ca o alternativa la inferenta Bayesiana cand priorul este neinformativ, dar este conceptual diferit si poate fi mai costisitor computational.

Tipare Cheie

1) Distributia de esantionare este definita prin repetarea ipotetica a experimentului. 2) Pentru N mic, distributia de esantionare poate fi departe de Gaussiana. 3) Bootstrap-ul non-parametric este model-free dar poate esua pentru statistici ale valorilor extreme. 4) Similitudinea bootstrap-posterior tine doar pentru priori slabi si modele simple.

Aplicatii Practice

Bootstrap-ul este folosit pe scara larga in ecologie (estimarea biodiversitatii), in finante (Value-at-Risk), in genomica (intervalele de incredere pentru arbori filogenetici), si in invatarea automata (bagging, random forests sunt forme de bootstrap aplicat la predictie).

Exemplu din Lumea Reala

In genomica, cand se construieste un arbore filogenetic (arborele evolutiei) din secvente ADN, bootstrap-ul este metoda standard pentru evaluarea increderii in fiecare ramura a arborelui. Se reeesantioneaza pozitiile din secventa, se reconstruieste arborele de fiecare data, si se raporteaza procentul de arbori bootstrap care contin fiecare ramura. O ramura cu 95%+ bootstrap support este considerata fiabila.

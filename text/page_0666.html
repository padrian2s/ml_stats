SLAM - Detalii tehnice si solutii

Aceasta pagina continua discutia despre SLAM, detaliind problema corelatiilor si solutiile: taierea muchiilor si FastSLAM.

Problema entanglement in SLAM: Initial reperele sunt necorelate, dar pe masura ce robotul se misca si observa diferite repere, pozitia robotului creaza corelatii intre toate reperele. Matricea de precizie devine densa, facand inferenta exacta O(K^3) unde K este numarul de repere. Doua solutii principale: junction tree thinning si FastSLAM.
FastSLAM: Exploateaza factorizarea p(L|x_{1:T}, y_{1:T}) = prod_i p(L^i|x_{1:T}, y_{1:T}): conditionat pe traiectoria completa a robotului x_{1:T}, reperele devin independente. FastSLAM combina filtrarea Kalman (pentru repere) cu particle filtering (pentru traiectorie), fiind scalabil la harti mari.

Comentariu

Rezumat

Problema principala in SLAM este ca corelatiile dintre repere cresc in timp. Doua abordari: (1) Junction tree thinning - elimina muchiile slabe din modelul grafic Gaussian, reducand complexitatea (Paskin 2003). (2) FastSLAM - observa ca daca cunoastem traiectoria robotului x_{1:T}, reperele sunt conditionate independente: p(L|x_{1:T}, y_{1:T}) = prod p(L^i|x_{1:T}, y_{1:T}). Traiectoria este estimata cu particle filtering, iar fiecare particula mentine K filtre Kalman independente pentru repere. Aceasta reduce complexitatea dramatic si permite aplicarea la harti cu mii de repere.

Tipare Cheie

Rao-Blackwellization: FastSLAM combina particle filtering (pentru componenta neliniara - traiectoria) cu filtrarea Kalman exacta (pentru componenta liniara - pozitiile reperelor). Aceasta reducere a variantei prin marginalizare partiala este un tipar general important in inferenta Monte Carlo.

Aplicatii Practice

FastSLAM este implementat in ROS (Robot Operating System) si este folosit de multi roboti autonomi. Alternativele moderne includ: ORB-SLAM (bazat pe features vizuale), LSD-SLAM (bazat pe imagini dense), si LOAM (bazat pe LiDAR). In practica, se folosesc combinatii de senzori (sensor fusion).

Exemplu din Lumea Reala

Dronele de livrare (Amazon Prime Air) folosesc variante ale SLAM-ului pentru a naviga in medii necunoscute, combinand date de la camere stereo, LiDAR si GPS. FastSLAM permite dronei sa construiasca o harta 3D a mediului si sa-si planifice traiectoria evitand obstacolele, chiar si cand semnalul GPS este slab sau absent.

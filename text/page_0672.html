Filtrul Kalman - Pasul de predictie si pasul de masurare

Aceasta pagina detaliaza ecuatiile filtrului Kalman: pasul de predictie si pasul de masurare (actualizare).

Pasul de predictie: p(z_t|y_{1:t-1}) = N(z_t|mu_{t|t-1}, Sigma_{t|t-1}), unde mu_{t|t-1} = A_t mu_{t-1} + B_t u_t si Sigma_{t|t-1} = A_t Sigma_{t-1} A_t^T + Q_t. Propaga starea de credinta prin modelul de tranzitie, crescand incertitudinea prin adaugarea zgomotului de sistem Q_t.
Pasul de masurare: p(z_t|y_{1:t}) = N(z_t|mu_t, Sigma_t), unde mu_t = mu_{t|t-1} + K_t r_t si Sigma_t = (I - K_t C_t) Sigma_{t|t-1}. Rezidualul r_t = y_t - y_hat_t (inovatia), matricea de castig Kalman K_t = Sigma_{t|t-1} C_t^T S_t^{-1}, si S_t = C_t Sigma_{t|t-1} C_t^T + R_t (covarianta inovatiei).

Comentariu

Rezumat

Filtrul Kalman alterna intre doua etape: (1) Predictie: propaga media si covarianta prin modelul de tranzitie liniar. Media predisa mu_{t|t-1} = A mu_{t-1} + B u_t, covarianta predisa Sigma_{t|t-1} = A Sigma_{t-1} A^T + Q (creste incertitudinea). (2) Masurare/Actualizare: incorporeaza noua observatie y_t. Rezidualul r_t = y_t - C mu_{t|t-1} masoara surpriza. Matricea de castig Kalman K_t determina cat de mult se ajusteaza estimarea: K_t = Sigma_{t|t-1} C^T (C Sigma_{t|t-1} C^T + R)^{-1}. Media actualizata mu_t = mu_{t|t-1} + K_t r_t, covarianta actualizata Sigma_t = (I - K_t C) Sigma_{t|t-1} (scade incertitudinea).

Tipare Cheie

K_t controleaza compromisul intre prior (modelul dinamic) si evidenta (observatia): K_t mic = incredere in model, K_t mare = incredere in date. Cand C=I, K_t = Sigma_{t|t-1} S_t^{-1} este raportul intre incertitudinea priorului si incertitudinea totala. R mare -> K mic (senzori zgomotosi, mai multa incredere in model). Sigma_{t|t-1} mare -> K mare (model incert, mai multa incredere in date).

Aplicatii Practice

Ecuatiile filtrului Kalman sunt implementate in biblioteci standard: FilterPy (Python), kalmanFilter (MATLAB), robot_localization (ROS). In practica se folosesc variante numerice stabile: filtrul informational (parametrizare canonica Lambda, eta) sau filtrul radacina patrata (descompunere Cholesky/UDU a covariantei).

Exemplu din Lumea Reala

In navigatia GPS, pasul de predictie foloseste modelul cinematic (pozitia noua = pozitia veche + viteza * dt) pentru a propaga estimarea intre masuratori GPS. Cand soseste o masurare GPS (la fiecare secunda), pasul de actualizare ajusteaza estimarea. Daca R este mare (masurare GPS zgomotoasa), filtrul se bazeaza mai mult pe model; daca semnalul GPS este puternic (R mic), filtrul se bazeaza pe masurare.

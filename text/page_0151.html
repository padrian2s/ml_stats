Constanta de Normalizare si Inferenta unui Scalar Necunoscut

Pagina 120 prezinta constanta de normalizare a sistemului Gaussian liniar si primul exemplu: inferenta unui scalar necunoscut din masuratori zgomotoase.

Constanta de normalizare (Ecuatia 4.126): p(y) = N(y|Aμ_x + b, Σ_y + AΣ_xAᵀ). Aceasta este verosimilitatea marginala (evidence), utila pentru compararea modelelor.
Exemplul 4.4.2.1 - Inferenta unui scalar: N masuratori zgomotoase yᵢ ale unui scalar necunoscut x, cu precizia masuratorii λ_y = 1/σ² si prior p(x) = N(x|μ₀, λ₀⁻¹).
Posteriorul (Ecuatiile 4.129-4.131): p(x|y) = N(x|μ_N, λ_N⁻¹), unde: λ_N = λ₀ + Nλ_y (precizia se aduna) μ_N = (Nλ_y * ȳ + λ₀ * μ₀) / λ_N (medie ponderata de precizii)
Interpretare: Precizia posterioara = precizia priorului + N * precizia fiecarei masuratori. Media posterioara este o combinatie convexa intre MLE (ȳ) si prior (μ₀), cu ponderi proportionale cu preciziile respective. Prior slab → posterior aproape de MLE; prior puternic → posterior aproape de prior.

Comentariu

Rezumat

Exemplul scalar ilustreaza perfect mecanismul inferentei Bayesiane: informatia se acumuleaza prin adunarea preciziilor, iar estimarea finala este un compromis intre prior si date. Cu cat avem mai multe masuratori (N mare), cu atat datele domina priorul. Aceasta este versiunea continua a modelului Beta-Binomial.

Tipare Cheie

Aditivitatea preciziei este o proprietate fundamentala: fiecare observatie noua adauga λ_y unitati de precizie. Echivalent, N masuratori cu precizie λ_y sunt echivalente cu o singura masurare cu precizie Nλ_y si valoare ȳ. Aceasta este "statisticile suficiente" in actiune.

Aplicatii Practice

Aceasta formula este utilizata in metrologie (estimarea unei constante fizice din masuratori repetate), sondaje (estimarea opiniei publice cu prior informativ) si calibrarea senzorilor.

Exemplu din Lumea Reala

La masurarea vitezei luminii, fiecare experiment produce o estimare cu o precizie cunoscuta. Posteriorul Bayesian combina toate experimentele, ponderand fiecare proportional cu precizia sa. Experimentele mai precise au influenta mai mare asupra estimarii finale.

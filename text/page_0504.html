Fields of experts (FoE): Model grafic alternativ (Roth and Black 2009) care codifica direct corelatiile intre patch-urile vecine, fara a folosi variabile latente. Mai costisitor computational, dar poate captura dependente spatiale mai complexe decat sparse coding cu patch-uri independente.

Comentariu

Rezumat

Sparse coding pentru inpainting functioneaza prin: (1) invatarea dictionarului din patch-urile intacte, (2) pentru fiecare patch corupt, estimarea coeficientilor sparse z din pixelii vizibili, (3) reconstructia pixelilor lipsa folosind Wz. Calitatea depinde de dictionarul invatat si de proportia de pixeli vizibili. Fields of experts ofera o alternativa care modeleaza explicit vecinatatea spatiala.

Tipare Cheie

Diferenta fundamentala: sparse coding trateaza patch-urile independent (eficient, paralelizabil), in timp ce fields of experts modeleaza dependentele inter-patch (calitate mai buna la marginile patch-urilor, dar mai costisitor). In practica, sparse coding cu overlap si averaging este un compromis bun.

Aplicatii Practice

Inpainting-ul modern combina sparse coding cu retele neurale profunde (deep image prior, generative models). Totusi, sparse coding ramane relevant pentru aplicatii unde datele de antrenare sunt limitate sau unde interpretabilitatea este importanta.

Exemplu din Lumea Reala

In restaurarea filmelor vechi (de exemplu, digitizarea arhivelor cinematografice), sparse coding este folosit pentru a elimina zgarierturile, petele si deteriorarile de pe cadre individuale, pastrand detaliile originale ale imaginii.

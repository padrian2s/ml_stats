Capitolul 1: Aplicatii Reale ale Clasificarii

Exemple practice de clasificare: clasificarea documentelor, filtrarea spam-ului si reprezentarea bag-of-words.

Clasificarea Documentelor: Clasificarea unui document (pagina web, email) intr-una din C clase, calculand p(y = c|x, D), unde x este o reprezentare a textului.
Filtrarea Spam: Caz special al clasificarii de documente cu doua clase: spam (y = 1) sau ham (y = 0).
Bag of Words: Reprezentare a documentelor de lungime variabila ca vectori de dimensiune fixa, unde x_ij = 1 daca cuvantul j apare in documentul i.
Figura 1.2: Subset de dimensiune 16242 x 100 din datele 20-newsgroups. Fiecare rand este un document (vector bag-of-words), fiecare coloana este un cuvant. Liniile rosii separa 4 clase (comp, rec, sci, talk). Se observa subseturi de cuvinte a caror prezenta sau absenta este indicativa pentru clasa.
Matrice de Co-aparitie: Aplicand transformarea bag-of-words la toate documentele, obtinem o matrice binara document x cuvant, reducand clasificarea la detectarea de tipare subtile in bitii matricei.

Comentariu

Rezumat

Sectiunea 1.2.1.3 prezinta aplicatii concrete ale clasificarii. Clasificarea documentelor transforma texte de lungime variabila in vectori de dimensiune fixa prin reprezentarea bag-of-words. Filtrarea spam-ului este un exemplu clasic unde prezenta anumitor cuvinte (buy, cheap, viagra) indica cu probabilitate mare un mesaj spam.

Tipare Cheie

Reprezentarea bag-of-words ignora ordinea cuvintelor, dar este surprinzator de eficace pentru clasificare. Matricea de co-aparitie din Figura 1.2 arata vizual cum diferite grupuri de cuvinte corespund diferitelor clase de documente. Aceasta este o simplificare puternica care reduce problema NLP la algebra liniara.

Aplicatii Practice

Clasificarea documentelor este utilizata in motoarele de cautare, sisteme de recomandare de stiri, analiza sentimentelor, detectarea limbii si organizarea automata a email-urilor in foldere.

Exemplu din Lumea Reala

Gmail foloseste un clasificator de spam care analizeaza cuvintele din email, linkurile, expeditorul si alte caracteristici pentru a decide automat daca un mesaj este spam. Sistemul invata continuu din feedback-ul utilizatorilor care marcheaza mesaje ca spam sau non-spam.

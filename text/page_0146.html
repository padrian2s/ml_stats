Imputarea Datelor, Imputare Multipla si Forma Informationala

Pagina 115 continua cu imputarea datelor lipsa, introduce imputarea multipla si forma informationala (canonica) a distributiei Gaussiene.

Figura 4.11: Ilustrarea imputarii datelor. Stanga: trei randuri din matricea de date cu valori lipsa (marcate cu cercuri). Centru: media predictiva posterioara bazata pe datele observate partial. Dreapta: valorile reale. Estimarile sunt foarte aproape de adevar.
Imputare multipla: In loc de a imputa o singura valoare (media), se extrag mai multe esantioane din p(x_h|x_v, θ) pentru a captura incertitudinea estimarii. Aceasta ofera o masura mai onesta a incertitudinii decat imputarea simpla.
Parametri canonici / naturali (Ecuatia 4.85): Λ = Σ⁻¹ (matricea de precizie) si ξ = Σ⁻¹μ (vectorul de precizie ponderat). Forma informationala: N_c(x|ξ, Λ).
Inmultirea Gaussienelor in forma informationala (Ecuatia 4.90): N_c(ξ_f, λ_f) * N_c(ξ_g, λ_g) = N_c(ξ_f + ξ_g, λ_f + λ_g). Simpla adunare a parametrilor! In forma cu momente, formula este mult mai complicata (Ecuatia 4.91).
Conditionarea in forma informationala (Ecuatia 4.89): p(x₁|x₂) = N_c(x₁|ξ₁ - Λ₁₂x₂, Λ₁₁). Conditionarea este simpla: se scade termenul Λ₁₂x₂ din ξ₁ si se extrage submatricea Λ₁₁.

Comentariu

Rezumat

Pagina introduce un concept important: dualitatea intre forma cu momente (μ, Σ) si forma informationala (ξ, Λ). Marginalizarea este usoara in forma cu momente (selecteaza submatricea), iar conditionarea si inmultirea sunt usoare in forma informationala (adunare/scadere). Aceasta dualitate este exploatata in algoritmii de inferenta pe modele grafice.

Tipare Cheie

Inmultirea Gaussienelor prin simpla adunare a parametrilor canonici este fundamentala in propagarea credintelor (belief propagation) pe modele grafice Gaussiene. Aceasta proprietate face inferenta exacta eficienta in retele Gaussiene.

Aplicatii Practice

Forma informationala este utilizata in filtrarea Kalman informationala (mai stabila numeric in unele cazuri), in fuziunea senzorilor (fiecare senzor contribuie cu propriul (ξ, Λ), iar fuziunea este simpla adunare) si in inferenta distribuita.

Exemplu din Lumea Reala

In fuziunea senzorilor pentru vehicule autonome, fiecare senzor (camera, lidar, radar) produce o estimare Gaussiana a pozitiei obiectelor. In forma informationala, fuziunea se face prin simpla adunare: Λ_total = Λ_camera + Λ_lidar + Λ_radar si ξ_total = ξ_camera + ξ_lidar + ξ_radar.

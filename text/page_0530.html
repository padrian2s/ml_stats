SVM pentru Clasificare - Pierderea Hinge și Optimizare Secvențială Minimală (SMO)

Pagina continuă cu clasificarea SVM, prezentând formularea duală cu programare pătratică, algoritmul SMO și predicția prin trucul kernel.

Optimizare Secvențială Minimală (SMO): Algoritm specializat pentru rezolvarea problemei de programare pătratică a SVM-urilor, care evită utilizarea solverilor QP generice. Complexitatea practică este O(N²), dar poate fi prea lentă pentru N mare.
Obiectivul SVM clasificare: min_(w,w₀,ξ) ½||w||² + C·Σᵢ₌₁ᴺ ξᵢ, cu constrângerile ξᵢ ≥ 0 și yᵢ(xᵢᵀw + w₀) ≥ 1 − ξᵢ.
Predicția kernelizată: ŷ(x) = sgn(ŵ₀ + Σᵢ₌₁ᴺ αᵢκ(xᵢ, x)), cu complexitate O(sD) unde s ≤ N este numărul de vectori suport.

Comentariu

Rezumat

Formularea duală a SVM transformă problema primală cu N + D + 1 variabile într-o problemă cu N variabile duale (multiplicatorii Lagrange). Soluția ŵ = Σᵢ αᵢxᵢ cu αᵢ = λᵢyᵢ este sparse datorită pierderii hinge. Algoritmul SMO rezolvă eficient problema duală.

Tipare Cheie

Complexitatea temporală la predicție depinde doar de numărul vectorilor suport s, nu de N total. Pentru N foarte mare, se preferă SVM-uri liniare cu complexitate O(N). Sparsitatea vectorului α este controlată de parametrul C.

Aplicatii Practice

SMO este implementat în biblioteci populare precum LIBSVM și scikit-learn. SVM-urile liniare (LIBLINEAR) sunt preferate pentru clasificarea textului cu milioane de caracteristici.

Exemplu din Lumea Reala

În clasificarea sentimentelor din recenzii de produse, un SVM liniar antrenat pe 100.000 de recenzii poate avea doar 10.000 de vectori suport, permițând clasificarea unei noi recenzii în microsecunde.

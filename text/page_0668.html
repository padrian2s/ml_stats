SSM pentru prognoza seriilor temporale si modelul de nivel local

Aceasta pagina introduce utilizarea SSM-urilor pentru prognoza seriilor temporale structurale si prezinta modelul de nivel local.

Model de serie temporala structurala: Abordarea SSM pentru serii temporale descompune semnalul in procese latente aditivecare capteaza diferite aspecte (nivel, trend, sezonalitate). Spre deosebire de metodele clasice y_{t+1} = f(y_{1:t}, theta) care opereaza direct pe observabile, SSM-urile creeaza un model generativ cu variabile latente si integreaza prin marginalizare.
Modelul de nivel local: y_t = a_t + epsilon_t^y (epsilon_t^y ~ N(0, R)), a_t = a_{t-1} + epsilon_t^a (epsilon_t^a ~ N(0, Q)). Starea ascunsa z_t = a_t modeleaza un nivel necunoscut care evolueaza lent (random walk cu varianta Q), observat cu zgomot (varianta R). Raportul Q/R controleaza netezimea: Q/R mic = neted, Q/R mare = reactiv la observatii.

Comentariu

Rezumat

SSM-urile sunt foarte potrivite pentru prognoza seriilor temporale deoarece descompun semnalul in componente interpretabile (nivel, trend, sezonalitate, cicluri). Modelul de nivel local este cel mai simplu: starea ascunsa a_t (nivelul) evolueaza ca un random walk si este observat cu zgomot. Aceasta este echivalent cu netezirea exponentiala (exponential smoothing) din statistica clasica. Parametrii Q si R controleaza compromisul intre adaptabilitate (urmarirea datelor) si stabilitate (netezirea zgomotului).

Tipare Cheie

Abordarea SSM vs abordarea clasica: metodele clasice (ARIMA, exponential smoothing) opereaza direct pe observabile; SSM-urile creeaza un model generativ cu variabile latente. Avantajele SSM: (1) interpretabilitate (fiecare componenta latenta are un sens fizic), (2) modularitate (se pot adauga/elimina componente), (3) gestionarea naturala a datelor lipsa si observatiilor irregulare.

Aplicatii Practice

Modele de serii temporale structurale sunt folosite in: prognoza vanzarilor (Amazon, Google), prognoza cererii de energie electrica, analiza macroeconomica (PIB, inflatie), epidemiologie (prognoza cazurilor), si clima (analiza tendintelor de temperatura). Pachetele Python: statsmodels (UnobservedComponents), TensorFlow Probability (tfp.sts).

Exemplu din Lumea Reala

Google foloseste modele de serii temporale structurale (implementate in CausalImpact si in tfp.sts) pentru a estima impactul cauzal al campaniilor publicitare: se descompune seria temporala a vanzarilor in nivel, trend si sezonalitate, apoi se compara prognoza contrafactuala (fara campanie) cu datele reale (cu campanie) pentru a estima efectul cauzal.

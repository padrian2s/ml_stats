Estimarea MAP a Covariantei - Shrinkage si Estimare Regularizata

Pagina 130 detaliaza estimarea MAP a covariantei ca o combinatie convexa intre prior si MLE, introduce shrinkage estimation si posteriorul univariat pentru varianta.

Estimarea MAP ca shrinkage (Ecuatia 4.183): Σ̂_map = λΣ₀ + (1 - λ)Σ̂_mle, unde λ = N₀/(N₀ + N). Combinatie convexa intre matricea de covarianta a priori Σ₀ si MLE.
Prior dependent de date (Ecuatia 4.184): Cu S₀ = diag(Σ̂_mle): Σ̂_map(i,j) = Σ̂_mle(i,j) daca i = j, altfel (1-λ)Σ̂_mle(i,j). Elementele diagonale raman neschimbate, iar elementele off-diagonale sunt "contractate" spre 0.
Shrinkage estimation / Estimare regularizata: Tehnica de contractie a estimarilor spre o tinta (prior) pentru a imbunatati stabilitatea si precizia estimarii. λ poate fi ales prin validare incrucisata sau formule in forma inchisa (Ledoit-Wolf).
Posteriorul univariat pentru σ² (Ecuatiile 4.185-4.189): p(σ²|D) = IG(σ²|a_N, b_N), unde a_N = a₀ + N/2 si b_N = b₀ + (1/2)Σᵢ(xᵢ - μ)².

Comentariu

Rezumat

Estimarea MAP a covariantei este elegant interpretata ca shrinkage: elementele off-diagonale (corelatii) sunt contractate spre zero, ceea ce reduce instabilitatea numerica si supraajustarea. In cazul univariat, posteriorul pentru varianta este o distributie Gamma inversa, analog perfect cu cazul multivariat (Wishart invers).

Tipare Cheie

Contractia elementelor off-diagonale spre zero este intuitiva: corelatiile estimate din date limitate sunt mai zgomotoase decat variantele, deci beneficiaza mai mult de regularizare. Priorul dependent de date (S₀ = diag(Σ̂_mle)) este un exemplu de prior empiric Bayes.

Aplicatii Practice

Estimatorul Ledoit-Wolf este standard in finante si genomica. In R, functia cov.shrink din pachetul corpcor implementeaza aceasta metoda. In Python, sklearn.covariance.LedoitWolf ofera implementarea.

Exemplu din Lumea Reala

In neuroimagistica (fMRI), matricea de covarianta a activitatii cerebrale intre sute de regiuni trebuie estimata din cateva sute de scanari temporale. Shrinkage-ul covariantei este esential pentru a obtine retele de conectivitate cerebrala stabile si reproductibile.
